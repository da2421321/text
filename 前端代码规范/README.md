# 前端代码规范

面向团队协作与可维护性的前端代码规范，涵盖 HTML/CSS、JavaScript/TypeScript、Vue 组件开发、以及 ESLint/Prettier 等工具链配置，配有完整代码示例与可直接复制的示例配置。

## 目录

- 01-HTML-CSS-规范.md
- 02-JS-TS-规范.md
- 03-Vue-组件规范.md
- 04-代码质量工具.md
- 05-快速开始.md

## 示例配置

示例配置位于 `前端代码规范/示例配置/` 下，可直接复制到你的项目根目录：

- `.eslintrc.cjs`
- `.prettierrc.json`
- `.eslintignore`
- `.prettierignore`
- `tsconfig.json`

## 使用方式

1. 先阅读各章节规范与示例代码。
2. 将 `示例配置` 中的文件复制到你的项目根目录。
3. 参考《05-快速开始.md》安装依赖与脚本，并运行 `lint` 与 `format`。

> 规范并非一成不变，建议在团队内进行最小公约数化，结合工具链在 CI 与提交钩子中强制执行，以确保代码质量与一致性。
