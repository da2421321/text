# 04 - 代码质量工具（ESLint + Prettier）

为 Vue3 + TS 项目提供统一的 Lint/Format 方案，配套示例配置可直接复制到项目根目录（见 `示例配置/`）。

## 安装依赖
```bash
# 任选一个包管理器
pnpm add -D eslint prettier eslint-plugin-vue @typescript-eslint/parser @typescript-eslint/eslint-plugin eslint-config-prettier eslint-plugin-prettier
# 可选：Git 钩子
pnpm add -D husky lint-staged
```

## package.json 脚本
```json
{
  "scripts": {
    "lint": "eslint . --ext .js,.ts,.vue",
    "lint:fix": "eslint . --ext .js,.ts,.vue --fix",
    "format": "prettier --write .",
    "format:check": "prettier --check .",
    "prepare": "husky install"
  },
  "lint-staged": {
    "*.{js,ts,vue,css,md}": [
      "eslint --fix",
      "prettier --write"
    ]
  }
}
```

## 启用提交钩子（可选）
```bash
# 初始化 husky 并添加 pre-commit 钩子
npx husky add .husky/pre-commit "npx lint-staged"
```

## 运行
```bash
pnpm run lint
pnpm run lint:fix
pnpm run format
pnpm run format:check
```
