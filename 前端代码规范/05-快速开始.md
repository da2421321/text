# 05 - 快速开始

1. 将 `前端代码规范/示例配置/` 下的全部文件复制到你的项目根目录。
2. 安装依赖（任选其一包管理器）：
```bash
pnpm add -D eslint prettier eslint-plugin-vue @typescript-eslint/parser @typescript-eslint/eslint-plugin eslint-config-prettier eslint-plugin-prettier
```
3. 在 `package.json` 添加脚本：
```json
{
  "scripts": {
    "lint": "eslint . --ext .js,.ts,.vue",
    "lint:fix": "eslint . --ext .js,.ts,.vue --fix",
    "format": "prettier --write .",
    "format:check": "prettier --check ."
  }
}
```
4. 执行校验与格式化：
```bash
pnpm run lint
pnpm run format
```
5. 在团队中约定：提交前必须通过 `lint` 与 `format:check`，CI 强制执行。
