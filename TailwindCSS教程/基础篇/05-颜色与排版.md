# 第五章：颜色与排版

## 5.1 颜色系统

TailwindCSS v4 使用 OKLCH 色彩空间，提供更均匀的感知亮度。

### 内置调色板

每种颜色有 11 个色阶（50、100、200...900、950）：

```html
<!-- 文字颜色 -->
<p class="text-gray-900">深灰文字</p>
<p class="text-blue-600">蓝色文字</p>
<p class="text-red-500">红色文字</p>

<!-- 背景颜色 -->
<div class="bg-white">白色背景</div>
<div class="bg-gray-50">浅灰背景</div>
<div class="bg-blue-600">蓝色背景</div>

<!-- 边框颜色 -->
<div class="border border-gray-200">浅灰边框</div>
<div class="border-2 border-blue-500">蓝色边框</div>
```

### 内置颜色列表

```
slate  gray  zinc  neutral  stone
red  orange  amber  yellow  lime
green  emerald  teal  cyan  sky
blue  indigo  violet  purple  fuchsia
pink  rose
```

### 透明度修饰符

v4 使用 `color-mix()` 实现透明度，语法更简洁：

```html
<!-- 背景透明度 -->
<div class="bg-blue-500/50">50% 透明蓝色背景</div>
<div class="bg-blue-500/20">20% 透明蓝色背景</div>

<!-- 文字透明度 -->
<p class="text-gray-900/70">70% 透明文字</p>

<!-- 边框透明度 -->
<div class="border border-blue-500/30">30% 透明边框</div>
```

### 自定义颜色

```css
@import "tailwindcss";

@theme {
  /* 品牌色 */
  --color-brand-50: oklch(97% 0.02 250);
  --color-brand-100: oklch(93% 0.05 250);
  --color-brand-500: oklch(60% 0.2 250);
  --color-brand-600: oklch(52% 0.22 250);
  --color-brand-900: oklch(25% 0.1 250);

  /* 语义化颜色 */
  --color-primary: var(--color-brand-600);
  --color-success: oklch(65% 0.2 145);
  --color-warning: oklch(75% 0.18 75);
  --color-danger: oklch(60% 0.22 25);
}
```

使用自定义颜色：

```html
<button class="bg-brand-600 hover:bg-brand-700 text-white">品牌按钮</button>
<span class="text-success">成功状态</span>
<span class="text-danger">错误状态</span>
```

## 5.3 文字排版

### 字体大小

| 工具类 | 字号 | 行高 |
|--------|------|------|
| `text-xs` | 12px | 16px |
| `text-sm` | 14px | 20px |
| `text-base` | 16px | 24px |
| `text-lg` | 18px | 28px |
| `text-xl` | 20px | 28px |
| `text-2xl` | 24px | 32px |
| `text-3xl` | 30px | 36px |
| `text-4xl` | 36px | 40px |
| `text-5xl` | 48px | 1 |
| `text-6xl` | 60px | 1 |

### 字重

```html
<p class="font-thin">100 - 极细</p>
<p class="font-light">300 - 细</p>
<p class="font-normal">400 - 正常</p>
<p class="font-medium">500 - 中等</p>
<p class="font-semibold">600 - 半粗</p>
<p class="font-bold">700 - 粗</p>
<p class="font-extrabold">800 - 极粗</p>
<p class="font-black">900 - 最粗</p>
```

### 行高

```html
<p class="leading-none">行高 1</p>
<p class="leading-tight">行高 1.25</p>
<p class="leading-snug">行高 1.375</p>
<p class="leading-normal">行高 1.5</p>
<p class="leading-relaxed">行高 1.625</p>
<p class="leading-loose">行高 2</p>
```

### 字间距

```html
<p class="tracking-tighter">字间距 -0.05em</p>
<p class="tracking-tight">字间距 -0.025em</p>
<p class="tracking-normal">字间距 0</p>
<p class="tracking-wide">字间距 0.025em</p>
<p class="tracking-wider">字间距 0.05em</p>
<p class="tracking-widest">字间距 0.1em</p>
```

### 文字对齐

```html
<p class="text-left">左对齐</p>
<p class="text-center">居中</p>
<p class="text-right">右对齐</p>
<p class="text-justify">两端对齐</p>
```

### 文字装饰

```html
<p class="underline">下划线</p>
<p class="line-through">删除线</p>
<p class="no-underline">无下划线</p>
<p class="underline decoration-blue-500 decoration-2">蓝色双线下划线</p>
<p class="underline decoration-wavy decoration-red-500">波浪下划线</p>
```

### 文字溢出

```html
<!-- 单行截断 -->
<p class="truncate">这是一段很长的文字，会被截断显示...</p>

<!-- 多行截断 -->
<p class="line-clamp-2">这是一段很长的文字，超过两行会被截断，并显示省略号...</p>
<p class="line-clamp-3">三行截断</p>

<!-- 不换行 -->
<p class="whitespace-nowrap">不换行文字</p>

<!-- 强制换行 -->
<p class="break-all">强制在任意字符处换行</p>
<p class="break-words">仅在单词边界换行</p>
```

## 5.4 自定义字体

```css
@import "tailwindcss";

@theme {
  --font-sans: 'Inter', 'PingFang SC', 'Microsoft YaHei', sans-serif;
  --font-mono: 'JetBrains Mono', 'Fira Code', monospace;
  --font-display: 'Playfair Display', serif;
}
```

```html
<p class="font-sans">无衬线字体</p>
<code class="font-mono">等宽字体</code>
<h1 class="font-display">展示字体</h1>
```

## 小结

- 颜色：11 个色阶，透明度修饰符 `/50`，`@theme` 自定义
- 渐变：`bg-gradient-to-*`、`bg-radial`、`bg-conic`
- 排版：`text-*`、`font-*`、`leading-*`、`tracking-*`
- 文字溢出：`truncate`、`line-clamp-*`
- 边框：`border-*`、`rounded-*`、`ring-*`
- 阴影：`shadow-*`，支持彩色阴影
