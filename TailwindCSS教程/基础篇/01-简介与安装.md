# 第一章：TailwindCSS 简介与安装

## 1.1 什么是 TailwindCSS

TailwindCSS 是一个**工具类优先（Utility-First）**的 CSS 框架。与 Bootstrap、Ant Design 等组件库不同，它不提供预制组件，而是提供大量原子化的工具类，让你直接在 HTML 中组合样式。

```html
<!-- 传统 CSS 写法 -->
<div class="card">
  <h2 class="card-title">Hello</h2>
</div>

<style>
.card { padding: 16px; border-radius: 8px; background: white; box-shadow: 0 2px 8px rgba(0,0,0,0.1); }
.card-title { font-size: 20px; font-weight: 600; color: #111; }
</style>

<!-- TailwindCSS 写法 -->
<div class="p-4 rounded-lg bg-white shadow-md">
  <h2 class="text-xl font-semibold text-gray-900">Hello</h2>
</div>
```

### 核心优势

- **无需命名**：不用再为 CSS 类名绞尽脑汁
- **按需生成**：只有用到的工具类才会出现在最终 CSS 中
- **一致性**：基于设计令牌，保证视觉一致性
- **响应式内置**：`md:text-xl` 这样的写法开箱即用

## 1.2 TailwindCSS v4 的重大变化

v4 是一次重大重写，核心变化：

### CSS 优先配置

v3 需要 `tailwind.config.js`：
```js
// v3 tailwind.config.js
module.exports = {
  content: ['./src/**/*.{html,js,vue}'],
  theme: {
    extend: {
      colors: {
        brand: '#3B82F6',
      }
    }
  }
}
```

v4 直接在 CSS 中配置：
```css
/* v4 main.css */
@import "tailwindcss";

@theme {
  --color-brand: #3B82F6;
}
```

### 新的导入语法

```css
/* v3 */
@tailwind base;
@tailwind components;
@tailwind utilities;

/* v4 */
@import "tailwindcss";
```

### 自动内容检测

v3 必须配置 `content` 数组，v4 自动扫描项目文件，无需任何配置。

### 全新构建引擎（Oxide）

基于 Rust 重写，构建速度提升 5-10 倍，增量构建提升更明显。

## 1.3 安装方式

### 方式一：Vite 插件（推荐）

适用于 Vite 驱动的项目（Vue、React、Svelte 等）。

```bash
# 1. 创建项目（以 Vue 为例）
npm create vue@latest my-app
cd my-app

# 2. 安装 TailwindCSS v4
npm install tailwindcss @tailwindcss/vite
```

修改 `vite.config.js`：

```js
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [
    vue(),
    tailwindcss(),  // 添加这一行
  ],
})
```

在 `src/main.css`（或 `src/style.css`）中：

```css
@import "tailwindcss";
```

确保在入口文件中引入这个 CSS：

```js
// main.js
import './main.css'
```

### 方式二：PostCSS 插件

适用于 Webpack、Rollup 等构建工具。

```bash
npm install tailwindcss @tailwindcss/postcss postcss
```

创建 `postcss.config.js`：

```js
export default {
  plugins: {
    '@tailwindcss/postcss': {},
  },
}
```

CSS 入口文件：

```css
@import "tailwindcss";
```

### 方式三：CDN（快速体验）

无需安装，直接在 HTML 中引入：

```html
<!DOCTYPE html>
<html>
<head>
  <script src="https://cdn.tailwindcss.com/4"></script>
</head>
<body>
  <h1 class="text-3xl font-bold text-blue-600">Hello TailwindCSS v4!</h1>
</body>
</html>
```

> 注意：CDN 方式仅用于学习和原型开发，生产环境请使用构建工具。

## 1.5 编辑器配置

### VS Code 推荐插件

1. **Tailwind CSS IntelliSense** - 自动补全、悬停预览、语法高亮
2. **PostCSS Language Support** - PostCSS 语法支持

### VS Code 设置

```json
{
  "editor.quickSuggestions": {
    "strings": "on"
  },
  "tailwindCSS.experimental.classRegex": [
    ["clsx\\(([^)]*)\\)", "(?:'|\"|`)([^']*)(?:'|\"|`)"]
  ]
}
```

## 小结

- TailwindCSS v4 采用 CSS 优先配置，用 `@theme` 替代 `tailwind.config.js`
- 新的导入语法：`@import "tailwindcss"`
- Vite 插件是最推荐的集成方式，零配置开箱即用
- 自动内容检测，无需手动配置 `content` 路径
