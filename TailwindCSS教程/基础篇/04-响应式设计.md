# 第四章：响应式设计

## 4.1 断点系统

TailwindCSS v4 默认断点：

| 断点 | 最小宽度 | 典型设备 |
|------|---------|---------|
| `sm` | 640px | 大手机、小平板 |
| `md` | 768px | 平板 |
| `lg` | 1024px | 笔记本 |
| `xl` | 1280px | 桌面显示器 |
| `2xl` | 1536px | 大屏显示器 |

## 4.2 移动优先策略

TailwindCSS 采用**移动优先**策略：不带断点前缀的工具类适用于所有尺寸，带前缀的工具类从该断点开始生效。

```html
<!-- 移动端：1列，平板：2列，桌面：4列 -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-4">
  <div class="bg-white rounded-lg p-4 shadow">卡片 1</div>
  <div class="bg-white rounded-lg p-4 shadow">卡片 2</div>
  <div class="bg-white rounded-lg p-4 shadow">卡片 3</div>
  <div class="bg-white rounded-lg p-4 shadow">卡片 4</div>
</div>
```

```html
<!-- 移动端：隐藏，桌面：显示 -->
<div class="hidden lg:block">
  仅桌面端显示
</div>

<!-- 移动端：显示，桌面：隐藏 -->
<div class="block lg:hidden">
  仅移动端显示
</div>
```

## 4.5 自定义断点

在 v4 中，通过 `@theme` 自定义断点：

```css
@import "tailwindcss";

@theme {
  /* 覆盖默认断点 */
  --breakpoint-sm: 480px;
  --breakpoint-md: 768px;
  --breakpoint-lg: 1024px;
  --breakpoint-xl: 1280px;
  --breakpoint-2xl: 1600px;

  /* 添加新断点 */
  --breakpoint-xs: 375px;
  --breakpoint-3xl: 1920px;
}
```

使用自定义断点：

```html
<div class="xs:text-sm 3xl:text-2xl">
  自定义断点响应式文字
</div>
```

## 4.6 容器查询（Container Queries）

v4 内置支持容器查询，无需额外插件。

```html
<!-- 定义容器 -->
<div class="@container">
  <!-- 根据容器宽度响应，而非视口宽度 -->
  <div class="grid grid-cols-1 @sm:grid-cols-2 @lg:grid-cols-3 gap-4">
    <div class="bg-white p-4 rounded-lg">卡片 1</div>
    <div class="bg-white p-4 rounded-lg">卡片 2</div>
    <div class="bg-white p-4 rounded-lg">卡片 3</div>
  </div>
</div>
```

容器查询断点（`@` 前缀）：

| 变体 | 最小宽度 |
|------|---------|
| `@xs` | 320px |
| `@sm` | 384px |
| `@md` | 448px |
| `@lg` | 512px |
| `@xl` | 576px |
| `@2xl` | 672px |

### 命名容器

```html
<div class="@container/sidebar">
  <div class="@lg/sidebar:hidden">
    侧边栏宽时隐藏
  </div>
</div>
```

## 4.7 最大宽度变体（`max-*`）

v4 新增 `max-*` 变体，用于"最大宽度以下"的样式：

```html
<!-- 仅在 md 以下（小于 768px）生效 -->
<div class="max-md:hidden">
  仅在平板及以上显示
</div>

<!-- 仅在 lg 以下生效 -->
<div class="max-lg:flex-col">
  小屏垂直排列
</div>
```

## 小结

- 移动优先：无前缀 = 所有尺寸，`md:` = 768px 以上
- 默认断点：`sm`(640) `md`(768) `lg`(1024) `xl`(1280) `2xl`(1536)
- `@theme` 自定义断点
- 容器查询：`@container` + `@sm:` 等变体
- `max-*` 变体：最大宽度以下生效
