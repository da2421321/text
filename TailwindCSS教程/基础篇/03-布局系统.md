# 第三章：布局系统

## 3.1 Display 工具类

```html
<div class="block">块级元素</div>
<span class="inline-block">行内块</span>
<div class="flex">Flex 容器</div>
<div class="grid">Grid 容器</div>
<div class="hidden">隐藏（display: none）</div>
<div class="invisible">不可见但占位（visibility: hidden）</div>
```

## 3.2 Flexbox 布局

### 基础 Flex

```html
<!-- 水平居中 -->
<div class="flex items-center justify-center h-screen">
  <p>完全居中</p>
</div>

<!-- 两端对齐 -->
<div class="flex items-center justify-between px-4 py-3">
  <span>左侧</span>
  <span>右侧</span>
</div>

<!-- 垂直排列 -->
<div class="flex flex-col gap-4">
  <div>第一行</div>
  <div>第二行</div>
  <div>第三行</div>
</div>
```

### Flex 方向

| 工具类 | CSS |
|--------|-----|
| `flex-row` | `flex-direction: row`（默认） |
| `flex-row-reverse` | `flex-direction: row-reverse` |
| `flex-col` | `flex-direction: column` |
| `flex-col-reverse` | `flex-direction: column-reverse` |

### 对齐方式

```html
<!-- 主轴对齐（justify-content） -->
<div class="flex justify-start">左对齐</div>
<div class="flex justify-center">居中</div>
<div class="flex justify-end">右对齐</div>
<div class="flex justify-between">两端对齐</div>
<div class="flex justify-around">均匀分布</div>
<div class="flex justify-evenly">等间距</div>

<!-- 交叉轴对齐（align-items） -->
<div class="flex items-start">顶部对齐</div>
<div class="flex items-center">垂直居中</div>
<div class="flex items-end">底部对齐</div>
<div class="flex items-stretch">拉伸（默认）</div>
<div class="flex items-baseline">基线对齐</div>
```

### Flex 子项

```html
<div class="flex gap-4">
  <!-- flex-1：平均分配剩余空间 -->
  <div class="flex-1 bg-blue-100 p-4">自动填充</div>
  <div class="flex-1 bg-green-100 p-4">自动填充</div>

  <!-- 固定宽度不参与分配 -->
  <div class="w-32 bg-red-100 p-4">固定 128px</div>
</div>

<!-- 换行 -->
<div class="flex flex-wrap gap-2">
  <div class="w-40 bg-gray-100 p-2">项目 1</div>
  <div class="w-40 bg-gray-100 p-2">项目 2</div>
  <div class="w-40 bg-gray-100 p-2">项目 3</div>
</div>
```

### 实战：导航栏

```html
<nav class="flex items-center justify-between px-6 py-4 bg-white shadow-sm">
  <!-- Logo -->
  <a href="/" class="text-xl font-bold text-blue-600">MyApp</a>

  <!-- 导航链接 -->
  <ul class="flex items-center gap-6">
    <li><a href="#" class="text-gray-600 hover:text-blue-600 transition-colors">首页</a></li>
    <li><a href="#" class="text-gray-600 hover:text-blue-600 transition-colors">产品</a></li>
    <li><a href="#" class="text-gray-600 hover:text-blue-600 transition-colors">关于</a></li>
  </ul>

  <!-- 操作按钮 -->
  <div class="flex items-center gap-3">
    <button class="text-gray-600 hover:text-gray-900">登录</button>
    <button class="bg-blue-600 text-white px-4 py-2 rounded-lg hover:bg-blue-700 transition-colors">
      注册
    </button>
  </div>
</nav>
```

## 3.3 Grid 布局

### 基础 Grid

```html
<!-- 三列等宽 -->
<div class="grid grid-cols-3 gap-4">
  <div class="bg-gray-100 p-4">1</div>
  <div class="bg-gray-100 p-4">2</div>
  <div class="bg-gray-100 p-4">3</div>
</div>

<!-- 响应式列数 -->
<div class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-6">
  <div class="bg-white rounded-lg shadow p-4">卡片 1</div>
  <div class="bg-white rounded-lg shadow p-4">卡片 2</div>
  <div class="bg-white rounded-lg shadow p-4">卡片 3</div>
  <div class="bg-white rounded-lg shadow p-4">卡片 4</div>
</div>
```

### 列跨越

```html
<div class="grid grid-cols-4 gap-4">
  <!-- 跨越 2 列 -->
  <div class="col-span-2 bg-blue-100 p-4">跨 2 列</div>
  <div class="bg-gray-100 p-4">1 列</div>
  <div class="bg-gray-100 p-4">1 列</div>

  <!-- 跨越全部列 -->
  <div class="col-span-full bg-green-100 p-4">跨全部列</div>
</div>
```

### 行跨越

```html
<div class="grid grid-cols-3 grid-rows-3 gap-4 h-64">
  <div class="row-span-2 bg-blue-100 p-4">跨 2 行</div>
  <div class="bg-gray-100 p-4">普通</div>
  <div class="bg-gray-100 p-4">普通</div>
  <div class="bg-gray-100 p-4">普通</div>
  <div class="col-span-2 bg-green-100 p-4">跨 2 列</div>
</div>
```

## 3.4 间距系统

TailwindCSS 使用 4px 为基础单位（1 = 4px）。

### 内边距（Padding）

```html
<div class="p-4">所有方向 16px</div>
<div class="px-6 py-3">水平 24px，垂直 12px</div>
<div class="pt-2 pr-4 pb-2 pl-4">各方向单独设置</div>
<div class="ps-4 pe-4">逻辑属性（start/end）</div>
```

### 外边距（Margin）

```html
<div class="m-4">所有方向 16px</div>
<div class="mx-auto">水平居中</div>
<div class="mt-8 mb-4">上 32px，下 16px</div>
<div class="-mt-4">负外边距</div>
```

### 间距（Space Between）

```html
<!-- 子元素之间的间距 -->
<div class="flex space-x-4">
  <div>A</div>
  <div>B</div>
  <div>C</div>
</div>

<!-- 推荐使用 gap（更现代） -->
<div class="flex gap-4">
  <div>A</div>
  <div>B</div>
</div>
```

## 3.5 定位

```html
<!-- 相对定位 -->
<div class="relative">
  <!-- 绝对定位，相对于父元素 -->
  <div class="absolute top-0 right-0 w-4 h-4 bg-red-500 rounded-full"></div>
  内容
</div>

<!-- 固定定位 -->
<div class="fixed bottom-4 right-4 bg-blue-600 text-white p-4 rounded-full shadow-lg">
  悬浮按钮
</div>

<!-- 粘性定位 -->
<header class="sticky top-0 z-50 bg-white shadow-sm">
  导航栏
</header>
```

### z-index

```html
<div class="z-10">层级 10</div>
<div class="z-50">层级 50</div>
<div class="z-[100]">自定义层级</div>
```

## 3.6 容器（Container）

```html
<!-- 居中容器，自动适应断点最大宽度 -->
<div class="container mx-auto px-4">
  页面内容
</div>
```

v4 中 `container` 默认不居中，需要配合 `mx-auto`。

自定义容器：

```css
@import "tailwindcss";

@theme {
  --container-padding: 2rem;
  --breakpoint-sm: 640px;
  --breakpoint-md: 768px;
  --breakpoint-lg: 1024px;
  --breakpoint-xl: 1280px;
}
```

## 小结

- Flexbox：`flex`、`items-*`、`justify-*`、`gap-*`
- Grid：`grid`、`grid-cols-*`、`col-span-*`、`gap-*`
- 间距：`p-*`、`m-*`、`gap-*`，基础单位 4px
- 定位：`relative`、`absolute`、`fixed`、`sticky`
