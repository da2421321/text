# 第七章：状态变体与交互

## 7.1 伪类变体

### 常用交互状态

```html
<!-- hover：鼠标悬停 -->
<button class="bg-blue-600 hover:bg-blue-700 text-white px-4 py-2 rounded-lg transition-colors">
  悬停变深
</button>

<!-- focus：键盘聚焦 -->
<input class="border border-gray-300 focus:border-blue-500 focus:ring-2 focus:ring-blue-500/20 focus:outline-none rounded-lg px-3 py-2">

<!-- active：点击按下 -->
<button class="bg-blue-600 active:scale-95 transition-transform">
  点击缩小
</button>

<!-- disabled：禁用状态 -->
<button class="bg-blue-600 text-white px-4 py-2 rounded-lg disabled:opacity-50 disabled:cursor-not-allowed" disabled>
  禁用按钮
</button>
```

### 表单状态

```html
<!-- checked：复选框/单选框选中 -->
<input type="checkbox" class="checked:bg-blue-600 checked:border-blue-600">

<!-- required：必填字段 -->
<input class="required:border-red-300" required>

<!-- valid / invalid：表单验证 -->
<input type="email" class="valid:border-green-500 invalid:border-red-500 focus:outline-none border rounded px-3 py-2">

<!-- placeholder -->
<input class="placeholder:text-gray-400 placeholder:italic" placeholder="请输入...">
```

### 结构性伪类

```html
<!-- first-child / last-child -->
<ul>
  <li class="first:pt-0 last:pb-0 py-3 border-b last:border-b-0">列表项</li>
  <li class="first:pt-0 last:pb-0 py-3 border-b last:border-b-0">列表项</li>
  <li class="first:pt-0 last:pb-0 py-3 border-b last:border-b-0">列表项</li>
</ul>

<!-- odd / even：奇偶行 -->
<tr class="odd:bg-white even:bg-gray-50">
  <td>表格行</td>
</tr>
```

## 7.2 v4 新变体

### `not-*` 变体

对不满足条件的元素应用样式：

```html
<!-- 非 hover 状态时透明 -->
<button class="not-hover:opacity-70 transition-opacity">
  非悬停时半透明
</button>

<!-- 非 focus 状态时隐藏边框 -->
<input class="not-focus:border-transparent border-2 border-blue-500 rounded px-3 py-2 outline-none">

<!-- 非 disabled 状态时显示 hover 效果 -->
<button class="not-disabled:hover:bg-blue-700 bg-blue-600 text-white px-4 py-2 rounded-lg">
  智能按钮
</button>
```

### `nth-*` 变体

```html
<!-- 第 3 个子元素 -->
<li class="nth-3:bg-yellow-100">第三项高亮</li>

<!-- 奇数项（等同于 odd:） -->
<li class="nth-odd:bg-gray-50">奇数行</li>

<!-- 每 3 个一组的第一个 -->
<li class="nth-[3n+1]:font-bold">每组第一个加粗</li>

<!-- 最后 2 个 -->
<li class="nth-last-2:text-gray-400">倒数第二个</li>
```

### `inert` 变体

对 `inert` 属性的元素应用样式：

```html
<div inert class="inert:opacity-50 inert:pointer-events-none">
  不可交互区域
</div>
```

### `starting` 变体（`@starting-style`）

用于元素首次出现时的动画：

```html
<dialog class="
  open:opacity-100 open:translate-y-0
  starting:open:opacity-0 starting:open:translate-y-4
  transition-all duration-300
">
  弹窗内容（从下方淡入）
</dialog>
```

### `open` 变体

```html
<!-- details/summary 展开状态 -->
<details class="border rounded-lg overflow-hidden">
  <summary class="px-4 py-3 cursor-pointer font-medium open:bg-blue-50">
    点击展开
  </summary>
  <div class="px-4 py-3 text-gray-600">
    展开的内容
  </div>
</details>

<!-- dialog 打开状态 -->
<dialog class="open:flex hidden items-center justify-center">
  弹窗
</dialog>
```

## 7.3 `group-*` 变体

当父元素处于某状态时，子元素改变样式：

```html
<!-- 基础用法 -->
<div class="group hover:bg-blue-50 p-4 rounded-lg cursor-pointer transition-colors">
  <h3 class="font-semibold text-gray-900 group-hover:text-blue-600 transition-colors">
    卡片标题
  </h3>
  <p class="text-gray-500 group-hover:text-gray-700 transition-colors">
    卡片描述
  </p>
  <span class="text-blue-600 opacity-0 group-hover:opacity-100 transition-opacity">
    查看详情 →
  </span>
</div>
```

### 命名 group

```html
<div class="group/card hover:shadow-lg transition-shadow">
  <div class="group/button">
    <button class="group-hover/card:bg-blue-50 group-hover/button:bg-blue-100 px-4 py-2 rounded">
      按钮
    </button>
  </div>
</div>
```

### group 与其他变体组合

```html
<div class="group">
  <!-- group-focus-within：组内有元素聚焦时 -->
  <label class="group-focus-within:text-blue-600 text-gray-600 text-sm font-medium">
    邮箱
  </label>
  <input class="border rounded px-3 py-2 w-full focus:outline-none focus:border-blue-500">
</div>
```

## 7.4 `peer-*` 变体

基于兄弟元素状态改变样式：

```html
<!-- 输入框验证提示 -->
<div>
  <input
    type="email"
    class="peer border rounded px-3 py-2 w-full focus:outline-none focus:border-blue-500 invalid:border-red-500"
    placeholder="请输入邮箱"
  >
  <p class="hidden peer-invalid:block text-red-500 text-sm mt-1">
    请输入有效的邮箱地址
  </p>
</div>

<!-- 自定义复选框 -->
<label class="flex items-center gap-3 cursor-pointer">
  <input type="checkbox" class="peer sr-only">
  <div class="w-5 h-5 border-2 border-gray-300 rounded peer-checked:bg-blue-600 peer-checked:border-blue-600 transition-colors flex items-center justify-center">
    <svg class="w-3 h-3 text-white hidden peer-checked:block" fill="currentColor" viewBox="0 0 20 20">
      <path d="M16.707 5.293a1 1 0 010 1.414l-8 8a1 1 0 01-1.414 0l-4-4a1 1 0 011.414-1.414L8 12.586l7.293-7.293a1 1 0 011.414 0z"/>
    </svg>
  </div>
  <span class="text-gray-700">同意服务条款</span>
</label>
```

## 7.5 `@variant` 自定义变体

v4 新增 `@variant` 指令，在 CSS 中定义自定义变体：

```css
@import "tailwindcss";

/* 定义自定义变体 */
@variant hocus {
  &:hover, &:focus {
    @slot;
  }
}

@variant theme-dark {
  .dark & {
    @slot;
  }
}

@variant supports-grid {
  @supports (display: grid) {
    @slot;
  }
}
```

使用自定义变体：

```html
<button class="hocus:bg-blue-700 bg-blue-600 text-white px-4 py-2 rounded">
  悬停或聚焦时变深
</button>

<div class="theme-dark:bg-gray-800 bg-white">
  深色主题背景
</div>
```

## 7.6 数据属性变体

```html
<!-- data-* 属性变体 -->
<div data-active="true" class="data-[active=true]:bg-blue-100 data-[active=true]:text-blue-800 p-4 rounded">
  激活状态
</div>

<!-- aria 属性变体 -->
<button aria-pressed="true" class="aria-pressed:bg-blue-600 aria-pressed:text-white px-4 py-2 rounded border">
  切换按钮
</button>

<div aria-expanded="false" class="aria-expanded:rotate-180 transition-transform">
  ▼
</div>
```

## 小结

- 伪类变体：`hover:`、`focus:`、`active:`、`disabled:`、`checked:` 等
- v4 新变体：`not-*`、`nth-*`、`inert`、`starting`、`open`
- `group-*`：父元素状态影响子元素
- `peer-*`：兄弟元素状态影响后续元素
- `@variant`：在 CSS 中自定义变体
- `data-[*]:`、`aria-*:`：数据属性变体
