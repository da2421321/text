# 第八章：动画与过渡

## 8.1 过渡（Transition）

### 基础过渡

```html
<!-- 过渡所有属性 -->
<button class="bg-blue-600 hover:bg-blue-700 transition-all duration-200">
  所有属性过渡
</button>

<!-- 只过渡颜色 -->
<button class="bg-blue-600 hover:bg-blue-700 transition-colors duration-150">
  颜色过渡
</button>

<!-- 只过渡变换 -->
<div class="hover:scale-105 transition-transform duration-300">
  缩放过渡
</div>

<!-- 只过渡透明度 -->
<div class="opacity-0 hover:opacity-100 transition-opacity duration-500">
  淡入淡出
</div>
```

### 过渡时长

```html
<div class="transition duration-75">75ms</div>
<div class="transition duration-100">100ms</div>
<div class="transition duration-150">150ms</div>
<div class="transition duration-200">200ms</div>
<div class="transition duration-300">300ms</div>
<div class="transition duration-500">500ms</div>
<div class="transition duration-700">700ms</div>
<div class="transition duration-1000">1000ms</div>
```

### 缓动函数

```html
<div class="transition ease-linear">线性</div>
<div class="transition ease-in">加速进入</div>
<div class="transition ease-out">减速退出</div>
<div class="transition ease-in-out">先加速后减速</div>
```

### 过渡延迟

```html
<div class="transition delay-100">延迟 100ms</div>
<div class="transition delay-300">延迟 300ms</div>
<div class="transition delay-500">延迟 500ms</div>
```

## 8.2 变换（Transform）

### 缩放

```html
<div class="hover:scale-105 transition-transform">放大 5%</div>
<div class="hover:scale-110 transition-transform">放大 10%</div>
<div class="hover:scale-95 active:scale-95 transition-transform">缩小 5%</div>
<div class="hover:scale-x-110 transition-transform">水平放大</div>
```

### 旋转

```html
<div class="hover:rotate-12 transition-transform">旋转 12°</div>
<div class="hover:-rotate-6 transition-transform">反向旋转 6°</div>
<div class="hover:rotate-180 transition-transform duration-500">旋转 180°</div>
```

### 平移

```html
<div class="hover:translate-x-2 transition-transform">向右移动</div>
<div class="hover:-translate-y-1 transition-transform">向上移动</div>
<div class="hover:translate-x-1 hover:-translate-y-1 transition-transform">右上移动</div>
```

### 3D 变换（v4 新增）

```html
<!-- 3D 旋转 -->
<div class="hover:rotate-x-12 transition-transform perspective-1000">
  X 轴旋转
</div>

<div class="hover:rotate-y-12 transition-transform perspective-1000">
  Y 轴旋转
</div>

<!-- 透视 -->
<div class="perspective-500">
  <div class="hover:rotate-y-45 transition-transform duration-500">
    3D 翻转卡片
  </div>
</div>
```

## 8.3 内置动画

```html
<!-- 旋转加载 -->
<div class="animate-spin w-8 h-8 border-4 border-blue-600 border-t-transparent rounded-full"></div>

<!-- 脉冲（呼吸灯） -->
<div class="animate-pulse bg-gray-200 rounded h-4 w-32"></div>

<!-- 弹跳 -->
<div class="animate-bounce text-2xl">↓</div>

<!-- 闪烁 -->
<div class="animate-ping absolute w-3 h-3 bg-blue-400 rounded-full"></div>
```

### 骨架屏（Skeleton）

```html
<div class="animate-pulse space-y-4 p-4">
  <!-- 头像 -->
  <div class="flex items-center gap-4">
    <div class="w-12 h-12 bg-gray-200 rounded-full"></div>
    <div class="flex-1 space-y-2">
      <div class="h-4 bg-gray-200 rounded w-3/4"></div>
      <div class="h-3 bg-gray-200 rounded w-1/2"></div>
    </div>
  </div>
  <!-- 内容 -->
  <div class="space-y-2">
    <div class="h-4 bg-gray-200 rounded"></div>
    <div class="h-4 bg-gray-200 rounded w-5/6"></div>
    <div class="h-4 bg-gray-200 rounded w-4/6"></div>
  </div>
</div>
```

## 8.4 自定义动画

```css
@import "tailwindcss";

@theme {
  /* 定义关键帧 */
  --animate-fade-in: fade-in 0.3s ease-out;
  --animate-slide-up: slide-up 0.4s ease-out;
  --animate-slide-down: slide-down 0.3s ease-out;
  --animate-shake: shake 0.5s ease-in-out;
}

@keyframes fade-in {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes slide-up {
  from { opacity: 0; transform: translateY(16px); }
  to { opacity: 1; transform: translateY(0); }
}

@keyframes slide-down {
  from { opacity: 0; transform: translateY(-16px); }
  to { opacity: 1; transform: translateY(0); }
}

@keyframes shake {
  0%, 100% { transform: translateX(0); }
  20%, 60% { transform: translateX(-8px); }
  40%, 80% { transform: translateX(8px); }
}
```

使用自定义动画：

```html
<div class="animate-fade-in">淡入</div>
<div class="animate-slide-up">从下滑入</div>
<div class="animate-shake">抖动（错误提示）</div>
```

## 8.5 `starting` 变体（入场动画）

v4 新增 `starting:` 变体，对应 CSS `@starting-style`，实现元素首次渲染时的动画：

```html
<!-- 弹窗入场动画 -->
<dialog
  id="modal"
  class="
    fixed inset-0 m-auto w-96 bg-white rounded-2xl shadow-2xl p-6
    open:opacity-100 open:scale-100
    starting:open:opacity-0 starting:open:scale-95
    transition-all duration-300
  "
>
  <h2 class="text-xl font-bold mb-4">弹窗标题</h2>
  <p class="text-gray-600">弹窗内容</p>
  <button onclick="this.closest('dialog').close()" class="mt-4 bg-blue-600 text-white px-4 py-2 rounded-lg">
    关闭
  </button>
</dialog>

<!-- Popover 入场动画 -->
<div
  popover
  class="
    bg-white rounded-xl shadow-lg p-4 border border-gray-100
    open:opacity-100 open:translate-y-0
    starting:open:opacity-0 starting:open:-translate-y-2
    transition-all duration-200
  "
>
  Popover 内容
</div>
```

## 8.6 实战：卡片悬停效果

```html
<div class="group relative overflow-hidden bg-white rounded-2xl shadow-md hover:shadow-xl transition-all duration-300 cursor-pointer">
  <!-- 图片 -->
  <div class="overflow-hidden aspect-video">
    <img
      src="image.jpg"
      alt="图片"
      class="w-full h-full object-cover group-hover:scale-110 transition-transform duration-500"
    >
  </div>

  <!-- 遮罩层 -->
  <div class="absolute inset-0 bg-gradient-to-t from-black/60 via-transparent to-transparent opacity-0 group-hover:opacity-100 transition-opacity duration-300"></div>

  <!-- 内容 -->
  <div class="p-5">
    <h3 class="font-bold text-gray-900 text-lg mb-2 group-hover:text-blue-600 transition-colors">
      卡片标题
    </h3>
    <p class="text-gray-500 text-sm line-clamp-2">
      卡片描述文字，这里是一段描述内容...
    </p>
  </div>

  <!-- 悬停时显示的操作按钮 -->
  <div class="absolute bottom-4 right-4 opacity-0 translate-y-2 group-hover:opacity-100 group-hover:translate-y-0 transition-all duration-300">
    <button class="bg-blue-600 text-white text-sm px-4 py-2 rounded-lg shadow-lg">
      查看详情
    </button>
  </div>
</div>
```

```html
    </svg>

  </span>
</button>
```

## 小结

- 过渡：`transition-*`、`duration-*`、`ease-*`、`delay-*`
- 变换：`scale-*`、`rotate-*`、`translate-*`，v4 新增 3D 变换
- 内置动画：`animate-spin`、`animate-pulse`、`animate-bounce`、`animate-ping`
- 自定义动画：`@theme` 中定义 `--animate-*` + `@keyframes`
- `starting:` 变体：元素首次出现时的入场动画
