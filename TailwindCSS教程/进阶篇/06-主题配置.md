# 第六章：主题配置（CSS 优先）

## 6.1 `@theme` 指令

v4 最大的变化：用 CSS 中的 `@theme` 指令替代 `tailwind.config.js`。

```css
@import "tailwindcss";

@theme {
  /* 在这里定义设计令牌 */
  --color-brand: #3B82F6;
  --spacing-18: 4.5rem;
  --font-size-2xs: 0.625rem;
}
```

`@theme` 中定义的变量会：
1. 自动生成对应的 TailwindCSS 工具类
2. 作为 CSS 变量暴露给整个页面

## 6.2 颜色令牌

### 定义颜色

```css
@import "tailwindcss";

@theme {
  /* 单色 */
  --color-brand: #3B82F6;

  /* 色阶 */
  --color-primary-50: oklch(97% 0.02 250);
  --color-primary-100: oklch(93% 0.05 250);
  --color-primary-200: oklch(87% 0.09 250);
  --color-primary-300: oklch(79% 0.13 250);
  --color-primary-400: oklch(70% 0.17 250);
  --color-primary-500: oklch(60% 0.2 250);
  --color-primary-600: oklch(52% 0.22 250);
  --color-primary-700: oklch(44% 0.2 250);
  --color-primary-800: oklch(36% 0.16 250);
  --color-primary-900: oklch(28% 0.12 250);
  --color-primary-950: oklch(18% 0.08 250);

  /* 语义化颜色 */
  --color-success: oklch(65% 0.2 145);
  --color-warning: oklch(75% 0.18 75);
  --color-error: oklch(60% 0.22 25);
  --color-info: oklch(65% 0.18 220);
}
```

使用：

```html
<button class="bg-primary-600 hover:bg-primary-700 text-white">主要按钮</button>
<span class="text-success">成功</span>
<span class="text-error">错误</span>
<div class="bg-primary-50 border border-primary-200 text-primary-800 p-4 rounded-lg">
  提示框
</div>
```

### 禁用默认颜色

```css
@import "tailwindcss";

@theme {
  /* 清除所有默认颜色 */
  --color-*: initial;

  /* 只保留自定义颜色 */
  --color-white: #ffffff;
  --color-black: #000000;
  --color-brand: #3B82F6;
}
```

## 6.3 间距令牌

```css
@import "tailwindcss";

@theme {
  /* 扩展间距 */
  --spacing-18: 4.5rem;   /* p-18, m-18, w-18 等 */
  --spacing-22: 5.5rem;
  --spacing-128: 32rem;
  --spacing-144: 36rem;
}
```

```html
<div class="p-18 w-128">自定义间距</div>
```

## 6.4 字体令牌

```css
@import "tailwindcss";

@theme {
  /* 字体族 */
  --font-sans: 'Inter', 'PingFang SC', 'Microsoft YaHei', system-ui, sans-serif;
  --font-mono: 'JetBrains Mono', 'Fira Code', 'Cascadia Code', monospace;
  --font-display: 'Playfair Display', Georgia, serif;

  /* 字体大小（同时设置行高） */
  --font-size-2xs: 0.625rem;
  --font-size-2xs--line-height: 1rem;

  /* 字重 */
  --font-weight-black: 900;
}
```

## 6.5 断点令牌

```css
@import "tailwindcss";

@theme {
  /* 覆盖默认断点 */
  --breakpoint-sm: 576px;
  --breakpoint-md: 768px;
  --breakpoint-lg: 992px;
  --breakpoint-xl: 1200px;
  --breakpoint-2xl: 1400px;

  /* 添加自定义断点 */
  --breakpoint-xs: 480px;
}
```

## 6.6 圆角令牌

```css
@import "tailwindcss";

@theme {
  --radius-sm: 4px;
  --radius-md: 8px;
  --radius-lg: 12px;
  --radius-xl: 16px;
  --radius-2xl: 24px;
  --radius-card: 20px;  /* 自定义名称 */
}
```

```html
<div class="rounded-card">自定义圆角卡片</div>
```

## 6.7 阴影令牌

```css
@import "tailwindcss";

@theme {
  --shadow-card: 0 2px 8px 0 rgb(0 0 0 / 0.08), 0 1px 2px 0 rgb(0 0 0 / 0.04);
  --shadow-modal: 0 20px 60px -10px rgb(0 0 0 / 0.3);
  --shadow-button: 0 1px 3px 0 rgb(0 0 0 / 0.1), 0 1px 2px -1px rgb(0 0 0 / 0.1);
}
```

```html
<div class="shadow-card">卡片阴影</div>
<div class="shadow-modal">弹窗阴影</div>
```

## 6.8 CSS 变量的直接使用

`@theme` 中定义的变量可以在任何 CSS 中直接使用：

```css
@import "tailwindcss";

@theme {
  --color-brand: #3B82F6;
  --spacing-section: 5rem;
}

/* 在普通 CSS 中使用 */
.hero {
  background: linear-gradient(135deg, var(--color-brand), oklch(60% 0.2 280));
  padding: var(--spacing-section) 0;
}
```

## 6.9 多主题（品牌切换）

```css
@import "tailwindcss";

/* 默认主题（蓝色品牌） */
@theme {
  --color-brand-500: oklch(60% 0.2 250);
  --color-brand-600: oklch(52% 0.22 250);
}

/* 绿色主题 */
.theme-green {
  --color-brand-500: oklch(65% 0.2 145);
  --color-brand-600: oklch(57% 0.22 145);
}

/* 紫色主题 */
.theme-purple {
  --color-brand-500: oklch(60% 0.2 300);
  --color-brand-600: oklch(52% 0.22 300);
}
```

```html
<!-- 切换主题只需改变根元素的 class -->
<body class="theme-green">
  <button class="bg-brand-600 text-white px-4 py-2 rounded-lg">
    绿色主题按钮
  </button>
</body>
```

## 6.10 完整企业级主题示例

```css
@import "tailwindcss";

@theme {
  /* ===== 颜色 ===== */

  --color-primary-500: oklch(60% 0.2 250);
  --color-primary-600: oklch(52% 0.22 250);
  --color-primary-700: oklch(44% 0.2 250);

  --color-gray-50: oklch(98% 0 0);
  --color-gray-100: oklch(95% 0 0);
  --color-gray-200: oklch(90% 0 0);
  --color-gray-500: oklch(65% 0 0);
  --color-gray-700: oklch(40% 0 0);
  --color-gray-900: oklch(20% 0 0);

  --color-success: oklch(65% 0.2 145);
  --color-warning: oklch(75% 0.18 75);
  --color-error: oklch(60% 0.22 25);

  /* ===== 字体 ===== */
  --font-sans: 'Inter', 'PingFang SC', 'Microsoft YaHei', sans-serif;
  --font-mono: 'JetBrains Mono', monospace;

  /* ===== 间距 ===== */
  --spacing-18: 4.5rem;
  --spacing-22: 5.5rem;

  /* ===== 圆角 ===== */
  --radius-card: 16px;
  --radius-button: 8px;

  /* ===== 阴影 ===== */
  --shadow-card: 0 1px 3px 0 rgb(0 0 0 / 0.1), 0 1px 2px -1px rgb(0 0 0 / 0.1);
  --shadow-elevated: 0 10px 25px -5px rgb(0 0 0 / 0.1), 0 4px 6px -2px rgb(0 0 0 / 0.05);

  /* ===== 断点 ===== */
  --breakpoint-xs: 480px;
}
```

## 小结

- `@theme` 替代 `tailwind.config.js`，在 CSS 中配置设计令牌
- 所有令牌自动生成工具类和 CSS 变量
- 支持颜色、间距、字体、断点、圆角、阴影等所有令牌
- 通过 CSS 变量覆盖实现多主题切换
- `--color-*: initial` 清除默认值
