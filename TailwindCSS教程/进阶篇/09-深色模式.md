# 第九章：深色模式

## 9.1 基础深色模式

使用 `dark:` 变体为深色模式添加样式：

```html
<div class="bg-white dark:bg-gray-900 text-gray-900 dark:text-white p-6 rounded-xl">
  <h2 class="text-xl font-bold mb-2">标题</h2>
  <p class="text-gray-600 dark:text-gray-400">描述文字</p>
</div>
```

## 9.2 深色模式策略

### 策略一：跟随系统（默认）

v4 默认使用媒体查询 `@media (prefers-color-scheme: dark)` 检测系统主题：

```css
@import "tailwindcss";
/* 默认即跟随系统，无需额外配置 */
```

```html
<!-- 系统切换深色时自动生效 -->
<body class="bg-white dark:bg-gray-950">
  内容
</body>
```

### 策略二：手动切换（class 模式）

通过给 `<html>` 添加 `dark` 类来控制：

```css
@import "tailwindcss";

@variant dark {
  /* 当 html 有 dark 类时生效 */
  .dark & {
    @slot;
  }
}
```

```html
<html class="dark">
  <body class="bg-white dark:bg-gray-950">
    深色模式内容
  </body>
</html>
```

JavaScript 切换：

```js
// 切换深色模式
function toggleDark() {
  document.documentElement.classList.toggle('dark')
  // 持久化到 localStorage
  const isDark = document.documentElement.classList.contains('dark')
  localStorage.setItem('theme', isDark ? 'dark' : 'light')
}

// 初始化时读取偏好
const savedTheme = localStorage.getItem('theme')
const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches

if (savedTheme === 'dark' || (!savedTheme && prefersDark)) {
  document.documentElement.classList.add('dark')
}
```

## 9.3 CSS 变量实现主题切换

更推荐的方式：用 CSS 变量定义语义化颜色，深色模式只需覆盖变量值。

```css
@import "tailwindcss";

@theme {
  /* 语义化颜色令牌 */
  --color-bg: oklch(100% 0 0);
  --color-bg-secondary: oklch(97% 0 0);
  --color-surface: oklch(100% 0 0);
  --color-border: oklch(90% 0 0);
  --color-text: oklch(15% 0 0);
  --color-text-secondary: oklch(45% 0 0);
  --color-text-muted: oklch(65% 0 0);
}

/* 深色模式覆盖 */
.dark {
  --color-bg: oklch(12% 0 0);
  --color-bg-secondary: oklch(16% 0 0);
  --color-surface: oklch(18% 0 0);
  --color-border: oklch(25% 0 0);
  --color-text: oklch(95% 0 0);
  --color-text-secondary: oklch(70% 0 0);
  --color-text-muted: oklch(50% 0 0);
}

/* 或者使用媒体查询 */
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg: oklch(12% 0 0);
    /* ... */
  }
}
```

使用语义化颜色：

```html
<body class="bg-bg text-text">
  <div class="bg-surface border border-border rounded-xl p-6">
    <h2 class="text-text font-bold">标题</h2>
    <p class="text-text-secondary">描述</p>
    <span class="text-text-muted text-sm">辅助信息</span>
  </div>
</body>
```

## 9.5 Vue 3 深色模式集成

```vue
<script setup>
import { ref, watchEffect } from 'vue'

const isDark = ref(
  localStorage.getItem('theme') === 'dark' ||
  (!localStorage.getItem('theme') && window.matchMedia('(prefers-color-scheme: dark)').matches)
)

watchEffect(() => {
  if (isDark.value) {
    document.documentElement.classList.add('dark')
    localStorage.setItem('theme', 'dark')
  } else {
    document.documentElement.classList.remove('dark')
    localStorage.setItem('theme', 'light')
  }
})
</script>

<template>
  <button
    @click="isDark = !isDark"
    class="p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800 transition-colors"
  >
    <span v-if="isDark">☀️</span>
    <span v-else>🌙</span>
  </button>
</template>
```

## 小结

- `dark:` 变体：为深色模式添加样式
- 两种策略：跟随系统（默认）或手动切换（class 模式）
- 推荐用 CSS 变量定义语义化颜色，深色模式只覆盖变量
- `@variant dark { .dark & { @slot; } }` 自定义深色模式触发方式
