## 练习 2：自定义变体

**目标**：使用 `@variant` 指令创建以下自定义变体：

1. `hocus`：hover 或 focus 时生效
2. `print`：打印时生效（隐藏不必要的元素）
3. `motion-safe`：用户没有设置减少动画时生效
4. `sidebar-open`：当 `.sidebar-open` 类存在于祖先元素时生效

```css
/* 在 main.css 中实现 */
@import "tailwindcss";

@variant hocus { /* ... */ }
@variant print { /* ... */ }
@variant motion-safe { /* ... */ }
@variant sidebar-open { /* ... */ }
```

<details>
<summary>参考答案</summary>

```css
@import "tailwindcss";

@variant hocus {
  &:hover, &:focus {
    @slot;
  }
}

@variant print {
  @media print {
    @slot;
  }
}

@variant motion-safe {
  @media (prefers-reduced-motion: no-preference) {
    @slot;
  }
}

@variant sidebar-open {
  .sidebar-open & {
    @slot;
  }
}
```

```html
<!-- 使用示例 -->
<button class="hocus:bg-blue-700 bg-blue-600 text-white px-4 py-2 rounded">
  悬停或聚焦变深
</button>

<div class="print:hidden">
  仅屏幕显示
</div>

<div class="motion-safe:animate-bounce motion-reduce:animate-none">
  尊重动画偏好
</div>

<div class="sidebar-open:ml-64 transition-all">
  侧边栏打开时右移
</div>
```
</details>

**目标**：实现一个"自适应卡片"组件，根据容器宽度自动调整布局：
- 容器宽度 < 400px：垂直布局（图片在上，文字在下）
- 容器宽度 >= 400px：水平布局（图片在左，文字在右）

```html
<!-- 在不同宽度的容器中使用同一个卡片组件 -->

  <ArticleCard />  <!-- 垂直布局 -->
</div>

<div class="w-full">
  <ArticleCard />  <!-- 水平布局 -->
</div>
```

<details>
<summary>参考答案</summary>
