# 基础练习题

## 练习 1：按钮组件

实现以下按钮变体，要求：
- 4 种颜色变体：primary（蓝）、success（绿）、warning（黄）、danger（红）
- 3 种尺寸：sm、md、lg
- 支持 disabled 状态（半透明 + 禁止点击）
- hover 时颜色加深，active 时轻微缩小

```html
<!-- 期望效果 -->
<button class="...">Primary</button>
<button class="...">Success</button>
<button class="...">Warning</button>
<button class="...">Danger</button>
```

<details>
<summary>参考答案</summary>

```html
<!-- Primary -->
<button class="bg-blue-600 hover:bg-blue-700 active:scale-95 text-white font-semibold px-4 py-2 rounded-lg transition-all disabled:opacity-50 disabled:cursor-not-allowed">
  Primary
</button>

<!-- Success -->
<button class="bg-green-600 hover:bg-green-700 active:scale-95 text-white font-semibold px-4 py-2 rounded-lg transition-all disabled:opacity-50 disabled:cursor-not-allowed">
  Success
</button>

<!-- Warning -->
<button class="bg-yellow-500 hover:bg-yellow-600 active:scale-95 text-white font-semibold px-4 py-2 rounded-lg transition-all disabled:opacity-50 disabled:cursor-not-allowed">
  Warning
</button>

<!-- Danger -->
<button class="bg-red-600 hover:bg-red-700 active:scale-95 text-white font-semibold px-4 py-2 rounded-lg transition-all disabled:opacity-50 disabled:cursor-not-allowed">
  Danger
</button>

<!-- Small -->
<button class="bg-blue-600 hover:bg-blue-700 text-white font-semibold px-3 py-1.5 text-sm rounded-md transition-colors">
  Small
</button>

<!-- Large -->
<button class="bg-blue-600 hover:bg-blue-700 text-white font-semibold px-6 py-3 text-lg rounded-xl transition-colors">
  Large
</button>
```
</details>

## 练习 3：表单验证样式

- 输入框错误状态：红色边框 + 红色提示文字
- 提交按钮：加载状态（旋转图标 + 禁用）

## 练习 5：深色模式切换

实现一个支持深色模式的卡片组件：
- 亮色：白色背景、深色文字、浅灰边框
- 深色：深灰背景、白色文字、深灰边框
- 添加一个切换按钮，点击切换模式

<details>
<summary>参考答案</summary>

```html
<!DOCTYPE html>
<html>
<head>
  <script src="https://cdn.tailwindcss.com/4"></script>
</head>
<body class="bg-gray-100 dark:bg-gray-950 min-h-screen flex items-center justify-center transition-colors">

  <div class="bg-white dark:bg-gray-800 border border-gray-200 dark:border-gray-700 rounded-2xl p-6 max-w-sm w-full mx-4 shadow-sm">
    <div class="flex items-center justify-between mb-4">
      <h3 class="font-bold text-gray-900 dark:text-white">深色模式卡片</h3>
      <button
        onclick="document.documentElement.classList.toggle('dark')"
        class="p-2 rounded-lg bg-gray-100 dark:bg-gray-700 text-gray-600 dark:text-gray-300 hover:bg-gray-200 dark:hover:bg-gray-600 transition-colors"
      >
        🌙
      </button>
    </div>
    <p class="text-gray-600 dark:text-gray-400 text-sm leading-relaxed">
      这是一个支持深色模式的卡片组件，点击右上角按钮切换主题。
    </p>
    <div class="mt-4 pt-4 border-t border-gray-100 dark:border-gray-700 flex gap-3">
      <button class="flex-1 bg-blue-600 hover:bg-blue-700 text-white text-sm font-medium py-2 rounded-lg transition-colors">
        确认
      </button>
      <button class="flex-1 bg-gray-100 dark:bg-gray-700 hover:bg-gray-200 dark:hover:bg-gray-600 text-gray-700 dark:text-gray-300 text-sm font-medium py-2 rounded-lg transition-colors">
        取消
      </button>
    </div>
  </div>

</body>
</html>
```
</details>
