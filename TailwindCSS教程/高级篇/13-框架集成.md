# 第十三章：框架集成

## 13.1 Vue 3 + TailwindCSS v4

### 安装

```bash
npm create vue@latest my-vue-app
cd my-vue-app
npm install tailwindcss @tailwindcss/vite
```

`vite.config.ts`：

```ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [vue(), tailwindcss()],
})
```

`src/assets/main.css`：

```css
@import "tailwindcss";

@theme {
  --font-sans: 'Inter', 'PingFang SC', sans-serif;
  --color-primary-600: oklch(52% 0.22 250);
}
```

`src/main.ts`：

```ts
import { createApp } from 'vue'
import './assets/main.css'
import App from './App.vue'

createApp(App).mount('#app')
```

### Vue 单文件组件最佳实践

```vue
<script setup lang="ts">
import { ref, computed } from 'vue'

  </ul>

```

### React 组件最佳实践

```tsx
// 使用 clsx + tailwind-merge 处理类名
import { clsx } from 'clsx'
import { twMerge } from 'tailwind-merge'

function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}

// 类型安全的变体组件
type BadgeVariant = 'default' | 'success' | 'warning' | 'error'

interface BadgeProps {
  variant?: BadgeVariant
  children: React.ReactNode
  className?: string
}

const variantStyles: Record<BadgeVariant, string> = {
  default: 'bg-gray-100 text-gray-700',
  success: 'bg-green-100 text-green-700',
  warning: 'bg-yellow-100 text-yellow-700',
  error: 'bg-red-100 text-red-700',
}

export function Badge({ variant = 'default', children, className }: BadgeProps) {
  return (
    <span className={cn(
      'inline-flex items-center px-2.5 py-0.5 rounded-full text-xs font-medium',
      variantStyles[variant],
      className
    )}>
      {children}
    </span>
  )
}
```

## 13.3 Next.js + TailwindCSS v4

### 安装

```bash
npx create-next-app@latest my-next-app --typescript --tailwind
# 选择 App Router
```

Next.js 15 已内置 TailwindCSS v4 支持，`create-next-app` 会自动配置。

手动配置：

```bash
npm install tailwindcss @tailwindcss/postcss postcss
```

`postcss.config.mjs`：

```js
const config = {
  plugins: {
    '@tailwindcss/postcss': {},
  },
}
export default config
```

`app/globals.css`：

```css
@import "tailwindcss";
```

### Next.js App Router 组件

```tsx
// app/components/Card.tsx
// Server Component（默认）
export function Card({ title, description }: { title: string; description: string }) {
  return (
    <div className="bg-white rounded-2xl shadow-sm border border-gray-100 p-6 hover:shadow-md transition-shadow">
      <h3 className="text-lg font-semibold text-gray-900 mb-2">{title}</h3>
      <p className="text-gray-600 text-sm leading-relaxed">{description}</p>
    </div>
  )
}

// app/components/ThemeToggle.tsx
'use client'  // Client Component
import { useState, useEffect } from 'react'

export function ThemeToggle() {
  const [isDark, setIsDark] = useState(false)

  useEffect(() => {
    const saved = localStorage.getItem('theme')
    const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches
    setIsDark(saved === 'dark' || (!saved && prefersDark))
  }, [])

  useEffect(() => {
    document.documentElement.classList.toggle('dark', isDark)
    localStorage.setItem('theme', isDark ? 'dark' : 'light')
  }, [isDark])

  return (
    <button
      onClick={() => setIsDark(!isDark)}
      className="p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800 transition-colors"
    >
      {isDark ? '☀️' : '🌙'}
    </button>
  )
}
```

## 13.4 Nuxt 3 + TailwindCSS v4

```bash
npx nuxi@latest init my-nuxt-app

```

`nuxt.config.ts`：

```ts
import tailwindcss from '@tailwindcss/vite'

export default defineNuxtConfig({
  vite: {
    plugins: [tailwindcss()],
  },
})
```

`assets/css/main.css`：

```css
@import "tailwindcss";
```

`nuxt.config.ts` 引入 CSS：

```ts
export default defineNuxtConfig({
  css: ['~/assets/css/main.css'],
  vite: {
    plugins: [tailwindcss()],
  },
})
```

## 13.5 Astro + TailwindCSS v4

```bash
npm create astro@latest my-astro-app
cd my-astro-app
npx astro add tailwind
```

Astro 的 `@astrojs/tailwind` 集成会自动配置 v4。

`astro.config.mjs`：

```js
import { defineConfig } from 'astro/config'
import tailwind from '@astrojs/tailwind'

export default defineConfig({
  integrations: [tailwind()],
})
```

npx shadcn@latest add button

## 小结

- Vue 3：`@tailwindcss/vite` 插件，单文件组件中直接使用工具类
- React：同上，配合 `clsx` + `tailwind-merge` 处理类名
- Next.js：`create-next-app` 内置支持，App Router 中区分 Server/Client 组件
- Nuxt 3：通过 `vite.plugins` 配置
- Astro：`@astrojs/tailwind` 集成
- shadcn/ui：基于 TailwindCSS 的高质量组件库
