# React 基础练习题

## 练习 1：创建个人资料卡片

创建一个个人资料卡片组件，显示用户信息。

**要求：**
- 显示用户头像、姓名、职位、邮箱
- 使用 Props 传递用户数据
- 添加适当的样式

**提示：**
```jsx
interface User {
  name: string;
  position: string;
  email: string;
  avatar: string;
}
```

## 学习建议

1. **循序渐进**：从简单的练习开始，逐步增加难度
2. **独立完成**：尽量自己完成练习，不要立即查看答案
3. **代码规范**：注意代码风格和命名规范
4. **测试代码**：确保代码功能正常
5. **总结反思**：完成练习后总结学到的知识
