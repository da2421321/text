# 练习 7：标签页组件

## 题目回顾

创建一个标签页组件。

**要求：**
- 支持多个标签页
- 点击切换标签页
- 支持自定义标签页内容
- 支持禁用某个标签页
- 添加过渡动画

---

## 参考答案

### Tabs.tsx

```tsx
import React, { useState } from 'react';
import './Tabs.css';

interface Tab {
  key: string;
  label: string;
  content: React.ReactNode;
  disabled?: boolean;
  icon?: React.ReactNode;
}

interface TabsProps {
  tabs: Tab[];
  defaultActiveKey?: string;
  onChange?: (key: string) => void;
  variant?: 'default' | 'card' | 'pills';
}

function Tabs({
  tabs,
  defaultActiveKey,
  onChange,
  variant = 'default',
}: TabsProps) {
  const [activeKey, setActiveKey] = useState(
    defaultActiveKey || tabs[0]?.key || ''
  );

  const handleTabClick = (key: string, disabled?: boolean) => {
    if (disabled) return;
    setActiveKey(key);
    onChange?.(key);
  };

  const activeTab = tabs.find(tab => tab.key === activeKey);

  return (
    <div className={`tabs tabs-${variant}`}>
      <div className="tabs-header" role="tablist">
        {tabs.map(tab => (
          <button
            key={tab.key}
            className={`tab-item ${activeKey === tab.key ? 'active' : ''} ${
              tab.disabled ? 'disabled' : ''
            }`}
            onClick={() => handleTabClick(tab.key, tab.disabled)}
            role="tab"
            aria-selected={activeKey === tab.key}
            aria-disabled={tab.disabled}
            tabIndex={tab.disabled ? -1 : 0}
          >
            {tab.icon && <span className="tab-icon">{tab.icon}</span>}
            {tab.label}
          </button>
        ))}
        <div
          className="tab-indicator"
          style={{
            width: `${100 / tabs.length}%`,
            transform: `translateX(${tabs.findIndex(t => t.key === activeKey) * 100}%)`,
          }}
        />
      </div>
      
      <div className="tabs-content" role="tabpanel">
        {activeTab?.content}
      </div>
    </div>
  );
}

export default Tabs;
```

### 使用示例

```tsx
import Tabs from './Tabs';

function App() {
  const tabs = [
    {
      key: 'home',
      label: '首页',
      icon: '🏠',
      content: <div>首页内容</div>,
    },
    {
      key: 'about',
      label: '关于',
      icon: 'ℹ️',
      content: <div>关于内容</div>,
    },
    {
      key: 'settings',
      label: '设置',
      icon: '⚙️',
      content: <div>设置内容</div>,
      disabled: true,
    },
  ];

  return (
    <Tabs
      tabs={tabs}
      defaultActiveKey="home"
      variant="pills"
      onChange={key => console.log('切换到:', key)}
    />
  );
}
```

## 知识点总结

1. **受控/非受控**：支持 defaultActiveKey 和 onChange
2. **滑动指示器**：使用 transform 实现平滑动画
3. **无障碍**：role、aria-selected 等属性
4. **多种变体**：通过 variant 支持不同样式
5. **禁用状态**：视觉反馈和交互阻止
