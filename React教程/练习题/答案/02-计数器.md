# 练习 2：计数器

## 题目回顾

创建一个计数器组件，包含增加、减少、重置功能。

**要求：**
- 使用 useState 管理计数状态
- 实现增加、减少、重置按钮
- 显示当前计数值
- 添加最小值和最大值限制

---

## 参考答案

### Counter.tsx

```tsx
import React, { useState } from 'react';
import './Counter.css';

interface CounterProps {
  initialValue?: number;
  min?: number;
  max?: number;
  step?: number;
}

function Counter({
  initialValue = 0,
  min = -100,
  max = 100,
  step = 1
}: CounterProps) {
  const [count, setCount] = useState(initialValue);

  const increment = () => {
    setCount(prev => Math.min(prev + step, max));
  };

  const decrement = () => {
    setCount(prev => Math.max(prev - step, min));
  };

  const reset = () => {
    setCount(initialValue);
  };

  const isAtMin = count <= min;
  const isAtMax = count >= max;

  return (
    <div className="counter">
      <div className="counter-display">
        <span className={`count ${count < 0 ? 'negative' : count > 0 ? 'positive' : ''}`}>
          {count}
        </span>
      </div>
      
      <div className="counter-controls">
        <button
          className="btn btn-decrement"
          onClick={decrement}
          disabled={isAtMin}
          aria-label="减少"
        >
          −
        </button>
        
        <button
          className="btn btn-reset"
          onClick={reset}
          aria-label="重置"
        >
          重置
        </button>
        
        <button
          className="btn btn-increment"
          onClick={increment}
          disabled={isAtMax}
          aria-label="增加"
        >
          +
        </button>
      </div>
      
      <div className="counter-info">
        <span>范围: {min} ~ {max}</span>
        <span>步长: {step}</span>
      </div>
    </div>
  );
}

export default Counter;
```

## 知识点总结

1. **useState Hook**：管理组件内部状态
2. **函数式更新**：使用 `prev => prev + 1` 确保状态更新正确
3. **可选 Props**：使用 `?` 和默认值处理可选属性
4. **条件禁用**：根据状态禁用按钮
5. **条件样式**：根据值动态添加 CSS 类名

## 进阶版本

```tsx
// 使用 useReducer 的版本
import React, { useReducer } from 'react';

type Action = 
  | { type: 'INCREMENT' }
  | { type: 'DECREMENT' }
  | { type: 'RESET' }
  | { type: 'SET'; payload: number };

interface State {
  count: number;
}

function reducer(state: State, action: Action): State {
  switch (action.type) {
    case 'INCREMENT':
      return { count: state.count + 1 };
    case 'DECREMENT':
      return { count: state.count - 1 };
    case 'RESET':
      return { count: 0 };
    case 'SET':
      return { count: action.payload };
    default:
      return state;
  }
}

function CounterWithReducer() {
  const [state, dispatch] = useReducer(reducer, { count: 0 });

  return (
    <div className="counter">
      <span>{state.count}</span>
      <button onClick={() => dispatch({ type: 'DECREMENT' })}>−</button>
      <button onClick={() => dispatch({ type: 'RESET' })}>重置</button>
      <button onClick={() => dispatch({ type: 'INCREMENT' })}>+</button>
    </div>
  );
}
```
