# 练习 10：折叠面板组件

## 题目回顾

创建一个折叠面板（Accordion）组件。

**要求：**
- 支持多个面板
- 点击展开/折叠
- 支持手风琴模式（一次只能展开一个）
- 支持默认展开的面板
- 平滑的动画效果

---

## 参考答案

### Accordion.tsx

```tsx
import React, { useState } from 'react';
import './Accordion.css';

interface AccordionItem {
  key: string;
  title: string;
  content: React.ReactNode;
  disabled?: boolean;
}

interface AccordionProps {
  items: AccordionItem[];
  defaultActiveKeys?: string[];
  accordion?: boolean; // 手风琴模式
  onChange?: (activeKeys: string[]) => void;
}

function Accordion({
  items,
  defaultActiveKeys = [],
  accordion = false,
  onChange,
}: AccordionProps) {
  const [activeKeys, setActiveKeys] = useState<string[]>(defaultActiveKeys);

  const togglePanel = (key: string, disabled?: boolean) => {
    if (disabled) return;

    let newActiveKeys: string[];

    if (accordion) {
      // 手风琴模式：只能展开一个
      newActiveKeys = activeKeys.includes(key) ? [] : [key];
    } else {
      // 普通模式：可以展开多个
      newActiveKeys = activeKeys.includes(key)
        ? activeKeys.filter(k => k !== key)
        : [...activeKeys, key];
    }

    setActiveKeys(newActiveKeys);
    onChange?.(newActiveKeys);
  };

  const isActive = (key: string) => activeKeys.includes(key);

  return (
    <div className="accordion">
      {items.map(item => (
        <div
          key={item.key}
          className={`accordion-item ${isActive(item.key) ? 'active' : ''} ${
            item.disabled ? 'disabled' : ''
          }`}
        >
          <button
            className="accordion-header"
            onClick={() => togglePanel(item.key, item.disabled)}
            aria-expanded={isActive(item.key)}
            aria-disabled={item.disabled}
          >
            <span className="accordion-title">{item.title}</span>
            <span className="accordion-icon">
              {isActive(item.key) ? '−' : '+'}
            </span>
          </button>
          
          <div className="accordion-content-wrapper">
            <div className="accordion-content">{item.content}</div>
          </div>
        </div>
      ))}
    </div>
  );
}

export default Accordion;
```

## 知识点总结

1. **手风琴模式**：一次只展开一个面板
2. **CSS Grid 动画**：grid-template-rows 实现高度动画
3. **状态管理**：数组存储多个展开的 key
4. **回调通知**：onChange 通知父组件状态变化
5. **无障碍**：aria-expanded 等属性
