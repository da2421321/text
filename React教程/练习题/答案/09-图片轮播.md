# 练习 9：图片轮播组件

## 题目回顾

创建一个图片轮播组件。

**要求：**
- 自动轮播
- 手动切换上一张/下一张
- 显示指示器
- 支持无限循环
- 鼠标悬停暂停

---

## 参考答案

### Carousel.tsx

```tsx
import React, { useState, useEffect, useCallback } from 'react';
import './Carousel.css';

interface CarouselProps {
  images: { src: string; alt: string; title?: string }[];
  autoPlay?: boolean;
  interval?: number;
  showArrows?: boolean;
  showDots?: boolean;
  pauseOnHover?: boolean;
}

function Carousel({
  images,
  autoPlay = true,
  interval = 3000,
  showArrows = true,
  showDots = true,
  pauseOnHover = true,
}: CarouselProps) {
  const [currentIndex, setCurrentIndex] = useState(0);
  const [isPaused, setIsPaused] = useState(false);
  const [isTransitioning, setIsTransitioning] = useState(false);

  const goToSlide = useCallback((index: number) => {
    if (isTransitioning) return;
    setIsTransitioning(true);
    
    let newIndex = index;
    if (index < 0) newIndex = images.length - 1;
    if (index >= images.length) newIndex = 0;
    
    setCurrentIndex(newIndex);
    setTimeout(() => setIsTransitioning(false), 500);
  }, [images.length, isTransitioning]);

  const goToPrev = () => goToSlide(currentIndex - 1);
  const goToNext = useCallback(() => goToSlide(currentIndex + 1), [currentIndex, goToSlide]);

  // 自动轮播
  useEffect(() => {
    if (!autoPlay || isPaused) return;

    const timer = setInterval(goToNext, interval);
    return () => clearInterval(timer);
  }, [autoPlay, isPaused, interval, goToNext]);

  // 键盘导航
  useEffect(() => {
    const handleKeyDown = (e: KeyboardEvent) => {
      if (e.key === 'ArrowLeft') goToPrev();
      if (e.key === 'ArrowRight') goToNext();
    };

    window.addEventListener('keydown', handleKeyDown);
    return () => window.removeEventListener('keydown', handleKeyDown);
  }, [goToNext]);

  return (
    <div
      className="carousel"
      onMouseEnter={() => pauseOnHover && setIsPaused(true)}
      onMouseLeave={() => pauseOnHover && setIsPaused(false)}
    >
      <div className="carousel-track">
        {images.map((image, index) => (
          <div
            key={index}
            className={`carousel-slide ${index === currentIndex ? 'active' : ''}`}
          >
            <img src={image.src} alt={image.alt} />
            {image.title && <div className="slide-title">{image.title}</div>}
          </div>
        ))}
      </div>

      {showArrows && (
        <>
          <button className="carousel-arrow prev" onClick={goToPrev}>
            ‹
          </button>
          <button className="carousel-arrow next" onClick={goToNext}>
            ›
          </button>
        </>
      )}

      {showDots && (
        <div className="carousel-dots">
          {images.map((_, index) => (
            <button
              key={index}
              className={`dot ${index === currentIndex ? 'active' : ''}`}
              onClick={() => goToSlide(index)}
              aria-label={`跳转到第 ${index + 1} 张`}
            />
          ))}
        </div>
      )}

      <div className="carousel-counter">
        {currentIndex + 1} / {images.length}
      </div>
    </div>
  );
}

export default Carousel;
```

### 使用示例

```tsx
import Carousel from './Carousel';

const images = [
  { src: '/img1.jpg', alt: '图片1', title: '美丽的风景' },
  { src: '/img2.jpg', alt: '图片2', title: '城市夜景' },
  { src: '/img3.jpg', alt: '图片3', title: '自然风光' },
];

function App() {
  return <Carousel images={images} autoPlay interval={4000} />;
}
```

## 知识点总结

1. **自动轮播**：setInterval + 清理函数
2. **暂停功能**：onMouseEnter/Leave 控制
3. **无限循环**：边界索引处理
4. **过渡动画**：CSS opacity 过渡
5. **键盘导航**：方向键控制切换
