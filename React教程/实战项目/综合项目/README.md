# 综合项目练习

本目录包含 5 个完整的 React 综合项目，每个项目都是一个可运行的小型应用。

## 项目列表

### 1. [博客系统](./blog-system/README.md)

完整的博客应用，包含文章发布、分类、搜索和评论功能。

- **技术栈**：React + TypeScript + Context API + React Router
- **功能**：文章列表、详情页、分类筛选、搜索、评论系统

### 2. [任务管理系统](./task-manager/README.md)

功能完整的任务管理应用，支持任务的创建、编辑、删除和状态管理。

- **技术栈**：React + TypeScript + Redux Toolkit
- **功能**：任务 CRUD、状态管理、优先级、截止日期、筛选排序

### 3. [天气预报应用](./weather-app/README.md)

实时天气查询应用，集成第三方天气 API。

- **技术栈**：React + TypeScript + Axios + 天气 API
- **功能**：城市搜索、当日天气、未来预报、收藏城市

### 4. [音乐播放器](./music-player/README.md)

功能丰富的音乐播放器，支持播放控制和播放列表管理。

- **技术栈**：React + TypeScript + HTML5 Audio API
- **功能**：播放控制、播放列表、进度条、音量控制、循环模式

### 5. [社交网络应用](./social-app/README.md)

简化版的社交网络应用，包含动态发布、评论和点赞功能。

- **技术栈**：React + TypeScript + Context API
- **功能**：发布动态、点赞评论、用户关注、消息通知

## 运行项目

每个项目目录下都有详细的运行说明：

```bash
# 进入项目目录
cd 项目名称

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

完成这些项目后，你将掌握：

- ✅ 完整的 React 应用开发流程
- ✅ 状态管理的多种方案
- ✅ 路由和导航的实现
- ✅ 表单和用户输入处理
- ✅ API 数据获取和展示
- ✅ 组件的封装和复用
- ✅ TypeScript 在 React 中的应用
