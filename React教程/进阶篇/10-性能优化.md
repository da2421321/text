# 性能优化

## React 性能优化的重要性

随着应用规模的增长，性能优化变得越来越重要。良好的性能可以提升用户体验，减少资源消耗。

## 使用 React.memo

React.memo 是一个高阶组件，用于记忆组件的渲染结果。

### 基本用法

```jsx
import React from 'react';

const ExpensiveComponent = React.memo(function ExpensiveComponent({ data }) {
  console.log('ExpensiveComponent 渲染');
  return <div>{data}</div>;
});

function App() {
  const [count, setCount] = React.useState(0);
  const data = '静态数据';

  return (
    <div>
      <button onClick={() => setCount(count + 1)}>
        计数: {count}
      </button>
      <ExpensiveComponent data={data} />
    </div>
  );
}
```

### 自定义比较函数

```jsx
const MyComponent = React.memo(function MyComponent({ user }) {
  return <div>{user.name}</div>;
}, (prevProps, nextProps) => {
  return prevProps.user.id === nextProps.user.id;
});
```

## 使用 useMemo

useMemo 用于缓存计算结果，避免不必要的重新计算。

### 基本用法

```jsx
import { useState, useMemo } from 'react';

function ExpensiveCalculation({ numbers }) {
  const [count, setCount] = useState(0);

  const sum = useMemo(() => {
    console.log('计算中...');
    return numbers.reduce((a, b) => a + b, 0);
  }, [numbers]);

  return (
    <div>
      <p>总和: {sum}</p>
      <p>计数: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        增加
      </button>
    </div>
  );
}
```

### 复杂计算优化

```jsx
function FilteredList({ items, filter }) {
  const filteredItems = useMemo(() => {
    console.log('过滤中...');
    return items.filter(item =>
      item.name.toLowerCase().includes(filter.toLowerCase())
    );
  }, [items, filter]);

  return (
    <ul>
      {filteredItems.map(item => (
        <li key={item.id}>{item.name}</li>
      ))}
    </ul>
  );
}
```

## 使用 useCallback

useCallback 用于缓存函数，避免子组件不必要的重新渲染。

### 基本用法

```jsx
import { useState, useCallback } from 'react';

function Parent() {
  const [count, setCount] = useState(0);

  const handleClick = useCallback(() => {
    console.log('按钮被点击');
  }, []);

  return (
    <div>
      <button onClick={() => setCount(count + 1)}>
        计数: {count}
      </button>
      <Child onClick={handleClick} />
    </div>
  );
}

const Child = React.memo(function Child({ onClick }) {
  console.log('Child 渲染');
  return <button onClick={onClick}>子组件按钮</button>;
});
```

### 依赖数组

```jsx
function Parent() {
  const [count, setCount] = useState(0);
  const [name, setName] = useState('');

  const handleClick = useCallback(() => {
    console.log('按钮被点击', count);
  }, [count]);

  return (
    <div>
      <input value={name} onChange={(e) => setName(e.target.value)} />
      <button onClick={() => setCount(count + 1)}>
        计数: {count}
      </button>
      <Child onClick={handleClick} />
    </div>
  );
}
```

## 代码分割

使用 React.lazy 和 Suspense 进行代码分割。

### 基本用法

```jsx
import { lazy, Suspense } from 'react';

const LazyComponent = lazy(() => import('./LazyComponent'));

function App() {
  return (
    <div>
      <Suspense fallback={<div>加载中...</div>}>
        <LazyComponent />
      </Suspense>
    </div>
  );
}
```

### 路由级别的代码分割

```jsx
import { lazy, Suspense } from 'react';
import { BrowserRouter, Routes, Route } from 'react-router-dom';

const Home = lazy(() => import('./Home'));
const About = lazy(() => import('./About'));
const Contact = lazy(() => import('./Contact'));

function App() {
  return (
    <BrowserRouter>
      <Suspense fallback={<div>加载中...</div>}>
        <Routes>
          <Route path="/" element={<Home />} />
          <Route path="/about" element={<About />} />
          <Route path="/contact" element={<Contact />} />
        </Routes>
      </Suspense>
    </BrowserRouter>
  );
}
```

## 避免不必要的渲染

### 使用 key

```jsx
function TodoList({ todos }) {
  return (
    <ul>
      {todos.map(todo => (
        <li key={todo.id}>{todo.text}</li>
      ))}
    </ul>
  );
}
```

### 避免内联函数

```jsx
// 不好
function App() {
  return (
    <button onClick={() => console.log('点击')}>
      点击
    </button>
  );
}

// 好
function App() {
  const handleClick = () => console.log('点击');

  return <button onClick={handleClick}>点击</button>;
}
```

### 避免内联对象

```jsx
// 不好
function App() {
  return (
    <div style={{ color: 'red', fontSize: '16px' }}>
      内容
    </div>
  );
}

// 好
function App() {
  const style = { color: 'red', fontSize: '16px' };

  return <div style={style}>内容</div>;
}
```

## 使用 Transition

使用 useTransition 标记非紧急的更新。

```jsx
import { useState, useTransition } from 'react';

function SearchInput() {
  const [isPending, startTransition] = useTransition();
  const [input, setInput] = useState('');
  const [list, setList] = useState([]);

  const handleChange = (e) => {
    const value = e.target.value;
    setInput(value);

    startTransition(() => {
      const filtered = largeList.filter(item =>
        item.toLowerCase().includes(value.toLowerCase())
      );
      setList(filtered);
    });
  };

  return (
    <div>
      <input value={input} onChange={handleChange} />
      {isPending && <p>加载中...</p>}
      <ul>
        {list.map(item => <li key={item}>{item}</li>)}
      </ul>
    </div>
  );
}
```

## 优化图片加载

### 懒加载

```jsx
import { useState, useRef, useEffect } from 'react';

function LazyImage({ src, alt }) {
  const [isVisible, setIsVisible] = useState(false);
  const imgRef = useRef(null);

  useEffect(() => {
    const observer = new IntersectionObserver(([entry]) => {
      if (entry.isIntersecting) {
        setIsVisible(true);
        observer.disconnect();
      }
    });

    if (imgRef.current) {
      observer.observe(imgRef.current);
    }

    return () => observer.disconnect();
  }, []);

  return (
    <img
      ref={imgRef}
      src={isVisible ? src : ''}
      alt={alt}
      style={{ opacity: isVisible ? 1 : 0, transition: 'opacity 0.3s' }}
    />
  );
}
```

## 实战示例：优化后的列表组件

```jsx
import { useState, useMemo, useCallback } from 'react';
import { FixedSizeList as List } from 'react-window';

const Row = React.memo(function Row({ index, style, data }) {
  const { items, onItemClick } = data;

  return (
    <div style={style} onClick={() => onItemClick(items[index].id)}>
      {items[index].name}
    </div>
  );
});

function OptimizedList() {
  const [items, setItems] = useState(
    Array.from({ length: 10000 }, (_, i) => ({
      id: i,
      name: `项目 ${i}`
    }))
  );
  const [filter, setFilter] = useState('');

  const filteredItems = useMemo(() => {
    return items.filter(item =>
      item.name.toLowerCase().includes(filter.toLowerCase())
    );
  }, [items, filter]);

  const handleItemClick = useCallback((id) => {
    console.log('点击项目:', id);
  }, []);

  const itemData = useMemo(() => ({
    items: filteredItems,
    onItemClick: handleItemClick
  }), [filteredItems, handleItemClick]);

  return (
    <div>
      <input
        value={filter}
        onChange={(e) => setFilter(e.target.value)}
        placeholder="搜索..."
      />
      <List
        height={400}
        itemCount={filteredItems.length}
        itemSize={35}
        width={300}
        itemData={itemData}
      >
        {Row}
      </List>
    </div>
  );
}

export default OptimizedList;
```

## 性能监控

### React DevTools Profiler

使用 React DevTools Profiler 分析组件渲染性能。

### 自定义性能监控

```jsx
import { useEffect } from 'react';

function usePerformanceMonitor(componentName) {
  useEffect(() => {
    const start = performance.now();

    return () => {
      const end = performance.now();
      console.log(`${componentName} 渲染耗时: ${end - start}ms`);
    };
  });
}

function MyComponent() {
  usePerformanceMonitor('MyComponent');

  return <div>内容</div>;
}
```

## 最佳实践

1. **避免不必要的渲染**：使用 React.memo、useMemo、useCallback
2. **代码分割**：使用 React.lazy 和 Suspense
3. **虚拟化长列表**：使用 react-window 或 react-virtualized
4. **优化图片**：懒加载、响应式图片
5. **使用 Web Workers**：将计算密集型任务放到后台
6. **监控性能**：使用 React DevTools Profiler 和自定义监控
