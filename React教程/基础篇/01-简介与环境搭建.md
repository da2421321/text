# React 简介与环境搭建

## 什么是 React？

React 是一个由 Facebook 开发的用于构建用户界面的 JavaScript 库。它主要用于构建单页应用程序（SPA），通过组件化的方式让开发者可以更高效地构建复杂的 UI。

### React 的核心特点

1. **组件化**：将 UI 拆分成独立、可复用的组件
2. **声明式**：通过声明式编程方式描述 UI，使代码更易理解和维护
3. **虚拟 DOM**：使用虚拟 DOM 提高渲染性能
4. **单向数据流**：数据从父组件流向子组件，使状态管理更清晰
5. **生态系统丰富**：拥有庞大的第三方库和工具支持

## 环境搭建

### 方式一：使用 Create React App（推荐初学者）

Create React App 是官方提供的脚手架工具，可以快速搭建 React 开发环境。

#### 安装 Node.js

首先需要安装 Node.js，React 开发需要 Node.js 14.0 或更高版本。

1. 访问 [Node.js 官网](https://nodejs.org/)
2. 下载并安装 LTS（长期支持）版本
3. 安装完成后，在终端验证安装：

```bash
node -v
npm -v
```

#### 创建 React 项目

```bash
# 使用 npx 创建新项目（推荐）
npx create-react-app my-react-app

# 或者使用 npm init
npm init react-app my-react-app

# 进入项目目录
cd my-react-app

# 启动开发服务器
npm start
```

项目创建后，浏览器会自动打开 `http://localhost:3000`，你将看到 React 的欢迎页面。

#### 项目结构

```
my-react-app/
├── node_modules/          # 依赖包
├── public/                # 静态资源
│   ├── index.html         # HTML 模板
│   └── favicon.ico        # 网站图标
├── src/                   # 源代码
│   ├── App.js             # 根组件
│   ├── App.css            # 根组件样式
│   ├── index.js           # 应用入口
│   └── index.css          # 全局样式
├── .gitignore             # Git 忽略文件
├── package.json           # 项目配置
└── README.md              # 项目说明
```

### 方式二：使用 Vite（更快的构建工具）

Vite 是新一代的前端构建工具，启动速度更快，热更新更高效。

```bash
# 创建 Vite + React 项目
npm create vite@latest my-react-app -- --template react

# 进入项目目录
cd my-react-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 方式三：使用 Next.js（React 框架）

Next.js 是一个功能完整的 React 框架，支持服务端渲染、静态站点生成等功能。

```bash
# 创建 Next.js 项目
npx create-next-app@latest my-next-app

# 进入项目目录
cd my-next-app

# 启动开发服务器
npm run dev
```

## 第一个 React 组件

让我们创建一个简单的 React 组件：

```jsx
// src/App.js
import React from 'react';
import './App.css';

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <h1>欢迎学习 React</h1>
        <p>这是我的第一个 React 应用</p>
      </header>
    </div>
  );
}

export default App;
```

```css
/* src/App.css */
.App {
  text-align: center;
}

.App-header {
  background-color: #282c34;
  min-height: 100vh;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  color: white;
}
```

## 学习建议

1. **掌握 JavaScript 基础**：在学习 React 之前，确保你熟悉 ES6+ 语法
2. **理解组件概念**：组件是 React 的核心，要理解组件的创建和使用
3. **实践为主**：多写代码，多动手实践
4. **阅读官方文档**：React 官方文档是最好的学习资源
5. **参与社区**：加入 React 社区，与其他开发者交流

## 常见问题

### Q: Create React App 和 Vite 应该选择哪个？

A: 对于初学者，建议使用 Create React App，它更稳定，文档丰富。对于追求开发体验和构建速度的开发者，推荐使用 Vite。

### Q: 需要学习 TypeScript 吗？

A: TypeScript 可以提高代码质量和开发效率，建议在掌握 React 基础后学习 TypeScript。

### Q: React 和 Vue 哪个更好？

A: 两者都是优秀的前端框架，选择哪个主要取决于个人喜好和项目需求。React 的生态更丰富，Vue 的学习曲线更平缓。
