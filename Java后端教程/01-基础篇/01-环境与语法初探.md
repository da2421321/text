## 一、什么是 Java？为什么要学它？

### 1.1 白话解释

想象你开了一家餐厅：
- **前端**（HTML/CSS/JS）= 餐厅的装修、菜单设计、服务员和客人的互动界面
- **后端**（Java）= 后厨系统：接收订单、查库存、做菜、算账、出餐
- **数据库**（MySQL）= 仓库：存放食材（数据）

Java 就是用来写"后厨系统"的语言，它的特点：
- **稳定可靠**：银行、电商、政府系统大量使用
- **生态丰富**：Spring Boot 框架让开发变得很快
- **就业面广**：Java 后端是最主流的后端方向之一

### 1.2 Java vs JavaScript 的区别

很多新手会搞混这两个，它们完全不同：

| 对比项 | Java | JavaScript |
|--------|------|------------|
| 运行环境 | JVM（Java虚拟机） | 浏览器 / Node.js |
| 类型系统 | 强类型（必须声明类型） | 弱类型（变量类型灵活） |
| 主要用途 | 后端服务、安卓开发 | 前端页面、Node.js后端 |
| 编译方式 | 先编译再运行 | 解释执行 |

## 二、环境搭建（手把手教学）

### 2.1 安装 JDK 17

**JDK 是什么？** JDK = Java Development Kit，Java 开发工具包。就像你要做菜需要刀具和锅，写 Java 代码需要 JDK。

**第一步：下载 JDK**

去 https://adoptium.net/ 下载 JDK 17（LTS 长期支持版）：
1. 选择你的操作系统（Windows / macOS / Linux）
2. 选择 Architecture: x64
3. 下载 `.msi`（Windows）或 `.pkg`（macOS）安装包
4. 双击安装，一路下一步即可

**第二步：验证安装是否成功**

打开终端（Windows: PowerShell，macOS: Terminal），输入：

```bash
# 查看 Java 运行时版本
java -version

# 查看 Java 编译器版本
javac -version
```

如果看到类似输出就说明安装成功了：
```
openjdk version "17.0.x" 2024-xx-xx
OpenJDK Runtime Environment Temurin-17.0.x+x (build 17.0.x+x)
```

**常见问题**：
- ❌ 提示 `java 不是内部命令`：说明环境变量没配好，需要把 JDK 的 `bin` 目录加到系统 PATH 中
- ❌ 版本号不是 17：可能装了多个版本，确认 PATH 中 JDK 17 排在最前面

### 2.2 安装 Maven

**Maven 是什么？** Maven 就是 Java 世界的 npm/pnpm，用来：
- 管理项目依赖（类似 `npm install`）
- 编译和打包项目（类似 `npm run build`）
- 运行测试（类似 `npm test`）

**安装步骤**：

1. 去 https://maven.apache.org/download.cgi 下载最新版
2. 解压到一个目录（如 `C:\Program Files\Maven`）
3. 配置环境变量：
   - 新建 `MAVEN_HOME` = Maven 解压路径
   - 在 `PATH` 中添加 `%MAVEN_HOME%\bin`

**验证**：
```bash
mvn -version
```

看到版本信息就说明安装成功。

> 💡 **小提示**：如果你用 IntelliJ IDEA，它自带 Maven，可以跳过手动安装。

### 2.3 安装 IntelliJ IDEA

**强烈推荐** 使用 IntelliJ IDEA 来写 Java（就像前端用 VS Code 一样）：

1. 去 https://www.jetbrains.com/idea/ 下载 **Community 版**（免费）
2. 安装时勾选 "Add to PATH" 和 ".java 文件关联"
3. 首次打开时，选择你喜欢的主题（Dark/Light）

## 三、第一个 Java 程序：Hello World

### 3.1 创建项目

在 IDEA 中：
1. `File → New → Project`
2. 选择 `Java`，SDK 选择 `JDK 17`
3. 项目名填 `hello-java`，点击 `Create`

### 3.2 编写代码

在 `src` 目录下创建文件 `HelloWorld.java`：

```java
// 这是 Java 的 "包" 声明，类似于文件的归属地址
// 如果文件直接在 src 根目录下，可以不写 package

// 这是一个类（Class），Java 里所有代码都必须写在类里面
// 类名必须和文件名一模一样（包括大小写）
public class HelloWorld {

    // 这是 main 方法，程序的入口点
    // 就像 Node.js 里的 app.listen()，程序从这里开始执行
    // public: 公开的（谁都能访问）
    // static: 静态的（不需要创建对象就能调用）
    // void: 没有返回值
    // String[] args: 命令行参数（先不管它）
    public static void main(String[] args) {

        // System.out.println() 就是 Java 的 console.log()
        System.out.println("Hello, World!");
        System.out.println("我的第一个 Java 程序！");
    }
}
```

### 3.3 运行程序

**方法一（IDEA 中）**：点击 `main` 方法左边的绿色三角形 ▶，选择 `Run`

**方法二（命令行）**：
```bash
# 第一步：编译（把 .java 文件编译成 .class 字节码文件）
javac HelloWorld.java

# 第二步：运行（JVM 执行字节码）
java HelloWorld
```

**输出**：
```
Hello, World!
我的第一个 Java 程序！
```

### 3.4 Java 程序运行流程图解

```
你写的代码(.java)  →  编译器(javac)  →  字节码(.class)  →  JVM运行  →  看到结果
                                                              ↑
                                                     这就是 Java 跨平台的秘密
                                                     同一份字节码可以在 Windows/Mac/Linux 运行
```

## 四、变量与数据类型

### 4.1 什么是变量？

**变量就是一个贴了标签的盒子**，用来存放数据。

```
┌──────────┐
│  name    │ ← 标签（变量名）
│ "张三"   │ ← 内容（值）
│ String   │ ← 盒子类型（数据类型）
└──────────┘
```

### 4.2 Java 的基本数据类型（8种）

Java 和 JavaScript 最大的不同：**Java 必须声明变量的类型**。

```java
public class DataTypes {
    public static void main(String[] args) {

        // ========== 整数类型 ==========
        byte  b = 127;           // 1字节，范围 -128 ~ 127（很少用）
        short s = 32767;         // 2字节，范围 -32768 ~ 32767（很少用）
        int   i = 2147483647;    // 4字节，最常用的整数类型 ⭐
        long  l = 9999999999L;   // 8字节，超大整数，注意末尾要加 L

        // ========== 小数类型 ==========
        float  f = 3.14f;        // 4字节，注意末尾要加 f
        double d = 3.14159;      // 8字节，最常用的小数类型 ⭐

        // ========== 字符类型 ==========
        char c = 'A';            // 单个字符，用单引号

        // ========== 布尔类型 ==========
        boolean flag = true;     // 只有 true 和 false 两个值

        // ========== 字符串（不是基本类型，是对象类型）==========
        String name = "张三";    // 字符串用双引号，注意 S 大写 ⭐

        // 打印所有变量
        System.out.println("整数: " + i);
        System.out.println("小数: " + d);
        System.out.println("字符: " + c);
        System.out.println("布尔: " + flag);
        System.out.println("字符串: " + name);
    }
}
```

### 4.3 新手必知：Java vs JavaScript 类型对比

| JavaScript | Java | 区别说明 |
|-----------|------|---------|
| `let x = 10` | `int x = 10;` | Java 必须写类型 |
| `let name = "张三"` | `String name = "张三";` | Java 的 String 大写 S |
| `let flag = true` | `boolean flag = true;` | Java 用 boolean 不是 bool |
| `let price = 9.9` | `double price = 9.9;` | Java 小数默认用 double |
| `const PI = 3.14` | `final double PI = 3.14;` | Java 用 final 声明常量 |

### 4.4 var 关键字（Java 10+）

Java 10 以后也可以用 `var` 自动推断类型（类似 TypeScript 的类型推断）：

```java
var age = 25;          // 编译器自动推断为 int
var name = "张三";     // 编译器自动推断为 String
var price = 19.9;      // 编译器自动推断为 double

// 但是！var 只能用在局部变量上，不能用在类的成员变量上
// 而且声明时必须赋值，否则编译器无法推断类型
```

### 4.5 类型转换

```java
public class TypeConversion {
    public static void main(String[] args) {

        // 自动转换（小类型 → 大类型，安全，自动完成）
        int a = 100;
        double b = a;     // int → double，自动转换，b = 100.0
        System.out.println(b); // 100.0

        // 强制转换（大类型 → 小类型，可能丢失精度，必须手动）
        double c = 3.99;
        int d = (int) c;  // double → int，小数部分直接砍掉！
        System.out.println(d); // 3（不是四舍五入，是直接截断！）

        // 字符串转数字
        String numStr = "123";
        int num = Integer.parseInt(numStr);       // 字符串 → int
        double dNum = Double.parseDouble("3.14"); // 字符串 → double

        // 数字转字符串
        String str1 = String.valueOf(123);    // 方式一
        String str2 = 123 + "";               // 方式二（利用字符串拼接）
        String str3 = Integer.toString(123);  // 方式三
    }
}
```

**常见坑**：
- `Integer.parseInt("abc")` 会抛出 `NumberFormatException` 异常
- `(int) 3.99` 结果是 3，不是 4！Java 的强制转换是截断不是四舍五入

## 五、运算符

### 5.1 算术运算符

```java
public class Operators {
    public static void main(String[] args) {
        int a = 10, b = 3;

        System.out.println(a + b);   // 13  加法
        System.out.println(a - b);   // 7   减法
        System.out.println(a * b);   // 30  乘法
        System.out.println(a / b);   // 3   整数除法！（不是3.33）⚠️
        System.out.println(a % b);   // 1   取余数

        // 想要小数结果？至少一个操作数要是 double
        System.out.println(a / (double) b);  // 3.3333...
        System.out.println(10.0 / 3);        // 3.3333...
    }
}
```

**超级常见坑**：`10 / 3` 在 Java 中结果是 `3` 不是 `3.33`！因为两个 int 相除结果还是 int。

### 5.2 比较运算符和逻辑运算符

```java
// 比较运算符（和 JavaScript 几乎一样）
System.out.println(5 > 3);    // true
System.out.println(5 < 3);    // false
System.out.println(5 >= 5);   // true
System.out.println(5 == 5);   // true
System.out.println(5 != 3);   // true

// 逻辑运算符
System.out.println(true && false);  // false  与（两个都是true才是true）
System.out.println(true || false);  // true   或（有一个true就是true）
System.out.println(!true);          // false  非（取反）
```

**注意**：Java 中比较字符串不能用 `==`，要用 `.equals()`：

```java
String s1 = "hello";
String s2 = "hello";
String s3 = new String("hello");

System.out.println(s1 == s2);       // true（但这只是碰巧，因为字符串常量池）
System.out.println(s1 == s3);       // false！⚠️ == 比较的是内存地址
System.out.println(s1.equals(s3));  // true ✅ equals 比较的是内容
```

## 六、条件判断

### 6.1 if-else

```java
public class IfDemo {
    public static void main(String[] args) {
        int score = 85;

        if (score >= 90) {
            System.out.println("优秀");
        } else if (score >= 80) {
            System.out.println("良好");    // ← 会执行这个
        } else if (score >= 60) {
            System.out.println("及格");
        } else {
            System.out.println("不及格");
        }
    }
}
```

### 6.2 switch（传统写法 vs 新写法）

```java
// 传统 switch（注意每个 case 后面要 break！）
String day = "Monday";

switch (day) {
    case "Monday":
        System.out.println("星期一，新的开始");
        break;       // 如果忘记 break，会继续执行下面的 case！⚠️
    case "Friday":
        System.out.println("星期五，快到周末了");
        break;
    default:
        System.out.println("普通的一天");
}

// Java 14+ 新写法（推荐，不需要 break）
switch (day) {
    case "Monday"  -> System.out.println("星期一，新的开始");
    case "Friday"  -> System.out.println("星期五，快到周末了");
    default        -> System.out.println("普通的一天");
}
```

### 6.3 三元运算符

```java
int age = 20;
String result = (age >= 18) ? "成年人" : "未成年人";
System.out.println(result); // 成年人
```

## 七、循环

### 7.1 for 循环

```java
// 基本 for 循环：打印 1 到 5
// 拆解：int i = 1（初始值）; i <= 5（条件）; i++（每次循环后 i 加 1）
for (int i = 1; i <= 5; i++) {
    System.out.println("第 " + i + " 次循环");
}
// 输出：
// 第 1 次循环
// 第 2 次循环
// 第 3 次循环
// 第 4 次循环
// 第 5 次循环
```

### 7.2 while 和 do-while

```java
// while：先判断条件，再执行
int count = 0;
while (count < 3) {
    System.out.println("count = " + count);
    count++;
}
// 输出: count = 0, count = 1, count = 2

// do-while：先执行一次，再判断条件（至少执行一次）
int num = 10;
do {
    System.out.println("num = " + num); // 即使条件不满足，也会执行一次
    num++;
} while (num < 5);
// 输出: num = 10（执行了一次就停了）
```

### 7.3 for-each（增强 for 循环）

```java
// 遍历数组或集合时，推荐用 for-each，更简洁
String[] fruits = {"苹果", "香蕉", "橙子"};

for (String fruit : fruits) {
    System.out.println("我喜欢吃 " + fruit);
}
// 输出：
// 我喜欢吃 苹果
// 我喜欢吃 香蕉
// 我喜欢吃 橙子
```

### 7.4 break 和 continue

```java
// break：直接跳出整个循环
for (int i = 1; i <= 10; i++) {
    if (i == 5) break;    // i 等于 5 时，整个循环结束
    System.out.print(i + " ");
}
// 输出: 1 2 3 4

// continue：跳过本次循环，继续下一次
for (int i = 1; i <= 5; i++) {
    if (i == 3) continue; // 跳过 i=3 这次
    System.out.print(i + " ");
}
// 输出: 1 2 4 5
```

## 八、数组

### 8.1 数组基础

```java
public class ArrayDemo {
    public static void main(String[] args) {

        // 方式一：声明并初始化（知道内容时用）
        int[] scores = {90, 85, 78, 92, 88};

        // 方式二：只声明大小（稍后赋值）
        String[] names = new String[3];
        names[0] = "张三";
        names[1] = "李四";
        names[2] = "王五";

        // 访问元素（下标从 0 开始）
        System.out.println(scores[0]);   // 90（第一个元素）
        System.out.println(scores[4]);   // 88（第五个元素）
        // scores[5] 会报 ArrayIndexOutOfBoundsException！⚠️

        // 数组长度
        System.out.println("成绩个数: " + scores.length); // 5

        // 遍历数组
        for (int i = 0; i < scores.length; i++) {
            System.out.println("第 " + (i + 1) + " 个成绩: " + scores[i]);
        }

        // 用 for-each 更简洁
        for (int score : scores) {
            System.out.println("成绩: " + score);
        }
    }
}
```

### 8.2 数组 vs ArrayList

Java 数组长度固定，不能动态增减。如果需要动态数组，用 `ArrayList`（下一章详细讲）：

```java
import java.util.ArrayList;

ArrayList<String> list = new ArrayList<>();
list.add("苹果");    // 添加
list.add("香蕉");
list.get(0);         // 获取第一个元素 → "苹果"
list.size();         // 长度 → 2
list.remove(0);      // 删除第一个元素
```

## 九、方法（函数）

### 9.1 什么是方法？

方法就是 **把一段代码打包成一个可重复调用的功能块**，相当于 JavaScript 中的函数。

```java
public class MethodDemo {

    // 定义一个方法：计算两个数的和
    // public: 公开访问
    // static: 静态方法（可以直接通过类名调用）
    // int: 返回值类型（这个方法会返回一个整数）
    // add: 方法名
    // (int a, int b): 参数列表
    public static int add(int a, int b) {
        return a + b;
    }

    // 没有返回值的方法，用 void
    public static void sayHello(String name) {
        System.out.println("你好，" + name + "！");
    }

    public static void main(String[] args) {
        // 调用方法
        int result = add(3, 5);
        System.out.println("3 + 5 = " + result); // 3 + 5 = 8

        sayHello("张三"); // 你好，张三！
        sayHello("李四"); // 你好，李四！
    }
}
```

### 9.2 方法重载（Overloading）

**同一个类中可以有多个同名方法**，只要参数不同就行（参数个数不同 或 参数类型不同）：

```java
public class Calculator {

    // 两个 int 相加
    public static int add(int a, int b) {
        return a + b;
    }

    // 三个 int 相加（参数个数不同）
    public static int add(int a, int b, int c) {
        return a + b + c;
    }

    // 两个 double 相加（参数类型不同）
    public static double add(double a, double b) {
        return a + b;
    }

    public static void main(String[] args) {
        System.out.println(add(1, 2));       // 调用第一个: 3
        System.out.println(add(1, 2, 3));    // 调用第二个: 6
        System.out.println(add(1.5, 2.5));   // 调用第三个: 4.0
    }
}
```

**白话解释**：方法重载就像一个人有多种技能 —— 你叫他"做饭"，他会根据你给的食材不同，做出不同的菜。

## 十、字符串操作（超高频使用）

```java
public class StringDemo {
    public static void main(String[] args) {
        String str = "Hello, Java!";

        // 常用操作
        System.out.println(str.length());           // 12（长度）
        System.out.println(str.charAt(0));           // 'H'（第一个字符）
        System.out.println(str.substring(7));        // "Java!"（从第7位截取到末尾）
        System.out.println(str.substring(0, 5));     // "Hello"（从第0位截取到第5位，不包含第5位）
        System.out.println(str.toUpperCase());       // "HELLO, JAVA!"
        System.out.println(str.toLowerCase());       // "hello, java!"
        System.out.println(str.contains("Java"));    // true（是否包含）
        System.out.println(str.indexOf("Java"));     // 7（第一次出现的位置）
        System.out.println(str.replace("Java", "World")); // "Hello, World!"
        System.out.println(str.trim());              // 去除首尾空格
        System.out.println(str.isEmpty());           // false（是否为空字符串）
        System.out.println(str.startsWith("Hello")); // true
        System.out.println(str.endsWith("!"));       // true

        // 字符串拼接
        String name = "张三";
        int age = 25;

        // 方式一：+ 号拼接（简单场景用）
        String info1 = "姓名: " + name + ", 年龄: " + age;

        // 方式二：String.format()（格式化字符串）
        String info2 = String.format("姓名: %s, 年龄: %d", name, age);

        // 方式三：StringBuilder（大量拼接时用，性能最好）⭐
        StringBuilder sb = new StringBuilder();
        sb.append("姓名: ").append(name);
        sb.append(", 年龄: ").append(age);
        String info3 = sb.toString();

        System.out.println(info1);
        System.out.println(info2);
        System.out.println(info3);
    }
}
```

**常见坑**：
- 字符串比较一定用 `.equals()`，不要用 `==`
- 大量字符串拼接（比如循环中拼接）要用 `StringBuilder`，否则性能极差

## 十一、包与访问修饰符

### 11.1 包（Package）

**包就是文件夹**，用来组织代码、避免类名冲突。

```
src/
├── com/
│   └── example/
│       ├── model/          ← 放数据模型类
│       │   └── User.java
│       ├── service/        ← 放业务逻辑类
│       │   └── UserService.java
```

### 11.2 访问修饰符

控制"谁能访问这个类/方法/变量"：

| 修饰符 | 同一个类 | 同一个包 | 子类 | 任何地方 | 使用频率 |
|--------|---------|---------|------|---------|---------|
| `public` | ✅ | ✅ | ✅ | ✅ | ⭐⭐⭐ 最常用 |
| `protected` | ✅ | ✅ | ✅ | ❌ | ⭐ 偶尔用 |
| 默认（不写） | ✅ | ✅ | ❌ | ❌ | ⭐ 偶尔用 |
| `private` | ✅ | ❌ | ❌ | ❌ | ⭐⭐⭐ 最常用 |

**白话总结**：
- `public`：谁都能用 → 对外暴露的接口
- `private`：只有自己能用 → 内部实现细节
- 新手阶段先记住这两个就够了

```java
import java.util.Scanner;

public class TempConverter {

        System.out.print("请输入摄氏温度: ");
```

### 练习3：猜数字游戏

```java
import java.util.Random;
import java.util.Scanner;

public class GuessNumber {
    public static void main(String[] args) {
        Random random = new Random();
        int answer = random.nextInt(100) + 1; // 生成 1~100 的随机数
        Scanner scanner = new Scanner(System.in);
        int attempts = 0;

        System.out.println("=== 猜数字游戏 ===");
        System.out.println("我想了一个 1~100 之间的数字，请你来猜！");

        while (true) {
            System.out.print("请输入你的猜测: ");
            int guess = scanner.nextInt();
            attempts++;

            if (guess < answer) {
                System.out.println("太小了！再试试");
            } else if (guess > answer) {
                System.out.println("太大了！再试试");
            } else {
                System.out.println("恭喜你猜对了！答案就是 " + answer);
                System.out.println("你一共猜了 " + attempts + " 次");
                break;
            }
        }

        scanner.close();
    }
}
```
