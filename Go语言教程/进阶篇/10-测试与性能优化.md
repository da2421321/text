# 测试与性能优化

## 测试与性能优化工具方法详解

### testing.T 测试对象方法

`testing.T` 是单元测试的核心对象，提供了丰富的方法来控制测试行为和报告测试结果。

| 方法 | 作用 | 使用场景 |
|------|------|----------|
| `Error(args ...interface{})` | 记录错误信息，但继续执行测试 | 记录多个错误信息 |
| `Errorf(format string, args ...interface{})` | 格式化记录错误信息，继续执行 | 输出格式化的错误信息 |
| `Fail()` | 标记测试失败，但继续执行 | 需要标记失败但继续检查其他条件 |
| `FailNow()` | 标记测试失败，立即停止当前测试 | 致命错误，无法继续测试 |
| `Fatal(args ...interface{})` | 等同于 `Log()` + `FailNow()` | 致命错误需要立即停止 |
| `Fatalf(format string, args ...interface{})` | 格式化输出后立即停止测试 | 格式化输出致命错误 |
| `Failed()` | 判断测试是否已标记为失败 | 检查测试状态 |
| `Log(args ...interface{})` | 记录日志信息（仅在测试失败或 -v 时输出） | 记录调试信息 |
| `Logf(format string, args ...interface{})` | 格式化记录日志信息 | 格式化输出日志 |
| `Name()` | 返回当前测试的名称 | 获取测试名称用于日志 |
| `Skip(args ...interface{})` | 跳过当前测试，记录原因 | 某些条件下跳过测试 |
| `Skipf(format string, args ...interface{})` | 格式化输出跳过原因 | 格式化输出跳过信息 |
| `SkipNow()` | 立即跳过当前测试 | 快速跳过测试 |
| `Skipped()` | 判断测试是否被跳过 | 检查测试是否跳过 |
| `Helper()` | 标记函数为测试辅助函数，错误报告会跳过此函数 | 编写测试辅助函数 |
| `Cleanup(f func())` | 注册清理函数，测试结束后执行（LIFO 顺序） | 资源清理、恢复状态 |
| `Parallel()` | 标记测试可并行执行 | 加速独立测试执行 |
| `Run(name string, f func(t *testing.T))` | 运行子测试 | 组织相关测试用例 |
| `Setenv(key, value string)` | 设置环境变量（Go 1.17+），自动清理 | 测试环境变量相关逻辑 |
| `TempDir()` | 创建临时目录并在测试结束后自动清理 | 需要文件系统操作的测试 |
| `Deadline()` | 返回测试的超时时间 | 检查测试时间限制 |

### testing.B 基准测试对象方法

`testing.B` 用于性能基准测试，继承了 `testing.T` 的大部分方法，并添加了性能测试专用方法。

| 方法 | 作用 | 使用场景 |
|------|------|----------|
| `ResetTimer()` | 重置计时器，忽略之前的初始化时间 | 初始化完成后重置计时 |
| `StartTimer()` | 开始计时 | 手动控制计时开始 |
| `StopTimer()` | 停止计时 | 暂停计时（如进行数据准备） |
| `N` | 基准测试循环次数（字段，非方法） | 在循环中使用：`for i := 0; i < b.N; i++` |
| `ReportAllocs()` | 报告内存分配统计信息 | 分析内存使用情况 |
| `ReportMetric(n float64, unit string)` | 报告自定义性能指标 | 添加自定义性能指标 |
| `SetBytes(n int64)` | 设置每次迭代处理的字节数 | 计算吞吐量（MB/s） |
| `SetParallelism(p int)` | 设置并行测试的并发度 | 控制并行基准测试 |
| `RunParallel(body func(*testing.PB))` | 并行运行基准测试 | 测试并发性能 |

### testing.PB 并行基准测试对象

| 方法 | 作用 | 使用场景 |
|------|------|----------|
| `Next()` | 返回是否应继续执行迭代 | 并行基准测试循环条件 |

### testing.M 主测试对象

### 常用测试模式

#### 1. 表驱动测试模式
```go
func TestFunction(t *testing.T) {
    tests := []struct {
        name     string
        input    Type
        expected Type
    }{
        {"case1", input1, expected1},
        {"case2", input2, expected2},
    }
    
    for _, tt := range tests {
        t.Run(tt.name, func(t *testing.T) {
            result := Function(tt.input)
            if result != tt.expected {
                t.Errorf("got %v, want %v", result, tt.expected)
            }
        })
    }
}
```

#### 2. 子测试模式
```go
func TestMain_SubTests(t *testing.T) {
    t.Run("subtest1", func(t *testing.T) {
        // 测试逻辑
    })
    
    t.Run("subtest2", func(t *testing.T) {
        // 测试逻辑
    })
}
```

#### 3. 并行测试模式
```go
func TestParallel(t *testing.T) {
    t.Parallel() // 标记为并行
    // 测试逻辑
}
```

#### 4. 基准测试模式
```go
func BenchmarkFunction(b *testing.B) {
    // 初始化
    b.ResetTimer() // 重置计时器
    
    for i := 0; i < b.N; i++ {
        Function() // 被测试的函数
    }
}
```

#### 5. 并行基准测试模式
```go
func BenchmarkFunctionParallel(b *testing.B) {
    b.RunParallel(func(pb *testing.PB) {
        for pb.Next() {
            Function()
        }
    })
}
```

#### 6. 测试辅助函数模式
```go
func assertEqual(t *testing.T, got, want interface{}) {
    t.Helper() // 标记为辅助函数
    if got != want {
        t.Errorf("got %v, want %v", got, want)
    }
}
```

#### 7. 测试清理模式
```go
func TestWithCleanup(t *testing.T) {
    resource := setupResource()
    t.Cleanup(func() {
        resource.Close() // 测试结束后自动调用
    })
    // 测试逻辑
}
```

#### 8. TestMain 模式
```go
func TestMain(m *testing.M) {
    // 测试前的全局设置
    setup()
    
    // 运行测试
    code := m.Run()
    
    // 测试后的全局清理
    teardown()
    
    os.Exit(code)
}
```

### 测试最佳实践

1. **命名规范**
   - 测试函数：`TestXxx`
   - 基准测试：`BenchmarkXxx`
   - 示例函数：`ExampleXxx`

2. **测试组织**
   - 一个源文件对应一个测试文件（`xxx.go` → `xxx_test.go`）
   - 相关测试用例使用子测试组织
   - 使用表驱动测试处理多个测试用例

3. **错误报告**
   - 使用 `t.Errorf` 提供清晰的错误信息
   - 包含实际值和期望值
   - 使用 `t.Helper()` 标记辅助函数

4. **性能测试**
   - 使用 `b.ResetTimer()` 排除初始化时间
   - 使用 `b.StopTimer()` 和 `b.StartTimer()` 控制计时
   - 使用 `-benchmem` 分析内存分配

5. **并发安全**
   - 使用 `-race` 标志检测数据竞争
   - 并行测试注意共享资源
   - 使用 `t.Parallel()` 加速测试

```go
// string_utils.go
package stringutils

import "strings"

// Reverse 字符串反转
func Reverse(s string) string {
    runes := []rune(s)
    for i, j := 0, len(runes)-1; i < j; i, j = i+1, j-1 {
        runes[i], runes[j] = runes[j], runes[i]
    }
    return string(runes)
}

// IsPalindrome 判断字符串是否为回文
func IsPalindrome(s string) bool {
    s = strings.ToLower(s)
    return s == Reverse(s)
}
```

### 子测试和并行测试

```go
// math_utils.go
package mathutils

import "math"

// Fibonacci 计算斐波那契数列第n项
func Fibonacci(n int) int {
    if n <= 1 {
        return n
    }
    a, b := 0, 1
    for i := 2; i <= n; i++ {
        a, b = b, a+b
    }
    return b
}

// Factorial 计算阶乘
func Factorial(n int) int {
    if n <= 1 {
        return 1
    }
    result := 1
    for i := 2; i <= n; i++ {
        result *= i
    }
    return result
}

// IsPrime 判断是否为质数
func IsPrime(n int) bool {
    if n < 2 {
        return false
    }
    if n == 2 {
        return true
    }
    if n%2 == 0 {
        return false
    }
    for i := 3; i*i <= n; i += 2 {
        if n%i == 0 {
            return false
        }
    }
    return true
}
```

```go
// profiler_example.go
package main

import (
    "net/http"
    _ "net/http/pprof"
    "time"
)

func main() {
    // 启动pprof服务器
    go func() {

    workLoad()

func workLoad() {
    ticker := time.NewTicker(100 * time.Millisecond)
    defer ticker.Stop()
    
    for {
        select {
        case <-ticker.C:
            // 模拟CPU密集型任务
            cpuIntensiveTask()
            
            // 模拟内存分配
            memoryIntensiveTask()
        }
    }
}

func cpuIntensiveTask() {
    result := 0
    for i := 0; i < 1000000; i++ {
        result += i
    }
}

func memoryIntensiveTask() {
    // 分配大量内存
    data := make([]byte, 1024*1024) // 1MB
    for i := range data {
        data[i] = byte(i % 256)
    }
}
```

## 代码优化技巧

### 减少内存分配

```go
// optimization_examples.go
package main

import (
    "fmt"
    "strings"
    "time"
)

// 优化前：频繁的字符串拼接
func inefficientStringConcat(strs []string) string {
    var result string
    for _, s := range strs {
        result += s
    }
    return result
}

// 优化后：使用strings.Builder
func efficientStringConcat(strs []string) string {
    var builder strings.Builder
    for _, s := range strs {
        builder.WriteString(s)
    }
    return builder.String()
}

// 优化前：频繁的切片操作
func inefficientSliceOperation() []int {
    var result []int
    for i := 0; i < 1000; i++ {
        result = append(result, i)
    }
    return result
}

// 优化后：预分配切片容量
func efficientSliceOperation() []int {
    result := make([]int, 0, 1000)
    for i := 0; i < 1000; i++ {
        result = append(result, i)
    }
    return result
}

// 优化前：频繁的map分配
func inefficientMapOperation() map[string]int {
    result := make(map[string]int)
    for i := 0; i < 1000; i++ {
        result[fmt.Sprintf("key%d", i)] = i
    }
    return result
}

// 优化后：预分配map容量
func efficientMapOperation() map[string]int {
    result := make(map[string]int, 1000)
    for i := 0; i < 1000; i++ {
        result[fmt.Sprintf("key%d", i)] = i
    }
    return result
}

func main() {
    // 测试字符串拼接性能
    strs := []string{"Hello", " ", "World", "!", " ", "Go", " ", "Language"}
    
    start := time.Now()
    result1 := inefficientStringConcat(strs)
    duration1 := time.Since(start)
    
    start = time.Now()
    result2 := efficientStringConcat(strs)
```

```go
// concurrent_optimization.go
package main

import (
    "fmt"
    "sync"
    "time"
)

// 串行处理
func serialProcessing(data []int) []int {
    result := make([]int, len(data))
    for i, v := range data {
        result[i] = processItem(v)
    }
    return result
}

// 并发处理
func concurrentProcessing(data []int) []int {
    result := make([]int, len(data))
    var wg sync.WaitGroup
    
    // 使用goroutine处理每个项目
    for i, v := range data {
        wg.Add(1)
        go func(index, value int) {
            defer wg.Done()
            result[index] = processItem(value)
        }(i, v)
    }
    
    wg.Wait()
    return result
}

// 工作池模式
func workerPoolProcessing(data []int, numWorkers int) []int {
    jobs := make(chan job, len(data))
    results := make(chan result, len(data))
    
    // 启动工作池
    var wg sync.WaitGroup
    for i := 0; i < numWorkers; i++ {
        wg.Add(1)
        go worker(jobs, results, &wg)
    }
    
    // 发送任务
    for i, v := range data {
        jobs <- job{index: i, value: v}
    }
    close(jobs)
    
    // 等待所有工作完成
    wg.Wait()
    close(results)
    
    // 收集结果
    result := make([]int, len(data))
    for res := range results {
        result[res.index] = res.value
    }
    
    return result
}

type job struct {
    index int
    value int
}

type result struct {
    index int
    value int
}

func worker(jobs <-chan job, results chan<- result, wg *sync.WaitGroup) {
    defer wg.Done()
    for job := range jobs {
        results <- result{index: job.index, value: processItem(job.value)}
    }
}

func processItem(value int) int {
    // 模拟耗时操作
    time.Sleep(10 * time.Millisecond)
    return value * value
}

func main() {
    data := make([]int, 100)
    for i := range data {
        data[i] = i
    }
    
    // 测试串行处理
    start := time.Now()
    result1 := serialProcessing(data)
    duration1 := time.Since(start)
    
    // 测试并发处理
    start = time.Now()
    result2 := concurrentProcessing(data)
    duration2 := time.Since(start)
    
    // 测试工作池处理
    start = time.Now()
    result3 := workerPoolProcessing(data, 10)
    duration3 := time.Since(start)
    
    fmt.Printf("串行处理耗时: %v\n", duration1)
    fmt.Printf("并发处理耗时: %v\n", duration2)
    fmt.Printf("工作池处理耗时: %v\n", duration3)
    
    // 验证结果一致性
    fmt.Printf("结果一致性: %t\n", isEqual(result1, result2) && isEqual(result2, result3))
}

func isEqual(a, b []int) bool {
    if len(a) != len(b) {
        return false
    }
    for i := range a {
        if a[i] != b[i] {
            return false
        }
    }
    return true
}
```

```go
package main

import (
    "errors"
    "fmt"
    "sync"
    "time"
)

type User struct {
    ID       int
    Username string
    Email    string
    Password string
    Created  time.Time
}

type UserManager struct {
    users map[string]*User
    mu    sync.RWMutex
    nextID int
}

func NewUserManager() *UserManager {
    return &UserManager{
        users: make(map[string]*User),
        nextID: 1,
    }
}

func (um *UserManager) Register(username, email, password string) (*User, error) {
    um.mu.Lock()
    defer um.mu.Unlock()
    
    // 检查用户名是否已存在
    if _, exists := um.users[username]; exists {
        return nil, errors.New("用户名已存在")
    }
    
    // 检查邮箱是否已存在
    for _, user := range um.users {
        if user.Email == email {
            return nil, errors.New("邮箱已被使用")
        }
    }
    
    // 创建新用户
    user := &User{
        ID:       um.nextID,
        Username: username,
        Email:    email,
        Password: password,
        Created:  time.Now(),
    }
    
    um.users[username] = user
    um.nextID++
    
    return user, nil
}

func (um *UserManager) Login(username, password string) (*User, error) {
    um.mu.RLock()
    defer um.mu.RUnlock()
    
    user, exists := um.users[username]
    if !exists {
        return nil, errors.New("用户不存在")
    }
    
    if user.Password != password {
        return nil, errors.New("密码错误")
    }
    
    return user, nil
}

func (um *UserManager) GetUser(username string) (*User, error) {
    um.mu.RLock()
    defer um.mu.RUnlock()
    
    user, exists := um.users[username]
    if !exists {
        return nil, errors.New("用户不存在")
    }
    
    return user, nil
}

func (um *UserManager) UpdateUser(username, email, password string) error {
    um.mu.Lock()
    defer um.mu.Unlock()
    
    user, exists := um.users[username]
    if !exists {
        return errors.New("用户不存在")
    }
    
    // 检查邮箱是否被其他用户使用
    for uname, u := range um.users {
        if uname != username && u.Email == email {
            return errors.New("邮箱已被其他用户使用")
        }
    }
    
    user.Email = email
    if password != "" {
        user.Password = password
    }
    
    return nil
}

func (um *UserManager) DeleteUser(username string) error {
    um.mu.Lock()
    defer um.mu.Unlock()
    
    if _, exists := um.users[username]; !exists {
        return errors.New("用户不存在")
    }
    
    delete(um.users, username)
    return nil
}

func (um *UserManager) GetUserCount() int {
    um.mu.RLock()
    defer um.mu.RUnlock()
    return len(um.users)
}
```
