# 第二章：变量、常量与数据类型

> 本章目标：学会在 Go 中存储和使用数据。理解变量、常量以及 Go 的基本数据类型。

## 2.1 什么是变量？

**变量**就是一个"容器"，用来存放数据。你可以把它想象成一个贴了标签的盒子：

```
┌──────────┐
│   "张三"  │  ← 盒子里的内容（值）
└──────────┘
   name        ← 盒子上的标签（变量名）
```

变量有三个要素：
- **变量名**：给这个容器起的名字，比如 `name`、`age`
- **数据类型**：这个容器能放什么类型的东西，比如文字、数字
- **值**：容器里实际存放的内容

## 2.2 声明变量

Go 中声明变量有三种方式：

### 方式一：var 关键字（完整写法）

```go
var name string = "张三"
var age int = 25
var height float64 = 1.75
var isStudent bool = true
```

格式：`var 变量名 类型 = 值`

### 方式二：var 关键字（类型推断）

Go 足够聪明，可以根据你给的值自动判断类型：

```go
var name = "张三"     // 自动推断为 string
var age = 25          // 自动推断为 int
var height = 1.75     // 自动推断为 float64
var isStudent = true  // 自动推断为 bool
```

### 方式三：短变量声明 `:=`（最常用）

```go
name := "张三"
age := 25
height := 1.75
isStudent := true
```

`:=` 是 Go 特有的语法，意思是"声明一个新变量并赋值"。这是日常开发中**最常用**的方式。

> **注意**：`:=` 只能在函数内部使用，不能在函数外面（全局）使用。

### 三种方式对比

```go
package main

import "fmt"

// 全局变量必须用 var，不能用 :=
var globalName string = "全局变量"

func main() {
    // 方式一：完整写法
    var a string = "Hello"

    // 方式二：类型推断
    var b = "World"

    // 方式三：短变量声明（推荐）
    c := "Go"

    fmt.Println(a, b, c)
}
```

### 同时声明多个变量

```go
// 方式一：逐个声明
var name string = "张三"
var age int = 25

// 方式二：批量声明（推荐，更整洁）
var (
    name   string = "张三"
    age    int    = 25
    height float64 = 1.75
)

// 方式三：短变量同时声明多个
name, age := "张三", 25
```

## 2.3 变量的使用规则

### 规则一：变量声明后必须使用

```go
func main() {
    name := "张三"
    age := 25       // 编译报错！声明了但没有使用
    fmt.Println(name)
}
```

Go 不允许声明了变量却不使用，这是为了保持代码整洁。

### 规则二：变量可以重新赋值，但不能改变类型

```go
age := 25
age = 30       // 可以：重新赋值
age = "三十"   // 报错：不能把字符串赋给 int 类型的变量
```

注意赋值用 `=`，不是 `:=`。`:=` 是"声明+赋值"，`=` 是单纯的"赋值"。

### 规则三：同一作用域内不能重复声明

```go
name := "张三"
name := "李四"   // 报错！name 已经声明过了

name = "李四"    // 正确：用 = 重新赋值
```

### 规则四：变量有零值（默认值）

如果声明了变量但没有赋值，Go 会自动给一个**零值**：

```go
var name string   // 零值是 ""（空字符串）
var age int       // 零值是 0
var height float64 // 零值是 0.0
var isOK bool     // 零值是 false

fmt.Println(name)   // 输出：（空）
fmt.Println(age)    // 输出：0
fmt.Println(height) // 输出：0
fmt.Println(isOK)   // 输出：false
```

这一点很重要：Go 中不存在"未初始化"的变量，每个变量都有明确的初始值。

## 2.4 常量

**常量**是一旦定义就不能修改的值。用 `const` 关键字声明：

```go
const pi = 3.14159
const appName = "我的应用"
const maxRetry = 3

// pi = 3.14  // 报错！常量不能修改
```

### 批量声明常量

```go
const (
    StatusOK    = 200
    StatusError = 500
    AppName     = "MyApp"
)
```

### iota：自动递增的常量

`iota` 是 Go 的一个特殊常量生成器，从 0 开始，每行自动加 1：

```go
const (
    Sunday    = iota  // 0
    Monday            // 1（自动 iota，省略了 = iota）
    Tuesday           // 2
    Wednesday         // 3
    Thursday          // 4
    Friday            // 5
    Saturday          // 6
)

fmt.Println(Wednesday) // 输出：3
```

实际应用举例 — 定义用户角色：

```go
const (
    RoleGuest  = iota // 0：游客
    RoleUser          // 1：普通用户
    RoleAdmin         // 2：管理员
    RoleSuper         // 3：超级管理员
)
```

## 2.5 基本数据类型

Go 有以下几种基本数据类型：

### 整数类型（存放整数）

| 类型 | 范围 | 说明 |
|------|------|------|
| `int` | 取决于系统（32位或64位） | **最常用**，一般用这个就够了 |
| `int8` | -128 ~ 127 | 占 1 字节 |
| `int16` | -32768 ~ 32767 | 占 2 字节 |
| `int32` | 约 -21 亿 ~ 21 亿 | 占 4 字节 |
| `int64` | 极大范围 | 占 8 字节 |
| `uint` | 0 ~ 正整数最大值 | 无符号（只有正数） |

日常开发中，**用 `int` 就够了**，不需要纠结用哪种。

```go
var count int = 100
var bigNumber int64 = 9999999999
var smallNumber int8 = 127
```

### 浮点数类型（存放小数）

| 类型 | 精度 | 说明 |
|------|------|------|
| `float32` | 约 7 位有效数字 | 精度较低 |
| `float64` | 约 15 位有效数字 | **最常用**，推荐使用 |

```go
price := 19.99          // 默认是 float64
var pi float64 = 3.14159265358979
var rate float32 = 0.618
```

> 小提示：如果你不知道该用 `float32` 还是 `float64`，就用 `float64`。

### 布尔类型（true / false）

布尔类型只有两个值：`true`（真）和 `false`（假）。

```go
isAdult := true
isVIP := false

fmt.Println(isAdult)  // 输出：true
fmt.Println(!isAdult) // 输出：false（! 表示取反）
```

布尔值常用于条件判断（下一章会详细讲）。

### 字符串类型

字符串就是一段文字，用**双引号**包裹：

```go
name := "张三"
greeting := "Hello, World!"
empty := ""  // 空字符串
```

**重要区别**：
- 双引号 `"..."` 表示字符串
- 反引号 `` `...` `` 表示原始字符串（可以包含换行，不转义）
- 单引号 `'...'` 表示单个字符（rune），不是字符串

```go
// 普通字符串（支持转义符）
path := "C:\\Users\\Documents"
multiLine := "第一行\n第二行"

// 原始字符串（所见即所得，适合写路径、正则等）
path2 := `C:\Users\Documents`
multiLine2 := `第一行
第二行`

// 单个字符
letter := 'A'       // 类型是 int32 (rune)
chinese := '中'     // 中文也是一个 rune
```

### 字符串拼接

```go
firstName := "张"
lastName := "三"
fullName := firstName + lastName  // "张三"

fmt.Println("你好，" + fullName + "！")
// 输出：你好，张三！
```

## 2.6 类型转换

Go 是**强类型语言**，不同类型之间不能直接运算，必须显式转换：

```go
var a int = 10
var b float64 = 3.14

// c := a + b  // 报错！int 和 float64 不能直接相加

c := float64(a) + b  // 正确：先把 a 转成 float64
fmt.Println(c)        // 输出：13.14

d := a + int(b)       // 也可以：把 b 转成 int（会丢失小数部分）
fmt.Println(d)        // 输出：13（3.14 变成了 3）
```

### 常见类型转换

```go
// 整数之间的转换
var small int8 = 100
var big int64 = int64(small)

// 整数转浮点数
age := 25
ageFloat := float64(age)  // 25.0

// 浮点数转整数（丢失小数部分，不是四舍五入）
price := 19.99
priceInt := int(price)  // 19

// 数字和字符串之间的转换需要用 strconv 包
import "strconv"

numStr := strconv.Itoa(42)         // int → string: "42"
num, _ := strconv.Atoi("42")      // string → int: 42
f, _ := strconv.ParseFloat("3.14", 64)  // string → float64: 3.14
```

## 2.7 完整示例：个人信息卡片

把本章学到的知识综合运用：

```go
package main

import "fmt"

const schoolName = "海牛学院"

func main() {
    // 用短变量声明创建变量
    name := "张三"
    age := 22
    height := 1.78
    isGraduated := false
    hobby := "编程"

    // 打印个人信息
    fmt.Println("===== 个人信息卡片 =====")
    fmt.Println("姓名：", name)
    fmt.Println("年龄：", age, "岁")
    fmt.Println("身高：", height, "米")
    fmt.Println("学校：", schoolName)
    fmt.Println("爱好：", hobby)

    if isGraduated {
        fmt.Println("状态：已毕业")
    } else {
        fmt.Println("状态：在读")
    }

    // 计算出生年份
    birthYear := 2026 - age
    fmt.Println("出生年份：", birthYear)
}
```

输出：
```
===== 个人信息卡片 =====
姓名： 张三
年龄： 22 岁
身高： 1.78 米
学校： 海牛学院
爱好： 编程
状态： 在读
出生年份： 2004
```

## 2.8 本章小结

| 概念 | 说明 |
|------|------|
| 变量声明 | `var name string = "值"` 或 `name := "值"` |
| 常量声明 | `const pi = 3.14` |
| 整数 | `int`（最常用）、`int8`、`int16`、`int32`、`int64` |
| 浮点数 | `float64`（最常用）、`float32` |
| 布尔 | `bool`：`true` 或 `false` |
| 字符串 | `string`：用双引号包裹 |
| 零值 | 每个类型都有默认值：`0`、`""`、`false` |
| 类型转换 | `float64(num)`、`int(decimal)` |
| iota | 自动递增的常量生成器 |

## 2.9 练习题

1. **声明变量**：用三种不同方式声明一个字符串变量 `city`，值为你所在的城市
2. **常量练习**：用 `const` 定义一组颜色常量（红=0，绿=1，蓝=2），使用 `iota`
3. **类型转换**：声明一个 `int` 变量值为 100，一个 `float64` 变量值为 3.14，计算它们的和（需要类型转换）
4. **零值探索**：声明 `int`、`string`、`bool`、`float64` 四个变量但不赋值，打印它们的零值
5. **综合练习**：写一个程序，声明商品名称（string）、价格（float64）、数量（int），计算并输出总价
