# 第四章：函数详解

> 本章目标：掌握 Go 函数的定义和使用，包括参数、返回值、多返回值、匿名函数、闭包和 defer。

## 4.1 什么是函数？

**函数**就是把一段代码"打包"起来，给它起个名字，以后可以反复使用。

就像做饭：你把"煮饭"这件事的步骤打包成一个"做法"，以后每次想吃饭，按这个做法来就行，不用每次重新想步骤。

```
函数 = 一个可以反复使用的代码块
```

## 4.2 定义和调用函数

### 最简单的函数

```go
package main

import "fmt"

// 定义函数：用 func 关键字
func sayHello() {
    fmt.Println("你好！")
}

func main() {
    sayHello()  // 调用函数
    sayHello()  // 可以反复调用
    sayHello()
}
```

输出：
```
你好！
你好！
你好！
```

### 函数的定义格式

```go
func 函数名(参数列表) 返回值类型 {
    // 函数体
    return 返回值
}
```

## 4.3 函数参数

参数就是你传递给函数的数据，让函数根据不同的输入做不同的事情。

### 一个参数

```go
func greet(name string) {
    fmt.Println("你好，" + name + "！")
}

func main() {
    greet("张三")  // 输出：你好，张三！
    greet("李四")  // 输出：你好，李四！
}
```

### 多个参数

```go
func add(a int, b int) {
    fmt.Println(a + b)
}

func main() {
    add(3, 5)   // 输出：8
    add(10, 20) // 输出：30
}
```

### 参数类型简写

如果多个参数类型相同，可以只写一次类型：

```go
// 完整写法
func add(a int, b int) { ... }

// 简写
func add(a, b int) { ... }

// 多种类型混合
func userInfo(name string, age, score int) { ... }
```

## 4.4 返回值

函数可以"返回"一个结果给调用者。

### 基本返回值

```go
func add(a, b int) int {   // 返回值类型写在参数列表后面
    return a + b
}

func main() {
    result := add(3, 5)
    fmt.Println(result)     // 输出：8

    // 也可以直接使用返回值
    fmt.Println(add(10, 20)) // 输出：30
}
```

### 多返回值（Go 的一大特色）

Go 函数可以返回多个值，这在其他语言中很少见：

```go
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, fmt.Errorf("除数不能为零")
    }
    return a / b, nil
}

func main() {
    result, err := divide(10, 3)
    if err != nil {
        fmt.Println("错误：", err)
    } else {
        fmt.Printf("结果：%.2f\n", result) // 输出：结果：3.33
    }

    result2, err2 := divide(10, 0)
    if err2 != nil {
        fmt.Println("错误：", err2) // 输出：错误：除数不能为零
    } else {
        fmt.Println(result2)
    }
}
```

这是 Go 中最常见的模式：**函数返回"结果 + 错误"**，调用者必须检查错误。

### 命名返回值

可以给返回值起名字，代码更清晰：

```go
func rectangle(width, height float64) (area float64, perimeter float64) {
    area = width * height
    perimeter = 2 * (width + height)
    return  // 可以直接 return，自动返回命名的变量
}

func main() {
    a, p := rectangle(5, 3)
    fmt.Println("面积：", a)    // 输出：面积：15
    fmt.Println("周长：", p)    // 输出：周长：16
}
```

### 忽略不需要的返回值

如果你不需要某个返回值，用 `_` 忽略它：

```go
area, _ := rectangle(5, 3)  // 只要面积，不要周长
fmt.Println("面积：", area)
```

## 4.5 可变参数

当你不确定会传入多少个参数时，可以使用 `...`：

```go
// nums 是一个 int 切片
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

func main() {
    fmt.Println(sum(1, 2))          // 3
    fmt.Println(sum(1, 2, 3, 4, 5)) // 15
    fmt.Println(sum())              // 0

    // 也可以把切片展开传入
    numbers := []int{10, 20, 30}
    fmt.Println(sum(numbers...))    // 60
}
```

## 4.6 匿名函数

**匿名函数**就是没有名字的函数。可以直接定义并使用。

### 直接调用

```go
func main() {
    // 定义匿名函数并立即调用
    result := func(a, b int) int {
        return a + b
    }(3, 5)

    fmt.Println(result) // 8
}
```

### 赋值给变量

```go
func main() {
    // 把匿名函数赋值给变量
    greet := func(name string) {
        fmt.Println("你好，" + name)
    }

    greet("张三") // 你好，张三
    greet("李四") // 你好，李四
}
```

### 作为参数传递

函数可以作为参数传给另一个函数：

```go
// 对切片中的每个元素执行某种操作
func forEach(nums []int, action func(int)) {
    for _, n := range nums {
        action(n)
    }
}

func main() {
    numbers := []int{1, 2, 3, 4, 5}

    // 打印每个数字
    forEach(numbers, func(n int) {
        fmt.Println(n)
    })

    // 打印每个数字的平方
    forEach(numbers, func(n int) {
        fmt.Println(n * n)
    })
}
```

## 4.7 闭包

**闭包**是一个函数"记住"了它外面的变量。听起来抽象，看个例子就明白了：

```go
func counter() func() int {
    count := 0
    return func() int {
        count++   // 内部函数引用了外部的 count
        return count
    }
}

func main() {
    c := counter()  // 创建一个计数器

    fmt.Println(c()) // 1
    fmt.Println(c()) // 2
    fmt.Println(c()) // 3

    // 再创建一个新的计数器，互不影响
    c2 := counter()
    fmt.Println(c2()) // 1
    fmt.Println(c())  // 4（c 继续累加）
}
```

`counter()` 返回了一个匿名函数，这个匿名函数"记住"了 `count` 变量。每次调用时，`count` 都会在上一次的基础上累加。

实际应用 — 创建带前缀的日志函数：

```go
func makeLogger(prefix string) func(string) {
    return func(msg string) {
        fmt.Printf("[%s] %s\n", prefix, msg)
    }
}

func main() {
    infoLog := makeLogger("INFO")
    errorLog := makeLogger("ERROR")

    infoLog("服务启动成功")       // [INFO] 服务启动成功
    errorLog("数据库连接失败")    // [ERROR] 数据库连接失败
}
```

## 4.8 defer：延迟执行

`defer` 关键字会让一个函数调用**延迟到当前函数返回时**再执行。

### 基本用法

```go
func main() {
    fmt.Println("开始")
    defer fmt.Println("这会最后执行")
    fmt.Println("结束")
}
```

输出：
```
开始
结束
这会最后执行
```

### 多个 defer：后进先出

多个 defer 的执行顺序是**后进先出**（像叠盘子，先放的最后拿出来）：

```go
func main() {
    defer fmt.Println("1")
    defer fmt.Println("2")
    defer fmt.Println("3")
}
```

输出：
```
3
2
1
```

### defer 的经典用途：关闭资源

```go
func readFile() {
    file, err := os.Open("data.txt")
    if err != nil {
        fmt.Println("打开失败:", err)
        return
    }
    defer file.Close()  // 无论后面代码怎么执行，文件最终一定会被关闭

    // 读取文件内容...
    // 即使这里出了错，file.Close() 也会执行
}
```

`defer` 保证了资源一定会被释放，不管中间是否出错。

## 4.9 init 函数

Go 有一个特殊的 `init` 函数，它会在 `main` 函数之前自动执行：

```go
package main

import "fmt"

func init() {
    fmt.Println("init 先执行")
}

func main() {
    fmt.Println("main 后执行")
}
```

输出：
```
init 先执行
main 后执行
```

`init` 通常用于初始化工作，比如加载配置、建立数据库连接等。

## 4.10 综合实战：简单计算器

```go
package main

import "fmt"

func calculate(a, b float64, op string) (float64, error) {
    switch op {
    case "+":
        return a + b, nil
    case "-":
        return a - b, nil
    case "*":
        return a * b, nil
    case "/":
        if b == 0 {
            return 0, fmt.Errorf("除数不能为零")
        }
        return a / b, nil
    default:
        return 0, fmt.Errorf("不支持的运算符: %s", op)
    }
}

func main() {
    operations := []string{"+", "-", "*", "/"}
    a, b := 10.0, 3.0

    for _, op := range operations {
        result, err := calculate(a, b, op)
        if err != nil {
            fmt.Printf("%.0f %s %.0f = 错误: %s\n", a, op, b, err)
        } else {
            fmt.Printf("%.0f %s %.0f = %.2f\n", a, op, b, result)
        }
    }
}
```

输出：
```
10 + 3 = 13.00
10 - 3 = 7.00
10 * 3 = 30.00
10 / 3 = 3.33
```

## 4.11 本章小结

| 概念 | 说明 | 示例 |
|------|------|------|
| 基本函数 | `func 名(参数) 返回值 {}` | `func add(a, b int) int` |
| 多返回值 | Go 特色，常用于返回结果+错误 | `func f() (int, error)` |
| 可变参数 | 参数个数不确定 | `func sum(nums ...int)` |
| 匿名函数 | 没有名字的函数 | `func(x int) { ... }` |
| 闭包 | 函数"记住"外部变量 | 计数器、日志前缀 |
| defer | 延迟执行，函数返回前执行 | `defer file.Close()` |
| init | 在 main 之前自动执行 | 初始化配置 |

## 4.12 练习题

1. **温度转换**：写一个函数 `celsiusToFahrenheit(c float64) float64`，把摄氏度转换为华氏度（公式：F = C × 9/5 + 32）
2. **最大值**：写一个函数 `max(nums ...int) int`，返回传入的所有数字中的最大值
3. **闭包累加器**：写一个函数 `accumulator(initial int) func(int) int`，返回一个累加器，每次调用时把传入的数字加到总和上并返回总和
4. **defer 练习**：写一个函数，用 defer 打印 "函数结束"，在 defer 之前打印 "函数开始" 和 "函数执行中"，观察输出顺序
5. **综合题**：写一个函数 `applyAll(nums []int, fns ...func(int) int) []int`，把 nums 中的每个元素依次经过所有 fns 处理后返回新的切片
