# 第三章：控制流程

> 本章目标：学会使用条件判断（if/else）、循环（for）和多路选择（switch），让程序能做出"选择"和"重复"。

## 3.1 为什么需要控制流程？

前面的程序都是从上到下逐行执行的。但真实的程序需要：

- **做判断**：如果用户年龄大于 18，显示"成年"，否则显示"未成年"
- **做循环**：重复执行某段代码，比如打印 1 到 100
- **做选择**：根据不同的值执行不同的代码

这就是**控制流程**的作用。

## 3.2 条件判断：if / else

### 基本语法

```go
if 条件 {
    // 条件为 true 时执行
}
```

示例：

```go
age := 20

if age >= 18 {
    fmt.Println("你是成年人")
}
```

### if...else

```go
age := 15

if age >= 18 {
    fmt.Println("你是成年人")
} else {
    fmt.Println("你是未成年人")
}
// 输出：你是未成年人
```

### if...else if...else（多个条件）

```go
score := 85

if score >= 90 {
    fmt.Println("优秀")
} else if score >= 80 {
    fmt.Println("良好")
} else if score >= 60 {
    fmt.Println("及格")
} else {
    fmt.Println("不及格")
}
// 输出：良好
```

程序会**从上往下**依次检查条件，一旦某个条件满足就执行对应的代码块，然后跳出整个 if 语句。

### Go 的 if 特殊写法：带初始化语句

Go 允许在 if 的条件前面加一条初始化语句，用分号 `;` 隔开：

```go
// 普通写法
num := 10
if num > 5 {
    fmt.Println("大于5")
}

// Go 特有的写法：在 if 中声明变量
if num := 10; num > 5 {
    fmt.Println("大于5")
    fmt.Println(num) // 在这里可以用 num
}
// 注意：这里的 num 出了 if 就不能用了
```

这种写法的好处是：变量的作用域被限制在 if 块内，不会"污染"外面的代码。

### 比较运算符

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `==` | 等于 | `a == b` |
| `!=` | 不等于 | `a != b` |
| `>` | 大于 | `a > b` |
| `<` | 小于 | `a < b` |
| `>=` | 大于等于 | `a >= b` |
| `<=` | 小于等于 | `a <= b` |

### 逻辑运算符

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `&&` | 并且（两个都要为真） | `age > 18 && age < 60` |
| `\|\|` | 或者（至少一个为真） | `day == "周六" \|\| day == "周日"` |
| `!` | 取反（真变假，假变真） | `!isVIP` |

```go
age := 25
hasID := true

// &&：两个条件都满足
if age >= 18 && hasID {
    fmt.Println("可以进入")
}

// ||：满足其中一个就行
day := "周六"
if day == "周六" || day == "周日" {
    fmt.Println("今天是周末！")
}

// !：取反
isRaining := false
if !isRaining {
    fmt.Println("今天不下雨，出去玩！")
}
```

## 3.3 循环：for

Go 中**只有 `for` 一种循环**（没有 while、do-while），但它能实现所有循环需求。

### 基本 for 循环

```go
// for 初始值; 条件; 每次循环后执行 {
//     循环体
// }

for i := 0; i < 5; i++ {
    fmt.Println(i)
}
```

输出：
```
0
1
2
3
4
```

拆解说明：
- `i := 0` — 循环开始前，创建变量 `i`，初始值为 0
- `i < 5` — 每次循环前检查条件，如果 `i < 5` 为 true 就继续循环
- `i++` — 每次循环结束后，`i` 加 1
- 当 `i` 变成 5 时，`5 < 5` 为 false，循环结束

### 实际例子：计算 1 到 100 的和

```go
sum := 0
for i := 1; i <= 100; i++ {
    sum = sum + i  // 也可以写成 sum += i
}
fmt.Println("1到100的和是：", sum) // 输出：5050
```

### 类似 while 的写法

省略初始值和递增语句，只保留条件：

```go
count := 0
for count < 5 {
    fmt.Println(count)
    count++
}
```

### 无限循环

省略所有部分，就是无限循环：

```go
for {
    fmt.Println("这会一直打印下去...")
    // 需要用 break 跳出，否则永远不会停
}
```

### break 和 continue

- `break`：立即跳出整个循环
- `continue`：跳过当前这次循环，直接进入下一次

```go
// break 示例：找到第一个偶数就停止
for i := 1; i <= 10; i++ {
    if i%2 == 0 {
        fmt.Println("找到第一个偶数：", i)
        break  // 跳出循环
    }
}
// 输出：找到第一个偶数：2

// continue 示例：只打印奇数
for i := 1; i <= 10; i++ {
    if i%2 == 0 {
        continue  // 跳过偶数，直接下一次循环
    }
    fmt.Println(i)
}
// 输出：1 3 5 7 9
```

### 嵌套循环

循环里面可以再套循环：

```go
// 打印九九乘法表
for i := 1; i <= 9; i++ {
    for j := 1; j <= i; j++ {
        fmt.Printf("%d×%d=%-4d", j, i, i*j)
    }
    fmt.Println() // 换行
}
```

输出（部分）：
```
1×1=1   
1×2=2   2×2=4   
1×3=3   2×3=6   3×3=9   
...
```

### for range：遍历集合

`for range` 用于遍历字符串、数组、切片、map 等：

```go
// 遍历字符串
message := "Hello"
for i, ch := range message {
    fmt.Printf("位置 %d: %c\n", i, ch)
}
// 输出：
// 位置 0: H
// 位置 1: e
// 位置 2: l
// 位置 3: l
// 位置 4: o

// 如果不需要索引，用 _ 忽略
for _, ch := range message {
    fmt.Printf("%c ", ch)
}
```

> `_` 是 Go 中的"空白标识符"，表示"我不需要这个值"。Go 不允许有未使用的变量，所以用 `_` 来占位。

## 3.4 多路选择：switch

当你需要判断一个变量的多种可能值时，用 `switch` 比一堆 `if...else if` 更清晰：

### 基本语法

```go
day := "周三"

switch day {
case "周一":
    fmt.Println("新的一周开始了")
case "周二", "周三", "周四":
    fmt.Println("工作日，加油！")
case "周五":
    fmt.Println("快到周末了！")
case "周六", "周日":
    fmt.Println("周末愉快！")
default:
    fmt.Println("无效的日期")
}
// 输出：工作日，加油！
```

要点：
- 每个 `case` 可以有多个值，用逗号分隔
- `default` 是所有 case 都不匹配时执行的（可以省略）
- Go 的 switch **不需要 `break`**！匹配到一个 case 后自动停止（这和 C/Java 不同）

### switch 判断条件

switch 还可以不写变量，直接在 case 中写条件表达式：

```go
score := 85

switch {
case score >= 90:
    fmt.Println("优秀")
case score >= 80:
    fmt.Println("良好")
case score >= 60:
    fmt.Println("及格")
default:
    fmt.Println("不及格")
}
// 输出：良好
```

这种写法等价于一串 if...else if...else，但更加整洁。

### switch 带初始化语句

和 if 一样，switch 也支持在前面加初始化语句：

```go
switch num := 15; {
case num < 0:
    fmt.Println("负数")
case num == 0:
    fmt.Println("零")
case num > 0:
    fmt.Println("正数")
}
```

### fallthrough（少用）

如果你确实需要"穿透"到下一个 case（很少用），可以加 `fallthrough`：

```go
num := 1
switch num {
case 1:
    fmt.Println("一")
    fallthrough  // 会继续执行下一个 case
case 2:
    fmt.Println("二")
}
// 输出：
// 一
// 二
```

## 3.5 综合实战：猜数字游戏

把本章学到的知识综合运用，写一个小游戏：

```go
package main

import (
    "fmt"
    "math/rand"
)

func main() {
    // 生成 1~100 的随机数
    target := rand.Intn(100) + 1
    attempts := 0

    fmt.Println("🎯 猜数字游戏！我想了一个 1~100 的数字，你来猜。")

    for {
        var guess int
        fmt.Print("请输入你的猜测：")
        fmt.Scan(&guess)
        attempts++

        switch {
        case guess < target:
            fmt.Println("太小了！再大一点。")
        case guess > target:
            fmt.Println("太大了！再小一点。")
        default:
            fmt.Printf("恭喜你猜对了！答案就是 %d，你一共猜了 %d 次。\n", target, attempts)
            return
        }
    }
}
```

这个程序用到了：
- `for` 无限循环
- `switch` 多路判断
- `fmt.Scan` 读取用户输入
- `return` 结束程序

## 3.6 本章小结

| 语法 | 用途 | 示例 |
|------|------|------|
| `if...else` | 条件判断 | `if age >= 18 { ... }` |
| `for` | 循环 | `for i := 0; i < 10; i++ { ... }` |
| `for 条件` | 类似 while | `for count < 5 { ... }` |
| `for {}` | 无限循环 | `for { ... break }` |
| `for range` | 遍历集合 | `for i, v := range arr { ... }` |
| `switch` | 多路选择 | `switch day { case "周一": ... }` |
| `break` | 跳出循环 | 在 for 中使用 |
| `continue` | 跳过本次循环 | 在 for 中使用 |

**Go 和其他语言的区别**：
- Go 的 `if` 条件**不需要括号**：`if age > 18` 而不是 `if (age > 18)`
- Go **只有 `for`** 一种循环，没有 while
- Go 的 `switch` **不需要 break**，自动停止

## 3.7 练习题

1. **成绩等级**：输入一个分数（0-100），输出对应等级：90+ 优秀、80+ 良好、60+ 及格、60 以下不及格
2. **偶数之和**：用 for 循环计算 1 到 100 中所有偶数的和
3. **打印三角形**：用嵌套循环打印以下图案（5 行）：
   ```
   *
   **
   ***
   ****
   *****
   ```
4. **FizzBuzz**：打印 1 到 50，但是 3 的倍数打印 "Fizz"，5 的倍数打印 "Buzz"，同时是 3 和 5 的倍数打印 "FizzBuzz"
5. **计算阶乘**：计算并输出 10 的阶乘（10! = 10 × 9 × 8 × ... × 1）
