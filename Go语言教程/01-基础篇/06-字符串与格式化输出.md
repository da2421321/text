# 第六章：字符串与格式化输出

> 本章目标：深入掌握 Go 的字符串操作和 fmt 包的格式化输出。

## 6.1 字符串基础回顾

在 Go 中，字符串是用**双引号**包裹的一串文字：

```go
name := "Hello, Go!"
```

Go 的字符串有两个重要特性：
- **不可变**：一旦创建就不能修改某个字符
- **UTF-8 编码**：天生支持中文、日文、韩文等多国语言

## 6.2 字符串的常见操作

Go 在 `strings` 包中提供了大量字符串操作函数：

```go
import "strings"
```

### 长度

```go
s := "Hello"
fmt.Println(len(s))  // 5（字节数）

// 中文每个字符占 3 个字节
s2 := "你好"
fmt.Println(len(s2)) // 6（2 个中文字 × 3 字节）

// 如果要获取字符数，使用 utf8 包
import "unicode/utf8"
fmt.Println(utf8.RuneCountInString(s2)) // 2
```

### 包含、前缀、后缀

```go
s := "Hello, World!"

fmt.Println(strings.Contains(s, "World"))    // true（包含）
fmt.Println(strings.HasPrefix(s, "Hello"))   // true（以 Hello 开头）
fmt.Println(strings.HasSuffix(s, "!"))       // true（以 ! 结尾）
```

### 查找

```go
s := "Hello, World!"

fmt.Println(strings.Index(s, "World"))    // 7（首次出现的下标）
fmt.Println(strings.Index(s, "Go"))       // -1（找不到返回 -1）
fmt.Println(strings.Count(s, "l"))        // 3（出现次数）
```

### 大小写转换

```go
s := "Hello, World!"

fmt.Println(strings.ToUpper(s))  // HELLO, WORLD!
fmt.Println(strings.ToLower(s))  // hello, world!
fmt.Println(strings.Title(s))    // Hello, World!（每个单词首字母大写）
```

### 去除空白

```go
s := "  Hello, World!  "

fmt.Println(strings.TrimSpace(s))         // "Hello, World!"
fmt.Println(strings.Trim(s, " "))         // "Hello, World!"
fmt.Println(strings.TrimLeft(s, " "))     // "Hello, World!  "
fmt.Println(strings.TrimRight(s, " "))    // "  Hello, World!"
```

### 替换

```go
s := "Hello, World!"

fmt.Println(strings.Replace(s, "World", "Go", 1))  // Hello, Go!
// 第四个参数是替换次数，-1 表示全部替换

s2 := "aaa-bbb-ccc"
fmt.Println(strings.ReplaceAll(s2, "-", "_")) // aaa_bbb_ccc
```

### 分割和拼接

```go
// 分割：字符串 → 切片
s := "苹果,香蕉,橙子,葡萄"
fruits := strings.Split(s, ",")
fmt.Println(fruits)    // [苹果 香蕉 橙子 葡萄]
fmt.Println(fruits[0]) // 苹果

// 拼接：切片 → 字符串
result := strings.Join(fruits, " | ")
fmt.Println(result) // 苹果 | 香蕉 | 橙子 | 葡萄
```

### 重复

```go
fmt.Println(strings.Repeat("Go! ", 3)) // Go! Go! Go! 
fmt.Println(strings.Repeat("-", 20))    // --------------------
```

### 常用函数速查表

| 函数 | 作用 | 示例 |
|------|------|------|
| `strings.Contains(s, sub)` | 是否包含 | `Contains("hello", "ell")` → true |
| `strings.HasPrefix(s, pre)` | 是否以某字符串开头 | `HasPrefix("hello", "he")` → true |
| `strings.HasSuffix(s, suf)` | 是否以某字符串结尾 | `HasSuffix("hello", "lo")` → true |
| `strings.Index(s, sub)` | 查找位置 | `Index("hello", "ll")` → 2 |
| `strings.Count(s, sub)` | 出现次数 | `Count("hello", "l")` → 2 |
| `strings.ToUpper(s)` | 转大写 | `ToUpper("hello")` → "HELLO" |
| `strings.ToLower(s)` | 转小写 | `ToLower("HELLO")` → "hello" |
| `strings.TrimSpace(s)` | 去首尾空白 | `TrimSpace(" hi ")` → "hi" |
| `strings.Replace(s,old,new,n)` | 替换 | `Replace("aab","a","x",1)` → "xab" |
| `strings.Split(s, sep)` | 分割成切片 | `Split("a-b-c", "-")` → ["a","b","c"] |
| `strings.Join(slice, sep)` | 切片拼接成字符串 | `Join(["a","b"], "-")` → "a-b" |

## 6.3 字符与 rune

Go 中单个字符有两种表示：

- `byte`（`uint8`）：表示 ASCII 字符（英文、数字、标点）
- `rune`（`int32`）：表示 Unicode 字符（任何文字，包括中文）

```go
var letter byte = 'A'     // ASCII 字符
var chinese rune = '中'   // Unicode 字符

fmt.Println(letter)   // 65（A 的 ASCII 码）
fmt.Println(chinese)  // 20013（中的 Unicode 码）

// 用 %c 打印字符本身
fmt.Printf("%c\n", letter)  // A
fmt.Printf("%c\n", chinese) // 中
```

### 遍历字符串中的字符

```go
s := "Hello你好"

// 按字节遍历（中文会乱码）
for i := 0; i < len(s); i++ {
    fmt.Printf("%c ", s[i])
}
// 输出：H e l l o ä ½ å ¥ ½（中文乱了）

fmt.Println()

// 按字符遍历（推荐，正确处理中文）
for _, ch := range s {
    fmt.Printf("%c ", ch)
}
// 输出：H e l l o 你 好
```

> **记住**：遍历字符串时用 `for range`，不要用下标访问，否则中文会出问题。

## 6.4 格式化输出：fmt 包详解

`fmt.Println` 只是最基础的输出方式。`fmt.Printf` 提供了强大的格式化能力。

### fmt.Printf 基本用法

```go
name := "张三"
age := 25

fmt.Printf("我叫%s，今年%d岁\n", name, age)
// 输出：我叫张三，今年25岁
```

`%s`、`%d` 叫做**格式化占位符**（动词），会被后面的变量值替换。

### 常用格式化占位符

| 占位符 | 含义 | 示例 |
|--------|------|------|
| `%s` | 字符串 | `Printf("%s", "hello")` → hello |
| `%d` | 整数（十进制） | `Printf("%d", 42)` → 42 |
| `%f` | 浮点数 | `Printf("%f", 3.14)` → 3.140000 |
| `%.2f` | 浮点数（保留 2 位小数） | `Printf("%.2f", 3.14159)` → 3.14 |
| `%t` | 布尔值 | `Printf("%t", true)` → true |
| `%c` | 字符 | `Printf("%c", 65)` → A |
| `%v` | 任意类型的默认格式 | `Printf("%v", 42)` → 42 |
| `%T` | 变量的类型 | `Printf("%T", 42)` → int |
| `%p` | 指针地址 | `Printf("%p", &x)` → 0xc0000... |
| `%%` | 输出一个 % 符号 | `Printf("100%%")` → 100% |
| `\n` | 换行 | |

### 详细示例

```go
package main

import "fmt"

func main() {
    // 字符串
    name := "Go语言"
    fmt.Printf("名称：%s\n", name)       // 名称：Go语言

    // 整数
    count := 42
    fmt.Printf("数量：%d\n", count)       // 数量：42
    fmt.Printf("八进制：%o\n", count)     // 八进制：52
    fmt.Printf("十六进制：%x\n", count)   // 十六进制：2a

    // 浮点数
    price := 19.99
    fmt.Printf("价格：%f\n", price)       // 价格：19.990000
    fmt.Printf("价格：%.2f\n", price)     // 价格：19.99
    fmt.Printf("价格：%.0f\n", price)     // 价格：20

    // 布尔值
    ok := true
    fmt.Printf("状态：%t\n", ok)          // 状态：true

    // 查看类型
    fmt.Printf("name 的类型：%T\n", name)   // name 的类型：string
    fmt.Printf("count 的类型：%T\n", count) // count 的类型：int
    fmt.Printf("price 的类型：%T\n", price) // price 的类型：float64

    // 万能占位符 %v
    fmt.Printf("值：%v\n", name)   // 值：Go语言
    fmt.Printf("值：%v\n", count)  // 值：42
    fmt.Printf("值：%v\n", price)  // 值：19.99
}
```

### 控制输出宽度和对齐

```go
// %10s：宽度为 10，右对齐
fmt.Printf("[%10s]\n", "hello")   // [     hello]

// %-10s：宽度为 10，左对齐
fmt.Printf("[%-10s]\n", "hello")  // [hello     ]

// %05d：宽度为 5，用 0 填充
fmt.Printf("[%05d]\n", 42)       // [00042]
```

### Println、Printf、Sprintf 的区别

```go
name := "张三"

// Println：自动换行，参数之间加空格
fmt.Println("你好", name)  // 你好 张三

// Printf：需要格式化字符串，不自动换行
fmt.Printf("你好 %s\n", name)  // 你好 张三

// Sprintf：不打印，返回格式化后的字符串
msg := fmt.Sprintf("你好 %s", name)
fmt.Println(msg) // 你好 张三
```

`Sprintf` 非常实用，当你需要把格式化的内容存到变量中（而不是直接打印）时使用。

## 6.5 字符串拼接的几种方式

```go
// 方式一：+ 拼接（简单场景）
s := "Hello" + " " + "World"

// 方式二：fmt.Sprintf（需要格式化时）
name := "张三"
age := 25
s2 := fmt.Sprintf("%s今年%d岁", name, age)

// 方式三：strings.Builder（大量拼接时，性能最好）
var builder strings.Builder
for i := 0; i < 100; i++ {
    builder.WriteString("Go")
}
result := builder.String()

// 方式四：strings.Join（拼接切片）
words := []string{"Go", "is", "great"}
s3 := strings.Join(words, " ")  // "Go is great"
```

选择建议：
- 简单拼接几个字符串 → 用 `+`
- 需要格式化 → 用 `fmt.Sprintf`
- 循环中大量拼接 → 用 `strings.Builder`
- 拼接切片 → 用 `strings.Join`

## 6.6 字符串与数字互转

```go
import "strconv"

// 整数 → 字符串
s1 := strconv.Itoa(42)           // "42"
s2 := fmt.Sprintf("%d", 42)     // "42"（另一种方式）

// 字符串 → 整数
n1, err := strconv.Atoi("42")
if err != nil {
    fmt.Println("转换失败：", err)
} else {
    fmt.Println(n1) // 42
}

// 字符串 → 浮点数
f1, err := strconv.ParseFloat("3.14", 64)

// 浮点数 → 字符串
s3 := strconv.FormatFloat(3.14, 'f', 2, 64)  // "3.14"
s4 := fmt.Sprintf("%.2f", 3.14)              // "3.14"（更常用）

// 布尔值 → 字符串
s5 := strconv.FormatBool(true) // "true"

// 字符串 → 布尔值
b1, _ := strconv.ParseBool("true") // true
```

## 6.7 综合实战：文本处理工具

```go
package main

import (
    "fmt"
    "strings"
)

func main() {
    text := "  Go is a GREAT programming language. Go is fast. Go is simple.  "

    // 1. 清理空白
    cleaned := strings.TrimSpace(text)
    fmt.Println("清理后：", cleaned)

    // 2. 转小写
    lower := strings.ToLower(cleaned)
    fmt.Println("小写：", lower)

    // 3. 统计单词数
    words := strings.Fields(lower) // Fields 按空白分割（比 Split 更智能）
    fmt.Println("单词数：", len(words))

    // 4. 统计每个单词出现次数
    wordCount := make(map[string]int)
    for _, w := range words {
        // 去掉标点
        w = strings.Trim(w, ".,!?;:")
        wordCount[w]++
    }

    fmt.Println("\n单词频率：")
    for word, count := range wordCount {
        fmt.Printf("  %-15s %d 次\n", word, count)
    }

    // 5. 替换
    replaced := strings.ReplaceAll(cleaned, "Go", "Golang")
    fmt.Println("\n替换后：", replaced)
}
```

## 6.8 本章小结

| 概念 | 说明 |
|------|------|
| 字符串不可变 | 创建后不能修改某个字符 |
| `strings` 包 | 提供 Contains、Split、Join、Replace 等操作 |
| `byte` vs `rune` | byte 是单字节字符，rune 支持中文等 Unicode |
| `fmt.Printf` | 格式化输出，用占位符控制格式 |
| `fmt.Sprintf` | 格式化但不打印，返回字符串 |
| `strconv` 包 | 字符串与数字、布尔值的互转 |
| `strings.Builder` | 高性能字符串拼接 |
