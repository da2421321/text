# 第七章：结构体与方法

> 本章目标：掌握结构体（struct）的定义和使用，学会为结构体绑定方法，理解组合（嵌入）的概念。

## 7.1 什么是结构体？

前面学过的变量只能存一个值。如果我们想描述一个"学生"，需要姓名、年龄、成绩等多个属性，用多个独立变量会很混乱：

```go
// 这样太散了，不好管理
studentName := "张三"
studentAge := 20
studentScore := 90.5
```

**结构体（struct）** 就是把多个不同类型的数据"打包"在一起，形成一个自定义类型：

```go
// 定义一个 Student 结构体
type Student struct {
    Name  string
    Age   int
    Score float64
}
```

你可以把结构体想象成一张**表格的一行**：

```
┌────────┬─────┬───────┐
│ Name   │ Age │ Score │
├────────┼─────┼───────┤
│ 张三   │ 20  │ 90.5  │
└────────┴─────┴───────┘
```

## 7.2 定义和使用结构体

### 定义

```go
type 结构体名 struct {
    字段名 类型
    字段名 类型
    ...
}
```

实际例子：

```go
type User struct {
    Name   string
    Age    int
    Email  string
    IsVIP  bool
}
```

### 创建结构体实例

```go
// 方式一：按字段名赋值（推荐，清晰不容易出错）
u1 := User{
    Name:  "张三",
    Age:   25,
    Email: "zhangsan@example.com",
    IsVIP: true,
}

// 方式二：按顺序赋值（不推荐，字段多了容易搞混）
u2 := User{"李四", 30, "lisi@example.com", false}

// 方式三：先声明再赋值
var u3 User
u3.Name = "王五"
u3.Age = 28
u3.Email = "wangwu@example.com"
u3.IsVIP = false

// 方式四：只给部分字段赋值（其余用零值）
u4 := User{
    Name: "赵六",
    Age:  22,
    // Email 和 IsVIP 用零值：""、false
}
```

### 访问和修改字段

用 `.` 点号访问结构体的字段：

```go
u := User{Name: "张三", Age: 25, Email: "zs@example.com", IsVIP: true}

// 读取
fmt.Println(u.Name)  // 张三
fmt.Println(u.Age)   // 25

// 修改
u.Age = 26
u.IsVIP = false
fmt.Println(u.Age)   // 26
```

## 7.3 结构体的零值

声明但不初始化时，每个字段都是对应类型的零值：

```go
var u User
fmt.Println(u.Name)  // ""（空字符串）
fmt.Println(u.Age)   // 0
fmt.Println(u.Email) // ""
fmt.Println(u.IsVIP) // false
```

## 7.4 结构体与函数

### 作为函数参数

```go
func printUser(u User) {
    fmt.Printf("姓名：%s，年龄：%d\n", u.Name, u.Age)
}

func main() {
    u := User{Name: "张三", Age: 25}
    printUser(u) // 姓名：张三，年龄：25
}
```

### 重要：结构体是值类型

结构体传参时会**复制一份**，函数内的修改不会影响外面：

```go
func birthday(u User) {
    u.Age++  // 只修改了副本
}

func main() {
    u := User{Name: "张三", Age: 25}
    birthday(u)
    fmt.Println(u.Age) // 25 ← 没有变！
}
```

如果想修改原始数据，需要传**指针**：

```go
func birthday(u *User) {
    u.Age++  // 修改的是原始数据
}

func main() {
    u := User{Name: "张三", Age: 25}
    birthday(&u)        // 传入地址
    fmt.Println(u.Age)  // 26 ← 变了！
}
```

## 7.5 为结构体绑定方法

Go 没有"类（class）"的概念，但你可以给结构体绑定**方法**，效果类似。

### 基本语法

```go
func (接收者 结构体类型) 方法名(参数) 返回值 {
    // 方法体
}
```

`(接收者 结构体类型)` 就是"这个方法绑定到哪个结构体上"。

### 值接收者

```go
type User struct {
    Name string
    Age  int
}

// 绑定一个 Greet 方法到 User
func (u User) Greet() string {
    return "你好，我是" + u.Name
}

func (u User) IsAdult() bool {
    return u.Age >= 18
}

func main() {
    u := User{Name: "张三", Age: 25}

    fmt.Println(u.Greet())    // 你好，我是张三
    fmt.Println(u.IsAdult())  // true
}
```

### 指针接收者

当你需要在方法中**修改结构体的字段**时，必须用指针接收者：

```go
// 值接收者：修改的是副本，不影响原对象
func (u User) GrowWrong() {
    u.Age++
}

// 指针接收者：修改的是原对象
func (u *User) Grow() {
    u.Age++
}

func main() {
    u := User{Name: "张三", Age: 25}

    u.GrowWrong()
    fmt.Println(u.Age) // 25（没变）

    u.Grow()
    fmt.Println(u.Age) // 26（变了）
}
```

### 什么时候用指针接收者？

| 场景 | 用值接收者 | 用指针接收者 |
|------|-----------|-------------|
| 只读取字段 | ✅ | ✅ |
| 需要修改字段 | ❌ | ✅ |
| 结构体很大（字段多） | ❌（复制成本高） | ✅ |
| 统一风格 | - | ✅（推荐统一用指针接收者） |

**实践建议**：如果拿不准，就统一用指针接收者 `*User`。

## 7.6 构造函数

Go 没有 `constructor` 关键字，但约定用 `New` 开头的函数来创建结构体：

```go
type User struct {
    Name  string
    Age   int
    Email string
}

// 构造函数，返回指针
func NewUser(name string, age int, email string) *User {
    return &User{
        Name:  name,
        Age:   age,
        Email: email,
    }
}

func main() {
    u := NewUser("张三", 25, "zs@example.com")
    fmt.Println(u.Name) // 张三
}
```

构造函数的好处：
- 可以做参数验证
- 可以设置默认值
- 隐藏创建细节

```go
func NewUser(name string, age int) *User {
    if age < 0 {
        age = 0
    }
    return &User{
        Name:  name,
        Age:   age,
        Email: name + "@default.com",
    }
}
```

## 7.7 组合（嵌入）—— Go 的"继承"

Go 不支持传统的类继承（extends），而是用**组合/嵌入**来复用代码。

### 普通字段组合

```go
type Address struct {
    City     string
    Street   string
    ZipCode  string
}

type User struct {
    Name    string
    Age     int
    Address Address  // 普通字段，需要通过 u.Address.City 访问
}

func main() {
    u := User{
        Name: "张三",
        Age:  25,
        Address: Address{
            City:   "北京",
            Street: "长安街",
        },
    }
    fmt.Println(u.Address.City) // 北京
}
```

### 匿名嵌入（推荐）

如果不给字段起名，直接用类型名嵌入，字段和方法会被"提升"：

```go
type Address struct {
    City   string
    Street string
}

func (a Address) FullAddress() string {
    return a.City + " " + a.Street
}

type User struct {
    Name string
    Age  int
    Address // 匿名嵌入！注意这里没有字段名
}

func main() {
    u := User{
        Name: "张三",
        Age:  25,
        Address: Address{
            City:   "北京",
            Street: "长安街",
        },
    }

    // 可以直接访问嵌入结构体的字段（被"提升"了）
    fmt.Println(u.City)          // 北京（等价于 u.Address.City）
    fmt.Println(u.FullAddress()) // 北京 长安街（方法也被提升）
}
```

### 多层嵌入

```go
type Base struct {
    ID int
}

type User struct {
    Base
    Name string
}

type Admin struct {
    User
    Role string
}

func main() {
    admin := Admin{
        User: User{
            Base: Base{ID: 1},
            Name: "管理员",
        },
        Role: "超级管理员",
    }

    fmt.Println(admin.ID)   // 1（从 Base 提升上来）
    fmt.Println(admin.Name) // 管理员（从 User 提升上来）
    fmt.Println(admin.Role) // 超级管理员
}
```

## 7.8 结构体标签（Tag）

结构体字段可以附加**标签（tag）**，在 JSON 序列化等场景中非常常用：

```go
type User struct {
    Name  string `json:"name"`
    Age   int    `json:"age"`
    Email string `json:"email,omitempty"`  // omitempty：空值时省略
}
```

使用示例：

```go
import "encoding/json"

func main() {
    u := User{Name: "张三", Age: 25}

    // 结构体 → JSON
    data, _ := json.Marshal(u)
    fmt.Println(string(data))
    // {"name":"张三","age":25}
    // 注意：Email 为空且有 omitempty，所以被省略了

    // JSON → 结构体
    jsonStr := `{"name":"李四","age":30,"email":"ls@test.com"}`
    var u2 User
    json.Unmarshal([]byte(jsonStr), &u2)
    fmt.Println(u2.Name)  // 李四
    fmt.Println(u2.Email) // ls@test.com
}
```

标签的作用：控制 JSON 字段名。没有标签时，JSON 的 key 就是字段名（首字母大写），这通常不符合 API 的命名规范。

## 7.9 公开与私有

Go 用**首字母大小写**来控制访问权限：

- **首字母大写** → 公开（Public），其他包可以访问
- **首字母小写** → 私有（Private），只有当前包内可以访问

```go
type User struct {
    Name string  // 公开：其他包可以访问
    Age  int     // 公开
    password string  // 私有：只有当前包内可以访问
}

// 公开方法
func (u *User) GetName() string {
    return u.Name
}

// 私有方法
func (u *User) hashPassword() string {
    // 只在包内使用
    return "hashed_" + u.password
}
```

## 7.10 综合实战：图书管理

```go
package main

import "fmt"

type Book struct {
    Title  string
    Author string
    Price  float64
    Stock  int
}

func NewBook(title, author string, price float64, stock int) *Book {
    return &Book{
        Title:  title,
        Author: author,
        Price:  price,
        Stock:  stock,
    }
}

func (b *Book) Sell(quantity int) bool {
    if b.Stock < quantity {
        fmt.Printf("《%s》库存不足（剩余 %d 本）\n", b.Title, b.Stock)
        return false
    }
    b.Stock -= quantity
    fmt.Printf("成功售出《%s》%d 本，剩余 %d 本\n", b.Title, quantity, b.Stock)
    return true
}

func (b *Book) Restock(quantity int) {
    b.Stock += quantity
    fmt.Printf("《%s》补货 %d 本，当前库存 %d 本\n", b.Title, quantity, b.Stock)
}

func (b Book) Info() string {
    return fmt.Sprintf("《%s》 作者：%s 价格：%.2f元 库存：%d本",
        b.Title, b.Author, b.Price, b.Stock)
}

type Library struct {
    Name  string
    Books []*Book
}

func (lib *Library) AddBook(book *Book) {
    lib.Books = append(lib.Books, book)
}

func (lib *Library) ListBooks() {
    fmt.Printf("\n===== %s 图书列表 =====\n", lib.Name)
    for i, book := range lib.Books {
        fmt.Printf("%d. %s\n", i+1, book.Info())
    }
}

func (lib *Library) FindByTitle(title string) *Book {
    for _, book := range lib.Books {
        if book.Title == title {
            return book
        }
    }
    return nil
}

func main() {
    lib := Library{Name: "海牛图书馆"}

    lib.AddBook(NewBook("Go 程序设计语言", "Alan Donovan", 79.00, 10))
    lib.AddBook(NewBook("Go 语言实战", "William Kennedy", 59.00, 5))
    lib.AddBook(NewBook("Go Web 编程", "谢大", 45.00, 8))

    lib.ListBooks()

    fmt.Println()

    book := lib.FindByTitle("Go 语言实战")
    if book != nil {
        book.Sell(3)
        book.Sell(3)
        book.Restock(10)
    }

    lib.ListBooks()
}
```

## 7.11 本章小结

| 概念 | 说明 |
|------|------|
| `struct` | 把多个字段打包成一个自定义类型 |
| 创建实例 | `u := User{Name: "张三", Age: 25}` |
| 访问字段 | `u.Name`、`u.Age` |
| 值接收者方法 | `func (u User) ...` — 不能修改原对象 |
| 指针接收者方法 | `func (u *User) ...` — 可以修改原对象 |
| 构造函数 | `func NewUser(...) *User` — 约定以 New 开头 |
| 匿名嵌入 | 类似"继承"，字段和方法被提升 |
| Tag | `json:"name"` — 控制 JSON 序列化 |
| 公开/私有 | 首字母大写公开，小写私有 |
