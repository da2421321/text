# 控制结构

## 条件语句

### if语句

Go语言的if语句与其他语言类似，但条件表达式不需要括号：

```go
if condition {
    // 执行语句
}
```

可以包含初始化语句：

```go
if variable := value; condition {
    // 执行语句
}
```

示例：

```go
package main

import "fmt"

func main() {
    age := 20
    
    if age >= 18 {
        fmt.Println("成年人")
    } else {
        fmt.Println("未成年人")
    }
    
    // 使用初始化语句
    if score := 85; score >= 90 {
        fmt.Println("优秀")
    } else if score >= 80 {
        fmt.Println("良好")
    } else if score >= 60 {
        fmt.Println("及格")
    } else {
        fmt.Println("不及格")
    }
}
```

### switch语句

Go的switch语句更加灵活，不需要break语句，且case条件可以是任何类型：

```go
switch expression {
case value1:
    // 执行语句
case value2:
    // 执行语句
default:
    // 默认执行语句
}
```

示例：

```go
package main

import "fmt"

func main() {
    grade := "B"
    
    switch grade {
    case "A":
        fmt.Println("优秀")
    case "B":
        fmt.Println("良好")
    case "C":
        fmt.Println("一般")
    case "D":
        fmt.Println("及格")
    default:
        fmt.Println("不及格")
    }
    
    // 不带表达式的switch，相当于if-else
    score := 85
    switch {
    case score >= 90:
        fmt.Println("优秀")
    case score >= 80:
        fmt.Println("良好")
    case score >= 60:
        fmt.Println("及格")
    default:
        fmt.Println("不及格")
    }
    
    // 使用fallthrough关键字
    switch score / 10 {
    case 10, 9:
        fmt.Println("优秀")
        fallthrough // 继续执行下一个case
    case 8:
        fmt.Println("良好")
    case 7:
        fmt.Println("一般")
    default:
        fmt.Println("需要努力")
    }
}
```

## 循环语句

### for循环

Go语言只有for循环，但有多种使用方式：

```go
// 基本for循环
for initialization; condition; post {
    // 执行语句
}

// while循环形式
for condition {
    // 执行语句
}

// 无限循环
for {
    // 执行语句
}

// range循环（遍历数组、切片、map等）
for key, value := range collection {
    // 执行语句
}
```

示例：

```go
package main

import "fmt"

func main() {
    // 基本for循环
    for i := 1; i <= 5; i++ {
        fmt.Printf("第%d次循环\n", i)
    }
    
    // while循环形式
    count := 0
    for count < 3 {
        fmt.Printf("count = %d\n", count)
        count++
    }
    
    // 无限循环（使用break退出）
    sum := 0
    for {
        sum++
        if sum > 5 {
            break
        }
        fmt.Printf("sum = %d\n", sum)
    }
    
    // range循环
    numbers := []int{1, 2, 3, 4, 5}
    for index, value := range numbers {
        fmt.Printf("索引: %d, 值: %d\n", index, value)
    }
    
    // 遍历map
    scores := map[string]int{
        "张三": 90,
        "李四": 85,
        "王五": 92,
    }
    
    for name, score := range scores {
        fmt.Printf("%s的分数是%d\n", name, score)
    }
    
    // 只遍历key
    for name := range scores {
        fmt.Println("学生:", name)
    }
    
    // 只遍历value
    for _, score := range scores {
        fmt.Println("分数:", score)
    }
}
```

## 跳转语句

### break语句

用于跳出循环或switch语句：

```go
for i := 1; i <= 10; i++ {
    if i == 5 {
        break // 跳出循环
    }
    fmt.Println(i)
}
```

带标签的break可以跳出多层循环：

```go
package main

import "fmt"

func main() {
OuterLoop:
    for i := 1; i <= 3; i++ {
        for j := 1; j <= 3; j++ {
            if i == 2 && j == 2 {
                fmt.Println("跳出所有循环")
                break OuterLoop
            }
            fmt.Printf("i=%d, j=%d\n", i, j)
        }
    }
    fmt.Println("循环结束")
}
```

### continue语句

用于跳过当前循环的剩余部分，直接进入下一次循环：

```go
for i := 1; i <= 10; i++ {
    if i%2 == 0 {
        continue // 跳过偶数
    }
    fmt.Println(i) // 只打印奇数
}
```

带标签的continue可以跳到指定循环：

```go
package main

import "fmt"

func main() {
OuterLoop:
    for i := 1; i <= 3; i++ {
        for j := 1; j <= 3; j++ {
            if j == 2 {
                fmt.Println("跳过内层循环的当前迭代")
                continue OuterLoop
            }
            fmt.Printf("i=%d, j=%d\n", i, j)
        }
    }
}
```

### goto语句

Go支持goto语句，但应谨慎使用：

```go
func main() {
    i := 0
    
Loop:
    fmt.Println(i)
    i++
    if i < 3 {
        goto Loop
    }
}
```

### 标签语句(label)

标签可以与break、continue和goto配合使用，用于控制程序流程：

```go
package main

import "fmt"

func main() {
    // 标签与break配合使用
    fmt.Println("标签与break示例:")
    
OuterLoop1:
    for i := 1; i <= 3; i++ {
        for j := 1; j <= 3; j++ {
            if i*j > 3 {
                fmt.Println("结束外层循环")
                break OuterLoop1
            }
            fmt.Printf("%d*%d=%d\n", i, j, i*j)
        }
    }
    
    // 标签与continue配合使用
    fmt.Println("\n标签与continue示例:")
    
OuterLoop2:
    for i := 1; i <= 3; i++ {
        for j := 1; j <= 3; j++ {
            if j > i {
                fmt.Printf("跳过i=%d的剩余内循环\n", i)
                continue OuterLoop2
            }
            fmt.Printf("%d*%d=%d\n", i, j, i*j)
        }
    }
}
```

## 代码示例

```go
package main

import "fmt"

func main() {
    // 综合示例：学生成绩管理系统
    students := map[string]float64{
        "张三": 92.5,
        "李四": 87.0,
        "王五": 95.5,
        "赵六": 78.0,
        "钱七": 88.5,
    }
    
    fmt.Println("学生成绩单:")
    fmt.Println("-------------")
    
    total := 0.0
    count := 0
    
    for name, score := range students {
        // 判断等级
        var grade string
        switch {
        case score >= 90:
            grade = "优秀"
        case score >= 80:
            grade = "良好"
        case score >= 70:
            grade = "中等"
        case score >= 60:
            grade = "及格"
        default:
            grade = "不及格"
        }
        
        fmt.Printf("%s: %.1f分 (%s)\n", name, score, grade)
        
        total += score
        count++
    }
    
    fmt.Println("-------------")
    average := total / float64(count)
    fmt.Printf("平均分: %.2f\n", average)
    
    // 统计各等级人数
    excellent := 0
    good := 0
    fair := 0
    pass := 0
    fail := 0
    
    for _, score := range students {
        switch {
        case score >= 90:
            excellent++
        case score >= 80:
            good++
        case score >= 70:
            fair++
        case score >= 60:
            pass++
        default:
            fail++
        }
    }
    
    fmt.Println("等级统计:")
    fmt.Printf("优秀: %d人\n", excellent)
    fmt.Printf("良好: %d人\n", good)
    fmt.Printf("中等: %d人\n", fair)
    fmt.Printf("及格: %d人\n", pass)
    fmt.Printf("不及格: %d人\n", fail)
}
```

## 练习题

1. 编写一个程序，判断一个年份是否为闰年
2. 编写一个程序，打印九九乘法表
3. 编写一个程序，找出100以内的所有质数

```go
// 练习题参考答案

package main

import "fmt"

// 练习题1: 判断闰年
func isLeapYear(year int) bool {
    if (year%4 == 0 && year%100 != 0) || year%400 == 0 {
        return true
    }
    return false
}

// 练习题2: 打印九九乘法表
func printMultiplicationTable() {
    for i := 1; i <= 9; i++ {
        for j := 1; j <= i; j++ {
            fmt.Printf("%d×%d=%d\t", j, i, i*j)
        }
        fmt.Println()
    }
}

// 练习题3: 找出100以内的质数
func findPrimes(limit int) {
    fmt.Printf("100以内的质数有:\n")
    count := 0
    
    for num := 2; num <= limit; num++ {
        isPrime := true
        for i := 2; i*i <= num; i++ {
            if num%i == 0 {
                isPrime = false
                break
            }
        }
        
        if isPrime {
            fmt.Printf("%d ", num)
            count++
            if count%10 == 0 {
                fmt.Println() // 每10个数换行
            }
        }
    }
    fmt.Printf("\n总共%d个质数\n", count)
}

func main() {
    fmt.Println("闰年判断:")
    testYears := []int{2000, 2004, 1900, 2023, 2024}
    for _, year := range testYears {
        if isLeapYear(year) {
            fmt.Printf("%d年是闰年\n", year)
        } else {
            fmt.Printf("%d年不是闰年\n", year)
        }
    }
    
    fmt.Println("\n九九乘法表:")
    printMultiplicationTable()
    
    fmt.Println("\n质数查找:")
    findPrimes(100)
}
```

## select语句

select语句是Go语言中的一个重要控制结构，主要用于处理多个通道（channel）操作。它的语法类似于switch语句，但是每个case必须是一个通道操作。

```go
select {
case <-channel1:
    // 如果channel1收到数据，执行这里
case data := <-channel2:
    // 如果channel2收到数据，执行这里
case channel3 <- data:
    // 如果成功向channel3发送数据，执行这里
default:
    // 如果上面都没有准备好，执行这里
}
```

select语句的特点：
- 每个case必须是一个通道操作（发送或接收）
- 如果多个case同时就绪，select会随机选择一个执行
- 如果没有case就绪且有default语句，则执行default
- 如果没有case就绪且没有default语句，select将阻塞直到某个case就绪

示例：

```go
package main

import (
    "fmt"
    "time"
)

func main() {
    ch1 := make(chan string) //创建通道
    ch2 := make(chan string) //创建通道
    
    go func() {
        time.Sleep(1 * time.Second)
        ch1 <- "来自通道1的消息" //通道发送操作
    }()
    
    go func() {
        time.Sleep(2 * time.Second)
        ch2 <- "来自通道2的消息" //通道发送操作
    }()
    
    // 使用select监听两个通道
    for i := 0; i < 2; i++ {
        select {
        case msg1 := <-ch1:
            fmt.Println("接收到:", msg1)
        case msg2 := <-ch2:
            fmt.Println("接收到:", msg2)
        case <-time.After(3 * time.Second):
            fmt.Println("超时")
            return
        }
    }
}
```

select语句在并发编程中非常有用，常用于以下场景：
- 非阻塞通道操作
- 超时处理
- 多通道数据处理
- 退出信号处理

## 常见错误和最佳实践

### 常见错误

1. **忘记break语句**：虽然Go的switch语句不需要break，但在其他语言转过来的开发者可能会忘记这一点。

```go
// 错误示例（其他语言习惯）
switch value {
case 1:
    fmt.Println("一")
    break // 在Go中不需要
case 2:
    fmt.Println("二")
    break // 在Go中不需要
}
```

2. **死循环**：创建无限循环但忘记退出条件。

```go
// 错误示例
for {
    // 没有break或return语句
    // 这将导致程序永远运行
}
```

3. **标签位置错误**：标签必须紧贴在循环或switch语句之前。

```go
// 错误示例
func main() {
    i := 0
    
    // 错误：标签和for之间有空行
OuterLoop:
    
    for i < 10 {
        // ...
    }
}
```

4. **select语句中的阻塞**：没有default分支的select在所有通道都没准备好时会阻塞。

```go
// 可能导致阻塞的代码
select {
case <-ch1:
    // 处理ch1
case <-ch2:
    // 处理ch2
    // 如果ch1和ch2都没准备好，将永久阻塞
}
```

### 最佳实践

1. **使用初始化语句**：在if和switch语句中使用初始化语句可以限制变量的作用域。

```go
// 推荐做法
if value, err := someFunction(); err == nil {
    // 使用value
} else {
    // 处理错误
}
// 这里无法访问value和err
```

2. **避免深层嵌套**：使用提前返回减少嵌套层级。

```go
// 不推荐
func process(data int) {
    if data > 0 {
        if checkSomething(data) {
            // 处理逻辑
        } else {
            return
        }
    } else {
        return
    }
}

// 推荐
func process(data int) {
    if data <= 0 {
        return
    }
    
    if !checkSomething(data) {
        return
    }
    
    // 处理逻辑
}
```

3. **谨慎使用goto**：goto可能导致代码难以理解和维护，应尽量避免使用。

4. **select语句添加超时**：在select语句中添加超时处理，避免长时间阻塞。

```go
select {
case data := <-ch:
    // 处理数据
case <-time.After(2 * time.Second):
    // 超时处理
}
```

5. **使用for-range遍历**：使用for-range遍历数组、切片和map，避免手动管理索引。

```go
// 推荐
for index, value := range array {
    // 使用index和value
}

// 不推荐
for i := 0; i < len(array); i++ {
    value := array[i]
    // 使用value
}
```
