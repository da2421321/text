# Go语言简介与环境搭建

## Go语言简介

Go（又称 Golang）是 Google 开发的一种静态强类型、编译型、并发型，并具有垃圾回收功能的编程语言。Go 语言于 2009 年 11 月正式对外发布。

### Go语言的特点

1. **简洁、快速、安全**
   - 语法简洁，易于学习
   - 编译速度快
   - 内存安全，类型安全

2. **并行机制**
   - 原生支持并发编程
   - Goroutine 轻量级线程
   - Channel 用于 goroutine 间通信

3. **内存管理**
   - 自动垃圾回收
   - 减少内存泄漏风险

4. **跨平台**
   - 支持多种操作系统和架构
   - 一键编译部署

## 环境搭建

### 安装 Go

1. 访问 [Go 官方网站](https://golang.org/dl/) 下载适合您操作系统的安装包
2. 运行安装程序，按照默认设置安装即可
3. 验证安装：
   ```bash
   go version
   ```

### 配置环境变量

安装完成后，Go 会自动配置环境变量。主要的环境变量包括：

- `GOROOT`：Go 的安装目录
- `GOPATH`：Go 的工作目录（Go 1.11 后推荐使用 Go Modules）
- `GOBIN`：Go 可执行文件目录

### 开发工具推荐

1. **Visual Studio Code**
   - 安装 Go 扩展
   - 提供代码补全、调试等功能

2. **GoLand**
   - JetBrains 出品的 Go IDE
   - 功能全面，适合大型项目

3. **Vim/Emacs**
   - 适合熟悉编辑器的开发者

## 第一个 Go 程序

创建一个名为 `hello.go` 的文件：

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

运行程序：

```bash
go run hello.go
```

编译程序：

```bash
go build hello.go
./hello
```

## Go Modules

Go 1.11 引入了 Go Modules 作为依赖管理工具。

### 初始化项目

```bash
go mod init project-name
```

### 添加依赖

```bash
go get github.com/gin-gonic/gin
```

### 查看依赖

```bash
go mod tidy
go list -m all
```

## 代码解析

让我们详细分析第一个 Go 程序：

```go
package main
```
这一行声明了当前文件属于 `main` 包。在 Go 中，`main` 包是程序的入口点。

```go
import "fmt"
```
导入 `fmt` 包，它提供了格式化输入输出的功能。

```go
func main() {
    fmt.Println("Hello, World!")
}
```
定义 `main` 函数，这是程序的入口函数。`fmt.Println` 用于打印文本并换行。
