# 基本语法与数据类型

## Go程序结构

Go程序由包（package）组成，每个程序必须有一个main包和main函数作为程序入口。

```go
package main

import (
    "fmt"
)

func main() {
    fmt.Println("Hello, World!")
}
```

## 变量声明

Go支持多种变量声明方式：

### 零值（Zero Values）

在Go语言中，变量声明后如果没有显式初始化，将自动赋予该类型的零值：

```go
var (
    intVar    int     // 零值为 0
    floatVar  float64 // 零值为 0.0
    boolVar   bool    // 零值为 false
    stringVar string  // 零值为 ""
    pointerVar *int   // 零值为 nil
)

func main() {
    fmt.Printf("整数零值: %v\n", intVar)
    fmt.Printf("浮点数零值: %v\n", floatVar)
    fmt.Printf("布尔零值: %v\n", boolVar)
    fmt.Printf("字符串零值: %q\n", stringVar)
    fmt.Printf("指针零值: %v\n", pointerVar)
}
```

### 1. 标准声明

```go
var name string
var age int
var isStudent bool
```

### 2. 声明并初始化

```go
var name string = "张三"
var age int = 25
```

### 3. 类型推导

```go
var name = "张三"
var age = 25
```

### 4. 简短声明（仅在函数内使用）

```go
name := "张三"
age := 25
```

## 常量

使用`const`关键字声明常量：

```go
const pi = 3.14159
```

### iota 常量生成器

`iota`是Go语言的常量计数器，只能在常量的表达式中使用：

1. 基本用法：
```go
const (
    Monday = iota    // 0
    Tuesday          // 1
    Wednesday        // 2
    Thursday         // 3
    Friday          // 4
)
```

2. 设置初始值和步长：
```go
const (
    KB = 1 << (10 * iota)  // 1 << (10 * 0) = 1
    MB                     // 1 << (10 * 1) = 1024
    GB                     // 1 << (10 * 2) = 1048576
    TB                     // 1 << (10 * 3) = 1073741824
)
```

3. 跳过值：
```go
const (
    A = iota  // 0
    _         // 1 (跳过)
    B         // 2
    C         // 3
)
```

4. 中间插队：
```go
const (
    A = iota  // 0
    B = 100   // 100
    C = iota  // 2
    D         // 3
)
```

5. 多个iota定义：
```go
const (
    A, B = iota + 1, iota + 2  // 1, 2
    C, D                       // 2, 3
    E, F                       // 3, 4
)
```

## 基本数据类型

### 1. 数值类型

```go
// 整数类型
var a int = 10
var b int8 = 127
var c int16 = 32767
var d int32 = 2147483647
var e int64 = 9223372036854775807

// 无符号整数类型
var f uint = 10
var g uint8 = 255

// 浮点类型
var h float32 = 3.14
var i float64 = 3.141592653589793

// 复数类型
var j complex64 = 1 + 2i
var k complex128 = 3 + 4i
```

### 2. 布尔类型

```go
var isTrue bool = true
var isFalse bool = false
```

### 3. 字符串类型

```go
var str string = "Hello, World!"
var rawStr string = `原始字符串
可以包含换行`
```

#### 字符串操作

1. 字符串连接：
```go
str1 := "Hello"
str2 := "World"
result := str1 + " " + str2  // "Hello World"

// 使用strings.Join
import "strings"
strSlice := []string{"Hello", "World"}
result = strings.Join(strSlice, " ")  // "Hello World"
```

2. 字符串长度：
```go
str := "Hello, 世界"
byteLen := len(str)                    // 13 (字节长度)
runeLen := len([]rune(str))           // 8 (字符数量)
```

3. 子字符串：
```go
str := "Hello, World"
substr := str[0:5]    // "Hello"
substr = str[:5]      // "Hello"
substr = str[7:]      // "World"
```

4. 字符串查找：
```go
import "strings"

str := "Hello, World"
contains := strings.Contains(str, "World")     // true
index := strings.Index(str, "o")              // 4
lastIndex := strings.LastIndex(str, "o")      // 7
hasPrefix := strings.HasPrefix(str, "He")     // true
hasSuffix := strings.HasSuffix(str, "ld")     // true
```
5. 字符串替换：
```go
import "strings"

str := "hello hello hello"
newStr := strings.Replace(str, "hello", "hi", 2)     // "hi hi hello"
newStr = strings.ReplaceAll(str, "hello", "hi")      // "hi hi hi"
```

6. 大小写转换：
```go
import "strings"

str := "Hello, World"
upper := strings.ToUpper(str)    // "HELLO, WORLD"
lower := strings.ToLower(str)    // "hello, world"
```

7. 字符串分割与合并：
```go
import "strings"

// 分割
str := "a,b,c"
parts := strings.Split(str, ",")    // []string{"a", "b", "c"}

// 合并
joined := strings.Join(parts, "-")  // "a-b-c"
```

8. 去除空白字符：
```go
import "strings"

str := "  Hello, World  \n"
trimmed := strings.TrimSpace(str)           // "Hello, World"
trimLeft := strings.TrimLeft(str, " ")      // "Hello, World  \n"
trimRight := strings.TrimRight(str, " \n")  // "  Hello, World"
```

### 4. 字符类型

Go没有专门的字符类型，字符是`byte`（uint8）或`rune`（int32）类型：

```go
var ch1 byte = 'A'
var ch2 rune = '中'
```

## 类型转换

Go在不同类型之间赋值时需要显式转换：

```go
var a int = 65
var b float64 = float64(a)
var c string = string(a) // 转换为字符"A"
```

## 代码示例

```go
package main

import (
    "fmt"
    "reflect"
)

func main() {
    // 变量声明
    var name string = "张三"
    var age int = 25
    isStudent := true // 简短声明
    
    fmt.Println("姓名:", name)
    fmt.Println("年龄:", age)
    fmt.Println("是否为学生:", isStudent)
    
    // 数据类型演示
    var (
        a int     = 10
        b float64 = 3.14
        c bool    = true
        d string  = "Hello"
    )
    
    fmt.Printf("a的类型: %T, 值: %v\n", a, a)
    fmt.Printf("b的类型: %T, 值: %v\n", b, b)
    fmt.Printf("c的类型: %T, 值: %v\n", c, c)
    fmt.Printf("d的类型: %T, 值: %v\n", d, d)
    
    // 类型转换
    var e int = 65
    var f float64 = float64(e)
    var g string = string(e)
    
    fmt.Println("整数:", e)
    fmt.Println("转换为浮点数:", f)
    fmt.Println("转换为字符串:", g) // ASCII 65对应字符"A"
    
    // 使用reflect包获取类型信息
    fmt.Println("使用reflect获取类型:", reflect.TypeOf(a))
}
```

## 格式化输出

Go语言使用`fmt.Printf`函数进行格式化输出，通过占位符来控制输出格式。以下是常用的占位符：

### 1. 通用占位符

| 占位符 | 说明 |
|--------|------|
| %v | 以默认格式输出值 |
| %+v | 输出结构体时会显示字段名 |
| %#v | 输出值的Go语法表示 |
| %T | 输出值的类型 |
| %% | 输出百分号本身 |

### 2. 整数占位符

| 占位符 | 说明 |
|--------|------|
| %d | 十进制整数 |
| %b | 二进制整数 |
| %o | 八进制整数 |
| %x | 十六进制整数（小写） |
| %X | 十六进制整数（大写） |
| %c | 输出单个字符 |
| %q | 输出带引号的字符 |

### 3. 浮点数和复数占位符

| 占位符 | 说明 |
|--------|------|
| %f | 浮点数（默认宽度和精度） |
| %.2f | 浮点数（保留2位小数） |
| %e | 科学计数法（小写） |
| %E | 科学计数法（大写） |
| %g | 根据值的大小自动选择 %f 或 %e |
| %G | 根据值的大小自动选择 %f 或 %E |

### 4. 字符串占位符

| 占位符 | 说明 |
|--------|------|
| %s | 普通字符串 |
| %q | 带双引号的字符串 |
| %x | 十六进制表示的字符串 |
| %X | 大写十六进制表示的字符串 |

### 5. 布尔值占位符

| 占位符 | 说明 |
|--------|------|
| %t | true 或 false |

### 6. 指针占位符

| 占位符 | 说明 |
|--------|------|
| %p | 十六进制表示的指针地址 |

### 示例代码

```go
package main

import "fmt"

func main() {
    // 基本类型格式化
    num := 65
    fmt.Printf("整数示例:\n")
    fmt.Printf("%%d: %d\n", num)      // 65
    fmt.Printf("%%b: %b\n", num)      // 1000001
    fmt.Printf("%%c: %c\n", num)      // A
    fmt.Printf("%%x: %x\n", num)      // 41
    fmt.Printf("%%X: %X\n", num)      // 41
    
    // 浮点数格式化
    pi := 3.141592653589793
    fmt.Printf("\n浮点数示例:\n")
    fmt.Printf("%%f: %f\n", pi)       // 3.141593
    fmt.Printf("%%.2f: %.2f\n", pi)   // 3.14
    fmt.Printf("%%e: %e\n", pi)       // 3.141593e+00
    
    // 字符串格式化
    str := "Hello, 世界"
    fmt.Printf("\n字符串示例:\n")
    fmt.Printf("%%s: %s\n", str)      // Hello, 世界
    fmt.Printf("%%q: %q\n", str)      // "Hello, 世界"
    
    // 布尔值格式化
    ok := true
    fmt.Printf("\n布尔值示例:\n")
    fmt.Printf("%%t: %t\n", ok)       // true
    
    // 指针格式化
    ptr := &num
    fmt.Printf("\n指针示例:\n")
    fmt.Printf("%%p: %p\n", ptr)      // 0xc0000b4008
    
    // 宽度和精度控制
    fmt.Printf("\n宽度和精度控制:\n")
    fmt.Printf("|%6d|%6d|\n", 12, 345)        // |    12|   345|
    fmt.Printf("|%-6d|%-6d|\n", 12, 345)      // |12    |345   |
    fmt.Printf("|%6.2f|%6.2f|\n", 1.2, 3.45)  // |  1.20|  3.45|
    
    // 结构体格式化
    type Person struct {
        Name string
        Age  int
    }
    p := Person{Name: "张三", Age: 25}
    fmt.Printf("\n结构体示例:\n")
    fmt.Printf("%%v: %v\n", p)       // {张三 25}
    fmt.Printf("%%+v: %+v\n", p)     // {Name:张三 Age:25}
    fmt.Printf("%%#v: %#v\n", p)     // main.Person{Name:"张三", Age:25}
}
```

## 练习题

1. 声明多个不同类型的变量并初始化
2. 创建一个程序，演示各种数值类型的取值范围
3. 编写一个程序，实现摄氏度和华氏度的转换

```go
// 练习题参考答案
package main

import (
    "fmt"
    "math"
)

func main() {
    // 练习题1: 声明多个不同类型的变量
    var (
        userName    string  = "李四"
        userAge     int     = 30
        userHeight  float32 = 175.5
        isMarried   bool    = false
        userScore   int8    = 95
    )
    
    fmt.Println("用户信息:")
    fmt.Printf("姓名: %s\n", userName)
    fmt.Printf("年龄: %d\n", userAge)
    fmt.Printf("身高: %.1f cm\n", userHeight)
    fmt.Printf("是否已婚: %t\n", isMarried)
    fmt.Printf("分数: %d\n", userScore)
    
    // 练习题2: 演示数值类型的取值范围
    fmt.Println("\n数值类型取值范围:")
    
    // 整数类型
    fmt.Printf("int8: %d 到 %d\n", math.MinInt8, math.MaxInt8)
    fmt.Printf("int16: %d 到 %d\n", math.MinInt16, math.MaxInt16)
    fmt.Printf("int32: %d 到 %d\n", math.MinInt32, math.MaxInt32)
    fmt.Printf("int64: %d 到 %d\n", math.MinInt64, math.MaxInt64)
    
    // 无符号整数
    fmt.Printf("uint8: 0 到 %d\n", math.MaxUint8)
    fmt.Printf("uint16: 0 到 %d\n", math.MaxUint16)
    fmt.Printf("uint32: 0 到 %d\n", math.MaxUint32)
    
    // 浮点数
    fmt.Printf("float32: %e 到 %e\n", math.SmallestNonzeroFloat32, math.MaxFloat32)
    fmt.Printf("float64: %e 到 %e\n", math.SmallestNonzeroFloat64, math.MaxFloat64)
    
    // 示例值
    var (
        maxInt8  int8  = 127
        minInt8  int8  = -128
        maxUint8 uint8 = 255
    )
    
    fmt.Printf("\n示例值:\n")
    fmt.Printf("最大int8值: %d\n", maxInt8)
    fmt.Printf("最小int8值: %d\n", minInt8)
    fmt.Printf("最大uint8值: %d\n", maxUint8)
    
    // 溢出示例
    fmt.Printf("\n溢出示例:\n")
    fmt.Printf("int8溢出: %d + 1 = %d\n", maxInt8, maxInt8+1) // 会发生溢出
    fmt.Printf("uint8溢出: %d + 1 = %d\n", maxUint8, maxUint8+1) // 会发生溢出
    
    // 练习题3: 摄氏度和华氏度转换
    celsius := 25.0
    fahrenheit := celsius*9/5 + 32
    fmt.Printf("\n温度转换:\n")
    fmt.Printf("%.1f°C = %.1f°F\n", celsius, fahrenheit)
    
    // 反向转换
    fahrenheit = 77.0
    celsius = (fahrenheit - 32) * 5 / 9
    fmt.Printf("%.1f°F = %.1f°C\n", fahrenheit, celsius)
}
```
