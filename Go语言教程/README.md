# Go 语言从零到一教程

欢迎来到 Go 语言教程！这份教程**面向编程新手**，从最基础的概念讲起，一步一步带你掌握 Go 语言。

> 不需要任何编程基础，只要你会用电脑，就能跟着学。

## 什么是 Go 语言？

Go（又叫 Golang）是 Google 在 2009 年推出的编程语言。它有几个显著特点：

- **简单易学** — 语法简洁，关键字只有 25 个（Java 有 50+，C++ 有 80+）
- **运行速度快** — 编译成机器码直接运行，比 Python、JavaScript 快很多
- **天生支持并发** — 用 goroutine 轻松实现多任务同时执行
- **一个文件搞定部署** — 编译后就是一个可执行文件，不需要安装运行环境

很多知名项目都用 Go 编写：Docker、Kubernetes、Gin 框架等。

## 推荐学习顺序

```
基础篇（1-10 章，约 1-2 周）
    ↓
进阶篇（指针 + 并发 + 测试 + IO，约 1 周）
    ↓
实战篇（Gin + GORM + Redis + JWT + 完整项目，约 1-2 周）
    ↓
企业级工程化篇（目录结构 + 日志 + 文档 + Docker，约 1-2 周）
```

**建议**：每学完一章就动手敲代码，不要只看不练。每章末尾都有练习题，一定要做！

## 运行环境

- **必须**：Go 1.21+（[下载地址](https://go.dev/dl/)）
- **推荐**：VS Code + Go 插件（语法高亮、代码补全、自动格式化）
- **数据库章节需要**：MySQL 8+
- **缓存章节需要**：Redis 7+
- **部署章节需要**：Docker 24+

## 学完你能做到什么？

1. **基础篇学完后**：能独立编写 Go 程序，处理各种数据类型和业务逻辑
2. **进阶篇学完后**：理解内存模型，能写高效的并发程序
3. **实战篇学完后**：能用 Gin + GORM 开发完整的 Web API
4. **工程化篇学完后**：能搭建符合企业标准的生产级项目

## 遇到问题怎么办？

1. 仔细看报错信息 — Go 的报错信息通常很明确
2. 用 `go vet` 检查代码问题
3. 搜索引擎搜报错信息
4. 查看 [Go 官方文档](https://go.dev/doc/)
5. 在 [Go Playground](https://go.dev/play/) 在线运行代码测试
