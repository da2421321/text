# 04 - 布局与样式

> Flutter 没有 CSS，布局全靠 Widget 嵌套。本章对比 CSS Flexbox 讲解。

---

## 1. 布局思维转变

```
CSS 思维：给元素加 class，写 CSS 样式
Flutter 思维：用布局 Widget 包裹子 Widget，通过属性控制样式
```

| CSS | Flutter Widget |
|-----|---------------|
| `display: flex; flex-direction: row` | `Row` |
| `display: flex; flex-direction: column` | `Column` |
| `position: absolute` | `Stack` + `Positioned` |
| `overflow: scroll` | `SingleChildScrollView` / `ListView` |
| `display: grid` | `GridView` |
| `padding` | `Padding` Widget 或 Container 的 padding |
| `margin` | `Container` 的 margin |
| `background-color` | `Container` 的 color |
| `border-radius` | `BoxDecoration` 的 borderRadius |

## 2. Row 和 Column（Flex 布局）

### Row（水平排列）

```dart
// 类比 CSS: display: flex; flex-direction: row;
Row(
  mainAxisAlignment: MainAxisAlignment.spaceBetween,  // 主轴对齐
  crossAxisAlignment: CrossAxisAlignment.center,       // 交叉轴对齐
  children: [
    Text('左'),
    Text('中'),
    Text('右'),
  ],
)
```

### Column（垂直排列）

```dart
// 类比 CSS: display: flex; flex-direction: column;
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  crossAxisAlignment: CrossAxisAlignment.stretch,  // 拉伸到最宽
  children: [
    Text('第一行'),
    SizedBox(height: 8),  // 间距
    Text('第二行'),
    Text('第三行'),
  ],
)
```

### 对齐方式对比

```
MainAxisAlignment（主轴）：
  start        → justify-content: flex-start
  end          → justify-content: flex-end
  center       → justify-content: center
  spaceBetween → justify-content: space-between
  spaceAround  → justify-content: space-around
  spaceEvenly  → justify-content: space-evenly

CrossAxisAlignment（交叉轴）：
  start        → align-items: flex-start
  end          → align-items: flex-end
  center       → align-items: center
  stretch      → align-items: stretch
  baseline     → align-items: baseline
```

### Expanded（弹性占比）

```dart
// 类比 CSS: flex: 1
Row(
  children: [
    Expanded(
      flex: 2,  // 占 2/3
      child: Container(color: Colors.red, height: 50),
    ),
    Expanded(
      flex: 1,  // 占 1/3
      child: Container(color: Colors.blue, height: 50),
    ),
  ],
)
```

### Flexible（弹性但不强制填满）

```dart
// 类比 CSS: flex-shrink: 1（可以缩小但不强制撑满）
Row(
  children: [
    Flexible(
      child: Text('这段文字可能很长很长很长很长很长'),
    ),
    Icon(Icons.arrow_forward),
  ],
)
```

### Spacer（弹性空白）

```dart
// 类比 CSS: flex: 1 的空白元素
Row(
  children: [
    Text('左边'),
    Spacer(),        // 撑开中间空白
    Text('右边'),
  ],
)
```

## 3. Stack（绝对定位）

类比 CSS 的 `position: relative` + `position: absolute`：

```dart
Stack(
  children: [
    // 底层：背景图
    Image.network('https://example.com/bg.jpg'),

    // 叠加层：文字
    Positioned(
      bottom: 16,
      left: 16,
      child: Text(
        '图片标题',
        style: TextStyle(color: Colors.white, fontSize: 18),
      ),
    ),

    // 右上角角标
    Positioned(
      top: 8,
      right: 8,
      child: Container(
        padding: EdgeInsets.all(4),
        decoration: BoxDecoration(
          color: Colors.red,
          borderRadius: BorderRadius.circular(12),
        ),
        child: Text('NEW', style: TextStyle(color: Colors.white, fontSize: 10)),
      ),
    ),
  ],
)
```

### Stack 对齐

```dart
Stack(
  alignment: Alignment.center,  // 所有子元素默认居中
  children: [
    Container(width: 200, height: 200, color: Colors.grey),
    Text('居中文字'),  // 自动居中
  ],
)
```

## 4. 列表布局

### ListView（滚动列表）

```dart
// 类比 Vue 的 v-for + overflow: scroll

// 简单列表
ListView(
  children: [
    ListTile(title: Text('Item 1')),
    ListTile(title: Text('Item 2')),
    ListTile(title: Text('Item 3')),
  ],
)

// 动态列表（推荐，懒加载）
ListView.builder(
  itemCount: items.length,
  itemBuilder: (context, index) {
    return ListTile(
      key: ValueKey(items[index].id),
      title: Text(items[index].name),
      subtitle: Text(items[index].desc),
      leading: Icon(Icons.star),
      trailing: Icon(Icons.arrow_forward_ios),
      onTap: () => print('点击了 ${items[index].name}'),
    );
  },
)

// 分隔线列表
ListView.separated(
  itemCount: items.length,
  separatorBuilder: (context, index) => Divider(),
  itemBuilder: (context, index) => ListTile(
    title: Text(items[index]),
  ),
)
```

### GridView（网格布局）

```dart
// 类比 CSS Grid

// 固定列数
GridView.count(
  crossAxisCount: 2,          // 2列
  crossAxisSpacing: 8,        // 列间距
  mainAxisSpacing: 8,         // 行间距
  childAspectRatio: 1.0,      // 宽高比
  children: items.map((item) => Card(
    child: Center(child: Text(item)),
  )).toList(),
)

// 动态网格（推荐）
GridView.builder(
  gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
    crossAxisCount: 3,
    crossAxisSpacing: 8,
    mainAxisSpacing: 8,
  ),
  itemCount: items.length,
  itemBuilder: (context, index) {
    return Container(
      color: Colors.blue,
      child: Center(child: Text('${index + 1}')),
    );
  },
)

// 自适应列宽
GridView.builder(
  gridDelegate: SliverGridDelegateWithMaxCrossAxisExtent(
    maxCrossAxisExtent: 150,  // 每列最大宽度 150
    crossAxisSpacing: 8,
    mainAxisSpacing: 8,
  ),
  itemCount: 20,
  itemBuilder: (context, index) => Container(
    color: Colors.primaries[index % Colors.primaries.length],
  ),
)
```

### SingleChildScrollView（单个可滚动内容）

```dart
// 当内容可能超出屏幕时使用
SingleChildScrollView(
  child: Column(
    children: [
      // 大量内容...
    ],
  ),
)
```

## 6. 样式系统

### 颜色

```dart
// Material 颜色
Colors.red
Colors.blue
Colors.green
Colors.grey
Colors.white
Colors.black
Colors.transparent

// 颜色深浅（100-900）
Colors.blue[100]   // 浅蓝
Colors.blue[900]   // 深蓝
Colors.blue.shade200

// 自定义颜色
Color(0xFF2196F3)           // ARGB 十六进制
Color.fromARGB(255, 33, 150, 243)
Color.fromRGBO(33, 150, 243, 1.0)

// 透明度
Colors.black.withOpacity(0.5)
Colors.blue.withAlpha(128)
```

### 文字样式

```dart
TextStyle(
  fontSize: 16,
  fontWeight: FontWeight.bold,    // w100-w900, bold=w700
  fontStyle: FontStyle.italic,
  color: Colors.black87,
  letterSpacing: 1.2,             // 字间距
  wordSpacing: 4,                 // 词间距
  height: 1.5,                    // 行高（倍数）
  decoration: TextDecoration.underline,
  decorationColor: Colors.red,
  shadows: [
    Shadow(color: Colors.grey, blurRadius: 4, offset: Offset(2, 2)),
  ],
)

// 主题文字样式（推荐使用，保持一致性）
Theme.of(context).textTheme.headlineLarge   // 大标题
Theme.of(context).textTheme.headlineMedium  // 中标题
Theme.of(context).textTheme.bodyLarge       // 正文大
Theme.of(context).textTheme.bodyMedium      // 正文中
Theme.of(context).textTheme.labelSmall      // 小标签
```

### BoxDecoration（容器装饰）

```dart
BoxDecoration(
  // 背景
  color: Colors.white,
  gradient: LinearGradient(
    begin: Alignment.topLeft,
    end: Alignment.bottomRight,
    colors: [Colors.blue, Colors.purple],
  ),

  // 圆角
  borderRadius: BorderRadius.circular(12),
  borderRadius: BorderRadius.only(
    topLeft: Radius.circular(12),
    topRight: Radius.circular(12),
  ),

  // 边框
  border: Border.all(color: Colors.grey, width: 1),
  border: Border(
    bottom: BorderSide(color: Colors.grey, width: 1),
  ),

  // 阴影
  boxShadow: [
    BoxShadow(
      color: Colors.black.withOpacity(0.1),
      blurRadius: 8,
      spreadRadius: 2,
      offset: Offset(0, 4),
    ),
  ],

  // 背景图
  image: DecorationImage(
    image: NetworkImage('https://example.com/bg.jpg'),
    fit: BoxFit.cover,
  ),
)
```

## 7. 主题系统（Theme）

类比 CSS 变量 / Tailwind 主题配置：

```dart
// 在 MaterialApp 中配置全局主题
MaterialApp(
  theme: ThemeData(
    useMaterial3: true,
    colorScheme: ColorScheme.fromSeed(
      seedColor: Colors.blue,
      brightness: Brightness.light,
    ),
    textTheme: TextTheme(
      headlineLarge: TextStyle(fontSize: 32, fontWeight: FontWeight.bold),
      bodyMedium: TextStyle(fontSize: 14, height: 1.5),
    ),
    elevatedButtonTheme: ElevatedButtonThemeData(
      style: ElevatedButton.styleFrom(
        shape: RoundedRectangleBorder(
          borderRadius: BorderRadius.circular(8),
        ),
      ),
    ),
    inputDecorationTheme: InputDecorationTheme(
      border: OutlineInputBorder(),
      contentPadding: EdgeInsets.symmetric(horizontal: 16, vertical: 12),
    ),
  ),
  darkTheme: ThemeData(
    colorScheme: ColorScheme.fromSeed(
      seedColor: Colors.blue,
      brightness: Brightness.dark,
    ),
  ),
  themeMode: ThemeMode.system,  // 跟随系统
  home: MyHomePage(),
)

// 在组件中使用主题
Widget build(BuildContext context) {
  final theme = Theme.of(context);
  final colors = theme.colorScheme;

  return Container(
    color: colors.surface,
    child: Text(
      '标题',
      style: theme.textTheme.headlineMedium?.copyWith(
        color: colors.primary,
      ),
    ),
  );
}
```

## 8. 响应式布局

```dart
// 获取屏幕尺寸
Widget build(BuildContext context) {
  final size = MediaQuery.of(context).size;
  final width = size.width;
  final height = size.height;

  // 根据屏幕宽度切换布局
  if (width > 600) {
    return _buildTabletLayout();
  } else {
    return _buildMobileLayout();
  }
}

// LayoutBuilder（类比 CSS 媒体查询）
LayoutBuilder(
  builder: (context, constraints) {
    if (constraints.maxWidth > 600) {
      return Row(children: [...]);   // 平板：横向
    } else {
      return Column(children: [...]);  // 手机：纵向
    }
  },
)

// FractionallySizedBox（百分比尺寸）
FractionallySizedBox(
  widthFactor: 0.8,   // 父容器宽度的 80%
  child: ElevatedButton(onPressed: () {}, child: Text('按钮')),
)
```

## 9. 常用布局模式

### 卡片布局

```dart
Card(
  elevation: 4,
  shape: RoundedRectangleBorder(
    borderRadius: BorderRadius.circular(12),
  ),
  child: Padding(
    padding: EdgeInsets.all(16),
    child: Column(
      crossAxisAlignment: CrossAxisAlignment.start,
      children: [
        Text('标题', style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold)),
        SizedBox(height: 8),
        Text('描述内容...'),
        SizedBox(height: 16),
        Row(
          mainAxisAlignment: MainAxisAlignment.end,
          children: [
            TextButton(onPressed: () {}, child: Text('取消')),
            ElevatedButton(onPressed: () {}, child: Text('确认')),
          ],
        ),
      ],
    ),
  ),
)
```

### 底部导航栏页面

```dart
class MainPage extends StatefulWidget {
  @override
  State<MainPage> createState() => _MainPageState();
}

class _MainPageState extends State<MainPage> {
  int _currentIndex = 0;

  final _pages = [
    HomePage(),
    SearchPage(),
    ProfilePage(),
  ];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: _pages[_currentIndex],
      bottomNavigationBar: BottomNavigationBar(
        currentIndex: _currentIndex,
        onTap: (index) => setState(() => _currentIndex = index),
        items: [
          BottomNavigationBarItem(icon: Icon(Icons.home), label: '首页'),
          BottomNavigationBarItem(icon: Icon(Icons.search), label: '搜索'),
          BottomNavigationBarItem(icon: Icon(Icons.person), label: '我的'),
        ],
      ),
    );
  }
}
```

## 10. 小结

| CSS 概念 | Flutter 实现 |
|---------|-------------|
| Flexbox Row | `Row` Widget |
| Flexbox Column | `Column` Widget |
| flex: 1 | `Expanded` |
| position: absolute | `Stack` + `Positioned` |
| overflow: scroll | `ListView` / `SingleChildScrollView` |
| CSS Grid | `GridView` |
| flex-wrap: wrap | `Wrap` |
| padding/margin | `Padding` Widget / `Container` 属性 |
| border-radius | `BoxDecoration.borderRadius` |
| box-shadow | `BoxDecoration.boxShadow` |
| CSS 变量 | `ThemeData` |
| 媒体查询 | `MediaQuery` / `LayoutBuilder` |
