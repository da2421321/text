# 01 - Flutter 简介与环境搭建

> 面向 Vue3 / UniApp 开发者的 Flutter 入门

---

## 1. Flutter 是什么？

Flutter 是 Google 开发的跨平台 UI 框架，用 **Dart** 语言编写，一套代码可以运行在：

- Android / iOS
- Web
- Windows / macOS / Linux

### 和 UniApp 的对比

| 对比项 | UniApp | Flutter |
|--------|--------|---------|
| 语言 | JavaScript / Vue | Dart |
| 渲染方式 | WebView / 原生组件 | 自绘引擎（Skia/Impeller） |
| 性能 | 中等 | 接近原生 |
| 生态 | 国内生态好，插件多 | 国际生态好，pub.dev |
| 学习曲线 | 低（会 Vue 即可） | 中等（需学 Dart） |
| 热更新 | 支持 | 不支持（审核限制） |

**核心差异**：Flutter 不用 HTML/CSS，所有 UI 都是 Widget（类似 Vue 组件），用代码描述界面，性能极好。

---

## 2. 核心概念速览

```
Vue3 概念          →    Flutter 对应概念
─────────────────────────────────────────
组件 (Component)   →    Widget
模板 (template)    →    build() 方法返回的 Widget 树
响应式数据 (ref)   →    setState / Provider / Riverpod
生命周期           →    StatefulWidget 生命周期
路由 (vue-router)  →    Navigator / GoRouter
Pinia              →    Provider / Riverpod / GetX
CSS 样式           →    Widget 属性（padding、color 等）
Flex 布局          →    Row / Column / Flex
```

## 3. 环境搭建

### 3.1 安装 Flutter SDK

**Windows：**

```bash
# 1. 下载 Flutter SDK
# 访问 https://flutter.dev/docs/get-started/install/windows
# 下载 flutter_windows_xxx.zip

# 2. 解压到合适目录（避免中文路径和空格）
# 推荐：C:\flutter

# 3. 配置环境变量
# 将 C:\flutter\bin 添加到 PATH

# 4. 验证安装
flutter doctor
```

**macOS：**

```bash
# 使用 Homebrew
brew install --cask flutter

# 或手动下载解压后配置 PATH
export PATH="$PATH:/Users/你的用户名/flutter/bin"

flutter doctor
```

### 3.2 配置国内镜像（重要！）

由于网络原因，需要配置国内镜像：

```bash
# Windows - 在系统环境变量中添加：
FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
PUB_HOSTED_URL=https://pub.flutter-io.cn

# macOS/Linux - 在 ~/.bashrc 或 ~/.zshrc 中添加：
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_HOSTED_URL=https://pub.flutter-io.cn
```

### 3.3 安装 Android Studio

1. 下载安装 [Android Studio](https://developer.android.com/studio)
2. 安装 Flutter 和 Dart 插件：
   - `File → Settings → Plugins → 搜索 Flutter → Install`
3. 安装 Android SDK（Android Studio 会引导）
4. 创建 Android 模拟器：`Tools → AVD Manager → Create Virtual Device`

### 3.4 VS Code 配置（推荐）

```bash
# 安装扩展
# 1. Flutter（官方扩展）
# 2. Dart（官方扩展）
# 3. Awesome Flutter Snippets（代码片段）
```

### 3.5 运行 flutter doctor

```bash
flutter doctor
```

输出示例（全部打勾才算完成）：

```

[✓] Android toolchain
[✓] Chrome - develop for the web
```

常见问题修复：

```bash

# 检查具体问题

```

## 4. 创建第一个项目

```bash
# 创建项目
flutter create my_app

# 进入项目目录
cd my_app

# 运行（需要先启动模拟器或连接真机）
flutter run

# 运行到 Chrome（Web）
flutter run -d chrome

# 查看可用设备
flutter devices
```

### 项目目录结构

```
my_app/
├── android/          # Android 原生代码（类似 UniApp 的 android 目录）
├── ios/              # iOS 原生代码
├── lib/              # 主要代码目录（类似 src/）
│   └── main.dart     # 入口文件（类似 main.js）
├── test/             # 测试文件
├── pubspec.yaml      # 依赖配置（类似 package.json）
└── pubspec.lock      # 锁定版本（类似 pnpm-lock.yaml）
```

### pubspec.yaml 对比 package.json

```yaml
# pubspec.yaml（Flutter）
name: my_app
description: A new Flutter project

dependencies:
  flutter:
    sdk: flutter
  http: ^1.1.0          # 类似 npm 包
  provider: ^6.1.0

dev_dependencies:
  flutter_test:
    sdk: flutter
  flutter_lints: ^3.0.0

flutter:
  uses-material-design: true
  assets:               # 静态资源（类似 Vue 的 public 目录）
    - assets/images/
  fonts:                # 自定义字体
    - family: MyFont
      fonts:
        - asset: assets/fonts/MyFont.ttf
```

## 5. Hello World 解析

```dart
// lib/main.dart

import 'package:flutter/material.dart';

// 程序入口，类似 Vue 的 createApp().mount('#app')
void main() {
  runApp(const MyApp());
}

// 根组件，类似 App.vue
class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',
      theme: ThemeData(
        colorScheme: ColorScheme.fromSeed(seedColor: Colors.deepPurple),
        useMaterial3: true,
      ),
      home: const MyHomePage(title: 'Flutter Demo Home Page'),
    );
  }
}

// 有状态组件，类似 Vue 的 <script setup> + ref
class MyHomePage extends StatefulWidget {
  const MyHomePage({super.key, required this.title});
  final String title;

  @override
  State<MyHomePage> createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  int _counter = 0;

  void _incrementCounter() {
    setState(() {           // 类似 Vue 的 ref 自动触发更新
      _counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(        // 页面脚手架，提供 AppBar、body、FAB 等
      appBar: AppBar(
        title: Text(widget.title),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            const Text('You have pushed the button this many times:'),
            Text(
              '$_counter',
              style: Theme.of(context).textTheme.headlineMedium,
            ),
          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: _incrementCounter,
        child: const Icon(Icons.add),
      ),
    );
  }
}
```

**和 Vue3 的对比理解：**

```vue
<!-- Vue3 等价写法 -->
<template>
  <div>
    <p>You have pushed the button this many times:</p>
    <p>{{ counter }}</p>
    <button @click="incrementCounter">+</button>
  </div>
</template>

<script setup>
import { ref } from 'vue'
const counter = ref(0)
const incrementCounter = () => counter.value++
</script>
```

## 6. 常用 Flutter 命令

```bash
# 运行项目
flutter run

# 热重载（运行中按 r）
# 热重启（运行中按 R）

# 构建 APK
flutter build apk
flutter build apk --release

# 构建 iOS
flutter build ios

# 构建 Web
flutter build web

# 安装依赖（类似 npm install）
flutter pub get

# 添加依赖（类似 npm install xxx）
flutter pub add http

# 升级依赖
flutter pub upgrade

# 清理构建缓存
flutter clean
```

## 7. 小结

- Flutter 用 Dart 语言，所有 UI 都是 Widget（类比 Vue 组件）
- `build()` 方法返回 Widget 树，类比 Vue 的 `template`
- `setState()` 触发 UI 更新，类比 Vue 的响应式
- `pubspec.yaml` 管理依赖，类比 `package.json`
- 下一章学习 Dart 语法，有 JS 基础上手很快
