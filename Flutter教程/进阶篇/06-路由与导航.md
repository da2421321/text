# 06 - 路由与导航

> 类比 Vue Router，Flutter 有内置 Navigator 和第三方 GoRouter。

---

## 1. 路由方案对比

| 方案 | 类比 | 特点 |
|------|------|------|
| `Navigator` | 编程式路由 | 内置，简单场景够用 |
| `GoRouter` | Vue Router | 声明式路由，支持深链接，推荐 |
| `AutoRoute` | Vue Router | 代码生成，类型安全 |

---

## 2. Navigator（内置路由）

### 基本跳转

```dart
// 跳转到新页面（类比 router.push）
Navigator.push(
  context,
  MaterialPageRoute(builder: (context) => DetailPage()),
);

// 带参数跳转
Navigator.push(
  context,
  MaterialPageRoute(
    builder: (context) => DetailPage(id: '123', title: '商品详情'),
  ),
);

// 返回（类比 router.back）
Navigator.pop(context);

// 返回并传值
Navigator.pop(context, '返回的数据');

// 接收返回值
final result = await Navigator.push(
  context,
  MaterialPageRoute(builder: (context) => SelectPage()),
);
print('选择了: $result');

// 替换当前页面（类比 router.replace）
Navigator.pushReplacement(
  context,
  MaterialPageRoute(builder: (context) => HomePage()),
);

// 清空栈并跳转（类比登录后跳首页）
Navigator.pushAndRemoveUntil(
  context,
  MaterialPageRoute(builder: (context) => HomePage()),
  (route) => false,  // 清空所有历史
);
```

### 命名路由

```dart
// main.dart 配置路由表（类比 Vue Router 的 routes 配置）
MaterialApp(
  initialRoute: '/',
  routes: {
    '/': (context) => HomePage(),
    '/login': (context) => LoginPage(),
    '/detail': (context) => DetailPage(),
    '/profile': (context) => ProfilePage(),
  },
)

// 跳转
Navigator.pushNamed(context, '/detail');

// 带参数（通过 arguments）
Navigator.pushNamed(
  context,
  '/detail',
  arguments: {'id': '123', 'title': '商品详情'},
);

// 接收参数
class DetailPage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    final args = ModalRoute.of(context)!.settings.arguments as Map;
    return Scaffold(
      appBar: AppBar(title: Text(args['title'])),
      body: Text('ID: ${args['id']}'),
    );
  }
}
```

## 3. GoRouter（推荐）

### 安装

```yaml
dependencies:
  go_router: ^13.0.0
```

### 配置路由（类比 Vue Router 的 createRouter）

```dart
// router/app_router.dart
import 'package:go_router/go_router.dart';

final appRouter = GoRouter(
  initialLocation: '/',
  routes: [
    GoRoute(
      path: '/',
      builder: (context, state) => HomePage(),
    ),
    GoRoute(
      path: '/login',
      builder: (context, state) => LoginPage(),
    ),
    GoRoute(
      path: '/product/:id',  // 动态路由参数（类比 /product/:id）
      builder: (context, state) {
        final id = state.pathParameters['id']!;
        return ProductDetailPage(id: id);
      },
    ),
    GoRoute(
      path: '/search',
      builder: (context, state) {
        final keyword = state.uri.queryParameters['q'] ?? '';
        return SearchPage(keyword: keyword);
      },
    ),
    // 嵌套路由（类比 Vue Router 的 children）
    ShellRoute(
      builder: (context, state, child) => MainLayout(child: child),
      routes: [
        GoRoute(path: '/home', builder: (context, state) => HomeTab()),
        GoRoute(path: '/cart', builder: (context, state) => CartTab()),
        GoRoute(path: '/profile', builder: (context, state) => ProfileTab()),
      ],
    ),
  ],
  // 路由守卫（类比 Vue Router 的 beforeEach）
  redirect: (context, state) {
    final isLoggedIn = AuthService.isLoggedIn;
    final isLoginPage = state.matchedLocation == '/login';

    if (!isLoggedIn && !isLoginPage) {
      return '/login';  // 未登录重定向到登录页
    }
    if (isLoggedIn && isLoginPage) {
      return '/home';   // 已登录不能访问登录页
    }
    return null;  // 不重定向
  },
);
```

```dart
// main.dart
MaterialApp.router(
  routerConfig: appRouter,
)
```

### 使用路由

```dart
// 跳转（类比 router.push('/path')）
context.go('/home');
context.go('/product/123');
context.go('/search?q=flutter');

// push（保留历史，可返回）
context.push('/product/123');

// 带额外参数（不在 URL 中）
context.push('/product/123', extra: productData);

// 接收 extra 参数
GoRoute(
  path: '/product/:id',
  builder: (context, state) {
    final id = state.pathParameters['id']!;
    final product = state.extra as Product?;
    return ProductDetailPage(id: id, product: product);
  },
)

// 返回
context.pop();
context.pop('返回的数据');

// 替换（类比 router.replace）
context.replace('/home');

// 检查是否可以返回
if (context.canPop()) {
  context.pop();
}
```

### 路由跳转动画

```dart
GoRoute(
  path: '/detail',
  pageBuilder: (context, state) => CustomTransitionPage(
    key: state.pageKey,
    child: DetailPage(),
    transitionsBuilder: (context, animation, secondaryAnimation, child) {
      // 从右滑入（默认）
      return SlideTransition(
        position: Tween<Offset>(
          begin: Offset(1, 0),
          end: Offset.zero,
        ).animate(animation),
        child: child,
      );
    },
  ),
)
```

## 4. 页面结构（Scaffold）

每个页面通常用 `Scaffold` 作为骨架：

```dart
Scaffold(
  // 顶部导航栏
  appBar: AppBar(
    title: Text('页面标题'),
    leading: IconButton(           // 左侧按钮（默认是返回键）
      icon: Icon(Icons.arrow_back),
      onPressed: () => context.pop(),
    ),
    actions: [                     // 右侧按钮组
      IconButton(icon: Icon(Icons.search), onPressed: () {}),
      IconButton(icon: Icon(Icons.more_vert), onPressed: () {}),
    ],
    backgroundColor: Colors.white,
    foregroundColor: Colors.black,
    elevation: 0,                  // 阴影高度
  ),

  // 主体内容
  body: MyPageContent(),

  // 底部导航栏
  bottomNavigationBar: BottomNavigationBar(...),

  // 悬浮按钮
  floatingActionButton: FloatingActionButton(
    onPressed: () {},
    child: Icon(Icons.add),
  ),
  floatingActionButtonLocation: FloatingActionButtonLocation.centerDocked,

  // 抽屉（侧边栏）
  drawer: Drawer(
    child: ListView(
      children: [
        DrawerHeader(child: Text('菜单')),
        ListTile(title: Text('首页'), onTap: () {}),
        ListTile(title: Text('设置'), onTap: () {}),
      ],
    ),
  ),

  // 底部弹出内容
  bottomSheet: Container(...),

  // 背景色
  backgroundColor: Colors.grey[100],
)
```

## 5. 弹窗和底部弹出

### Dialog（对话框）

```dart
// 确认对话框（类比 uni.showModal）
showDialog(
  context: context,
  builder: (context) => AlertDialog(
    title: Text('确认删除'),
    content: Text('删除后无法恢复，确定要删除吗？'),
    actions: [
      TextButton(
        onPressed: () => Navigator.pop(context, false),
        child: Text('取消'),
      ),
      ElevatedButton(
        onPressed: () => Navigator.pop(context, true),
        style: ElevatedButton.styleFrom(backgroundColor: Colors.red),
        child: Text('删除'),
      ),
    ],
  ),
).then((confirmed) {
  if (confirmed == true) {
    // 执行删除
  }
});

// 自定义对话框
showDialog(
  context: context,
  barrierDismissible: false,  // 点击外部不关闭
  builder: (context) => Dialog(
    shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(16)),
    child: Padding(
      padding: EdgeInsets.all(24),
      child: Column(
        mainAxisSize: MainAxisSize.min,
        children: [
          Icon(Icons.check_circle, color: Colors.green, size: 64),
          SizedBox(height: 16),
          Text('操作成功！', style: TextStyle(fontSize: 18)),
          SizedBox(height: 24),
          ElevatedButton(
            onPressed: () => Navigator.pop(context),
            child: Text('确定'),
          ),
        ],
      ),
    ),
  ),
);
```

## 7. 路由对比总结

| Vue Router | GoRouter | Navigator |
|-----------|---------|-----------|
| `createRouter` | `GoRouter(routes: [...])` | `MaterialApp(routes: {...})` |
| `router.push('/path')` | `context.go('/path')` | `Navigator.pushNamed(context, '/path')` |
| `router.replace('/path')` | `context.replace('/path')` | `Navigator.pushReplacementNamed(...)` |
| `router.back()` | `context.pop()` | `Navigator.pop(context)` |
| `route.params.id` | `state.pathParameters['id']` | `ModalRoute.of(context).settings.arguments` |
| `route.query.q` | `state.uri.queryParameters['q']` | 手动解析 |
| `beforeEach` | `redirect` 回调 | `onGenerateRoute` |
| `<router-view>` | `ShellRoute` 的 `child` | 自动处理 |
