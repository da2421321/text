## 🚀 快速开始

### 环境要求
- Node.js 16.0 或更高版本
- npm 或 yarn 包管理器

### 安装依赖
```bash
# 进入教程目录
cd Node.js教程

# 安装所有依赖
npm install

# 或者使用yarn
yarn install
```

### 运行示例
```bash
# 运行基础示例
node 基础篇/01-hello-world.js

# 运行HTTP服务器
node 进阶篇/07-http-server.js

# 运行Express应用
node 高级篇/11-express-basic.js

# 运行Nuxt.js应用 (需要先安装Nuxt.js)
cd Nuxt.js框架
npx nuxi dev
```
