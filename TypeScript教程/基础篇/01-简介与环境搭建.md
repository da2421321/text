# 01 - TypeScript 简介与环境搭建

## 什么是 TypeScript？

TypeScript 是由 Microsoft 开发的 JavaScript 的超集，它在 JavaScript 的基础上添加了可选的静态类型和基于类的面向对象编程。

### TypeScript 的优势

1. **静态类型检查**：在编译时发现错误，而不是运行时
2. **更好的 IDE 支持**：代码补全、智能提示、重构
3. **代码可维护性**：类型注解就是最好的文档
4. **最新的 JavaScript 特性**：支持 ES6+ 的所有特性
5. **渐进式采用**：可以在现有 JavaScript 项目中逐步引入

### TypeScript vs JavaScript

```typescript
// JavaScript
function add(a, b) {
    return a + b;
}
add(1, '2'); // "12" - 可能不是预期结果

// TypeScript
function add(a: number, b: number): number {
    return a + b;
}
add(1, '2'); // 编译错误：类型不匹配
```

## 环境搭建

### 1. 安装 Node.js

确保已安装 Node.js（建议 14.0 或更高版本）

```bash
node --version
npm --version
```

### 2. 安装 TypeScript

```bash
# 全局安装
npm install -g typescript

# 验证安装
tsc --version
```

### 3. 创建第一个 TypeScript 项目

```bash
# 创建项目目录
mkdir hello-typescript
cd hello-typescript

# 初始化 package.json
npm init -y

# 安装 TypeScript（项目本地）
npm install --save-dev typescript

# 初始化 tsconfig.json
npx tsc --init
```

### 4. 配置 tsconfig.json

```json
{
    "compilerOptions": {
        "target": "ES2020",
        "module": "commonjs",
        "outDir": "./dist",
        "rootDir": "./src",
        "strict": true,
        "esModuleInterop": true,
        "skipLibCheck": true,
        "forceConsistentCasingInFileNames": true
    },
    "include": ["src/**/*"],
    "exclude": ["node_modules"]
}
```

### 5. 编写第一个 TypeScript 程序

创建 `src/hello.ts`：

```typescript
// src/hello.ts
function greet(name: string): string {
    return `Hello, ${name}!`;
}

const user: string = "TypeScript";
console.log(greet(user));

// 类型推断
let message = greet("World"); // message 自动推断为 string 类型
console.log(message);
```

### 6. 编译并运行

```bash
# 编译 TypeScript
npx tsc

# 运行编译后的 JavaScript
node dist/hello.js
```

### 7. 使用 ts-node（推荐开发时使用）

```bash
# 安装 ts-node
npm install --save-dev ts-node @types/node

# 直接运行 TypeScript 文件
npx ts-node src/hello.ts
```

### 8. 配置自动编译

在 `package.json` 中添加脚本：

```json
{
    "scripts": {
        "build": "tsc",
        "dev": "ts-node src/hello.ts",
        "watch": "tsc --watch"
    }
}
```

运行：
```bash
npm run build  # 编译
npm run dev    # 开发运行
npm run watch  # 监听文件变化自动编译
```

```bash
# 安装依赖

```

创建 `.eslintrc.json`：
```json
{
    "parser": "@typescript-eslint/parser",
    "plugins": ["@typescript-eslint"],
    "extends": [
        "eslint:recommended",
        "plugin:@typescript-eslint/recommended",

}
```

创建 `.prettierrc`：
```json
{
    "semi": true,
    "trailingComma": "es5",
    "singleQuote": true,
    "printWidth": 100,
    "tabWidth": 4
}
```

## 实践练习

创建一个简单的计算器程序：

```typescript
// src/calculator.ts
class Calculator {
    add(a: number, b: number): number {
        return a + b;
    }

    subtract(a: number, b: number): number {
        return a - b;
    }

    multiply(a: number, b: number): number {
        return a * b;
    }

    divide(a: number, b: number): number {
        if (b === 0) {
            throw new Error("除数不能为零");
        }
        return a / b;
    }
}

const calc = new Calculator();
console.log("10 + 5 =", calc.add(10, 5));
console.log("10 - 5 =", calc.subtract(10, 5));
console.log("10 * 5 =", calc.multiply(10, 5));
console.log("10 / 5 =", calc.divide(10, 5));
```
